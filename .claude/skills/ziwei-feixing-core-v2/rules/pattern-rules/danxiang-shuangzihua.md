# P07 单象加双自化

## trigger
- 某宫有一个生年四化象。
- 同宫有两个以上自化象，或自化加同宫飞出象形成双象。

## rc-sources
- RC-xu3-ch6-03
- RC-xu3-ch5-23
- RC-xu2-ch3-05

## 操作步骤
1. 先生年单象定体，确定该宫先天已经有何人/事/物。
2. 将生年单象分别与每一个自化象组合，不把所有象压成一句话。
3. 每个组合各判一件事、一类人或一类物；例如生年禄加自化科是一组，生年禄加自化忌又是一组。
4. 判箭头：自化到对宫时，重心可转向对宫；本宫自化则重心留在本宫。
5. 综合时区分开象与合象：先列两组结论，再说明它们如何共同影响主题。

## 判读规则
- 单象加双自化常表示两件事、两个人或两样物并行。
- 组合中有忌时，该组有不利、收束、纠缠或破耗面；但无忌组合仍可保留良性面。
- 若双自化正好组成禄忌、权忌、科忌，要按双象组合表判性质。

## 边界
- 不可把双自化压成一个吉凶词。
- 若先天已有双象，再遇双自化，走 P08。
- 若只是双象媒介问题，另跑 P02。

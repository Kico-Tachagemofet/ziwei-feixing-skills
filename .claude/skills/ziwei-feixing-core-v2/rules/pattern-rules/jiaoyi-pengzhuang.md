# P13 交易象碰撞

## trigger
- 飞宫四化 X 从 A 宫落入 B 宫。
- B 宫已有生年四化 Y，或 B 宫有自化 Z。

## rc-sources
- RC-xu2-ch3-13
- RC-xu2-ch3-05
- RC-xu3-ch1-04

## 操作步骤
1. 写出交易公式：A 宫 + B 宫 = X，A 为发射/注脚，B 为受象/落点。
2. 列出 B 宫原有的伏象：生年 Y；若 B 有自化，再列自化 Z。
3. 先判 X 与 Y 的碰撞：飞宫交易象为用，生年伏象为体，用象与体象相加成双象或三象。
4. 再判 X 与 Z 的碰撞：自化重心在自化宫，说明后天变化、时间与不稳定。
5. 若组合含禄忌、权忌、科忌，按双象表翻译；禄逢忌常转为双忌性质，必须提示起伏、变化、周转或不利面。
6. 输出时保留 A/B 两宫语义，不只输出组合名。

## 判读规则
- 飞宫逢生年叫交易象逢伏象，表示命与运交错。
- 飞宫逢自化叫交易象逢自化象，表示当下交易与后天变化碰撞。
- 逢生年和逢自化都不能只看飞入象，必须把原有象加进去。

## 输出格式
```yaml
pattern: P13
name: 交易象碰撞
formula: "A宫 + B宫 = 禄"
landing_birth_symbols: [忌]
landing_self_symbols: [禄]
combined_symbols: [禄忌, 禄禄]
reading: "..."
```

## 边界
- B 宫没有生年也没有自化时，不触发本卡，只按普通飞宫公式解。
- 不可把 B 宫大运同层宫职误当本命落点；层级重叠时由小归大命名。
- 组合解释仍须分人、物、事，不可固定套一句。

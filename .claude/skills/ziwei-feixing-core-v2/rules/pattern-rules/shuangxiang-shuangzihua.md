# P08 双象加双自化

## trigger
- 某宫或本对轴已有两个以上生年/飞宫象。
- 同宫或本对轴又有两个以上自化象。

## rc-sources
- RC-xu3-ch6-04
- RC-xu3-ch5-23
- RC-xu3-ch5-04

## 操作步骤
1. 列出先天/飞宫双象组，确认是否含忌；双象盘至少要有一个忌才成强烈气数组。
2. 列出自化双象组，分辨本宫自化与视同自化，并记录箭头重心。
3. 将“原有双象”与“自化双象”分层判读：先生年/飞宫体，再自化用。
4. 若两组都含忌，先列为变故、刑伤、冲突或破耗；再按宫职翻译成婚姻、健康、财物、事业等具体事项。
5. 用媒介象补完整解义：双象缺什么、三象缺什么，就把缺象作为时间或触发点。
6. 应期取自化宫、同类生年宫、媒介象宫成为大命/流命/相关流宫的交会点。

## 判读规则
- 双象自化双象主刑伤或变故；论物多作破耗、损毁、耗损。
- 在夫妻宫可转为姻缘变故、排斥、分合；在疾厄宫可转为身体或精神困扰；在财物宫可转为财物破耗。
- “刑伤”必须按宫职翻译，不直接输出恐吓式断语。

## 边界
- 没有双象基础时，不走本卡。
- 不可只写“必有刑伤”，必须说明是哪一类宫职事项。
- 若证据只够三象媒介，不够双象自化双象，走 P02。

# P10 出忌 / 刺出忌

## trigger
- 飞宫化忌由 A 宫入 B 宫。
- B 的对宫 C 已有生年忌或自化忌。

## rc-sources
- RC-xu3-ch3-01
- RC-xu3-ch4-05
- RC-xu3-ch6-08

## 操作步骤
1. 记录完整链条：A 宫化忌入 B 宫，化忌冲 C 宫。
2. 检查 C 宫是否有生年忌、自化忌，或生年忌加自化忌；有则 B 宫归藏之气被 C 带出。
3. 先按原本忌入 B 的得失判一次，再用 C 的忌修正最后究竟。
4. 按主题翻译：财物为血本无归或再破财；关系为分手、离散、缘薄至极；健康为刑伤、破败或重大消散。
5. 若 C 同时有生年忌与自化忌，等级加重；若有生年科或同干飞科，列为制化因素。
6. 应期取 A、B、C、同类忌宫成为大命/流命/相关流宫的交会点。

## 判读规则
- 忌入六内本可主得，但对宫 C 有离心忌时，最后究竟会从“归藏”改为“散出”。
- 出忌重在对宫已有生年忌；刺出忌重在自化忌或忌气被冲出、带出、破出。
- 名称不如链条重要；输出必须写明 A -> B -> C。

## 输出格式
```yaml
pattern: P10
name: 出忌/刺出忌
chain: "A宫化忌入B宫，冲C宫；C宫有生年忌/自化忌"
initial_reading: "忌入B原本..."
final_reading: "被C忌带出后..."
severity_factors: ["C宫生年忌", "C宫自化忌"]
```

## 边界
- 没有 B 对宫忌，不称出忌。
- 不可只因化忌冲某宫就升级为重大凶象。
- 寿元、重病等结论必须另有疾奴田、福德或健康主题证据。

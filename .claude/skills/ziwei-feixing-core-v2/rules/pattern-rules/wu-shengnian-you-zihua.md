# P05 无生年有自化

## trigger
- 某宫 `native_transforms == []`。
- 同宫 `self_transforms` 非空。

## rc-sources
- RC-xu2-ch3-16
- RC-xu3-ch1-12
- RC-xu3-ch1-13

## 操作步骤
1. 论格：以本命宫职为本质，与自化象合成宫象合一；六亲宫常可作该六亲自力更生格。
2. 论运：看该宫与大运何宫职重叠。本宫论以大运宫职为重心、本命宫职为形容；对宫论以对宫本命宫职为重心、大运宫职为形容。
3. 法象：按自化象找同类生年四化宫，把自化的用归到同类生年之体。
4. 论时间：自化宫或同类生年宫成为流命、流月、流日、流时或相关流宫时应数；本宫与对宫都要检查。
5. 若同一宫有多个自化，先分禄忌组、权科组，再依主题取用，不必四象全取。

## 判读规则
- 无生年有自化属于飞宫体系，不能把自化当成独立先天定数。
- 该宫有变化、旧去新来、有时间观念，但“有变化”不等于凶。
- 自化重心在较小时间单位；大运是十年重心，本命宫职多作修饰。

## 边界
- 只要同宫已有任一生年四化，就不属本卡，改走有生年有自化类规则。
- 不能只说“自化不稳定”；必须完成论格、论运、论时间三步。
- 流年只定应期，不重新决定该事项有没有。

# P06 单象加单自化

## trigger
- 某宫恰好一个生年四化象。
- 同宫恰好一个自化或视同自化象。

## rc-sources
- RC-xu3-ch6-01
- RC-xu3-ch6-02
- RC-xu3-ch5-23

## 操作步骤
1. 先生年、后自化：先生年象为体，说明该宫人/事/物的既有质；再说自化为用，说明后天变化。
2. 判箭头：本宫自化时重心留在本宫；化到对宫视同自化时，箭头转向对宫，重心可由人转事或由事转人。
3. 判同组：同类象如生年权又自化权，作象一数二，有强化也有反其意；禄忌为因果组，权科为互因组。
4. 若自化带忌，另按 P09 标不利面；若同类生年被带出，另按 P11 标震荡。
5. 应期取自化宫、箭头所指宫、同类生年宫成为大命/流命/相关流宫之时。

## 判读规则
- 生年体不被自化推翻；正确语句是“先有而后变化”。
- 单象单自化可是一人一事一物，也可因箭头转向变成对宫主题。
- 同组同类象即使是单象，也按双象语法处理。

## 边界
- 若自化数量超过一个，走 P07 或 P08。
- 若同宫无生年象，走 P05。
- 不可先看自化就改写生年本质。

# P03 万象归一元

## trigger
- 大命或流命所在宫正好坐某一同类生年四化。
- 或大命/流命宫自身出现自化，且可找到同类生年四化宫。

## rc-sources
- RC-xu3-ch1-16
- RC-xu2-ch8-04
- RC-xu3-ch5-21

## 操作步骤
1. 判定方向：大命/流命走到生年 X 为“入”；大命/流命自身自化 X 为“出”。
2. “入”时，搜集其余宫位所有同类自化 X，归入当前大命/流命宫一起判读。
3. “出”时，找到同类生年 X 所在宫，把该宫职视为被带动、震荡或不稳定。
4. 将 X 的象义转成主题语言：禄为缘起/财禄，权为变化/掌控，科为名誉/贵人/延续，忌为收束/亏欠/变动。
5. 应期以大命/流命所在宫、同类自化宫、同类生年宫三者交会为强。

## 判读规则
- 入多作有、成局、主得；即使是忌入，也先说现象成立，再按忌的性质说明冷门、变动、不顺。
- 出多作不稳定、带出、震荡，常表现为同类生年宫职的损失、散动或被迫变化。
- 大运层优先判十年空间，流年层负责确认哪一年触发。


## 边界
- 必须满足“命宫层坐生年同类象”或“命宫层自化带出生年同类象”，不是任意自化都归一元。
- 入/出只定气的方向，不直接等于吉凶。
- 若只是自化同类生年象震荡但命宫层未参与，优先走 P11。

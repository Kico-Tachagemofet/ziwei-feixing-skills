# P11 自化带出震荡

## trigger
- 某宫有自化 X。
- 同宫或全盘存在同类生年 X；同宫同类时触发最强。

## rc-sources
- RC-xu2-ch8-03
- RC-xu3-ch1-16
- RC-xu3-ch5-23

## 操作步骤
1. 标出自化 X 的宫位 A，以及同类生年 X 的宫位 B；若 A=B，标记为同宫带出。
2. 按自化类型命名震荡：自化禄带出生年禄为禄出，自化忌带出生年忌为忌出，自化权为权出，自化科为科出。
3. 以 A 宫定变化发生处，以 B 宫定被带出的先天事项。
4. 若 B 宫还有其他生年象，双象一并被带动，须按双象组合表补充。
5. 应期用 A 宫定大运/流年层，用 B 宫或 B 的同宫双象定更细时间。

## 判读规则
| 自化 | 同类生年象状态 |
|---|---|
| 自化禄 | 生年禄震荡，为禄出 |
| 自化忌 | 生年忌震荡，为忌出 |
| 自化权 | 生年权震荡，为权出 |
| 自化科 | 生年科震荡，为科出 |

- 带出表示先天象被后天变化牵动而不安，不是单纯加强。
- 禄出、科出在一般事项可有显象/应期作用；在健康寿元语境是高风险线索，需要另行合参。

## 输出格式
```yaml
pattern: P11
name: 自化带出震荡
self_palace: A宫
self_symbol: 禄
birth_symbol_palace: B宫
out_type: 禄出
affected_topic: "B宫宫职"
timing_candidates: ["A宫成大命", "B宫成流命"]
```

## 边界
- 没有同类生年象时，走 P05/P04，不输出带出。
- 带出不是自动凶；需按主题、宫职、内外和双象状态翻译。
- 若大命/流命坐生年象并收集全盘同类自化，走 P03。

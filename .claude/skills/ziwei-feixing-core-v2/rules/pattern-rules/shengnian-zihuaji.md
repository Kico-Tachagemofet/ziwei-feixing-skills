# P09 生年加自化忌

## trigger
- 某宫有生年禄、权、科或忌。
- 同宫或视同自化中带化忌。

## rc-sources
- RC-xu3-ch6-05
- RC-xu3-ch5-23
- RC-xu2-ch3-15

## 操作步骤
1. 先生年象定体：说明该宫原本已有的质、事项与来因。
2. 再读自化忌：标记该宫在后天运用中出现不利、排斥、收束、纠缠或破耗面。
3. 分人、物、事翻译：人可作排斥/拒绝合作，物可作无福气/努力未必收获，事可作不顺/变动/结束。
4. 找同类生年忌或禄忌因果组：自化忌要追化禄，说明因何而忌；若生年忌宫被带动，补 P11/P03。
5. 应期取自化忌宫、同类生年忌宫、化禄因宫成为流命或相关流宫之时。

## 判读规则
- 有生年又有自化忌，一律先列为该宫主题的不利面。
- 不利不等于全盘皆凶；要看六内六外、来因宫、是否有生年科/飞科制化。
- 生年忌所在宫自化禄权科忌时，要法象生年同类，再以内外局判吉凶。

## 边界
- 只见自化忌而无生年象，走 P05。
- 若化忌飞入后被对宫忌带出，另走 P10。
- 不可把自化忌解释为生年象不存在；正确顺序是先有后变。

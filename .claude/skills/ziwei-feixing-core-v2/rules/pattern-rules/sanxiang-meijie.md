# P02 三象媒介

## trigger
- 任一宫或本对轴累计命中三个以上四化象：生年、飞宫、自化均可计入。
- 能列出已出现的禄、权、科、忌组合。

## rc-sources
- RC-xu3-ch5-04
- RC-xu2-ch3-05

## 操作步骤
1. 在命中的宫/轴列出实际出现的象，不重复计数同一来源；同类象重复时标为“象一数二”。
2. 若正好三象成局，用缺少的一象作媒介；若只有双象，用双象媒介表找媒介。
3. 先用成局三象判空间与事件成立，再用媒介象找时间、人物或触发宫。
4. 媒介象若落在自化宫、同类生年宫、大命/流命所到宫，列为应期候选。
5. 最后回写宫位主题：发射宫说明来因，落宫说明事件场域，对宫说明照/冲后的得失。

## 判读规则
| 组合 | 媒介 |
|---|---|
| 禄忌 | 权 |
| 科忌 | 禄 |
| 权忌 | 科 |
| 禄权、禄科、权科 | 忌 |
| 禄权忌 | 科 |
| 禄科忌 | 权 |
| 权科忌 | 禄 |

- 三象主空间成局，媒介常代表时间。
- 双象媒介兼具时间与空间，要同时看它落在哪个宫职。
- 四象全见时不再找缺象，改判四象一元：事件完整但需要分主次与强弱。

## 输出格式


## 边界
- 媒介表不能脱离宫职机械套用。
- 若三象来自不同层级，先生年为体，再飞宫/自化为用。
- 只见单象或没有明确宫位，不触发本卡。

# P12 洛书宫职重叠

## trigger
- 大限命宫、大限十二宫或飞宫落点与本命宫职形成洛书重叠关系。
- 同时有生年四化、飞宫四化或自化激活该重叠宫位。

## rc-sources
- RC-xu3-ch1-20
- RC-xu3-ch5-22

## 操作步骤
1. 标出大限宫职与本命宫职同处的宫位，记录是大命、本命，还是某大限事项宫与本命事项宫重叠。
2. 套洛书关系表，确定该重叠主哪类事项。
3. 检查该宫是否坐生年四化、被飞宫四化打到，或有自化；无象不下事项结论。
4. 以生年四化为不动核心，以大限宫职为动态空间，解释这十年为何出现该类事项。
5. 若要定年，再看流命/流宫是否走到生年象、飞化落点或自化宫。

## 判读规则
| 洛书关系 | 重点事项 |
|---|---|
| 戴九履一 / 一五九 / 命财官一体 | 运势、能力、财官、求学事业 |
| 二四为肩 / 兄子一体 | 桃花、学生缘、异性缘、合伙缘 |
| 左三右七 | 驿马、外勤、迁动、远近移动 |
| 六八为足 / 疾奴重叠 | 身体健康、灾厄 |
| 奴疾田重叠 | 健康兼寿元，凶象成立时才进入重大风险 |


## 边界
- 洛书不是脱离飞宫的另一套系统；没有象激活，只能列为背景结构。
- 寿元风险必须同时有疾奴田或健康凶象，不能只凭六八/奴疾田重叠。
- 宫职重叠决定事项类别，不直接决定吉凶。

# P01 退马忌

## trigger
- `flying_transforms` 中存在 A 宫化忌入隔一宫 B。
- B 宫同时有本宫自化忌或视同自化忌。
- 盘面能标出 A、B、B 对宫；缺任一项则不触发。

## rc-sources
- RC-xu3-ch5-01
- RC-xu3-ch1-13
- RC-xu2-ch8-03

## 操作步骤
1. 标出发射宫 A、化忌落宫 B、B 对宫 C，并确认 A 到 B 是隔一宫位，不是普通对冲或三合飞忌。
2. 确认 B 的自化忌类型：本宫自化忌、B 化忌到 C 视同自化忌，或 C 化忌入 B 的直线往来；只有忌气在 B 轴上自化，才成立退马忌。
3. 将忌气归回 A 宫判读：事件表面落在 B，最后归因、后果或压力回到 A 的宫职。
4. 若同盘还有同类生年忌、生年禄自化禄、禄出或科出，另列为带出/应期线索；不要把它并入退马忌本身。
5. 应期优先看 A、B、同类生年忌宫成为大命、流命或相关流宫的时间，交会点最强。

## 判读规则
- 退马忌是“忌入 B 后又由 B 自化忌退回 A”的回弹结构，重点在忌气归处。
- B 宫说明事件场域，A 宫说明最后承受者或根因；C 宫只作被冲/直线往来的辅助证据。
- 财物语境可作得而复失、周转回压；人事语境可作关系压力回到发射宫；健康寿元语境需再有疾奴田等凶象才升级。

## 输出格式
```yaml
pattern: P01

triggered: true

focus_palace: A宫
supporting_palaces: [B宫, B对宫]
reading: "..."
timing_candidates: ["A宫成流命", "B宫成流命", "同类生年忌宫成流命"]
```

## 边界
- 普通化忌入对宫、忌冲命、忌入三合，不自动称退马忌。
- 只见 B 有生年忌而无自化忌，改走 P10 出忌/刺出忌。
- “退回”不是吉凶结论；吉凶仍按六内六外、宫职、人事物主题和制化判断。

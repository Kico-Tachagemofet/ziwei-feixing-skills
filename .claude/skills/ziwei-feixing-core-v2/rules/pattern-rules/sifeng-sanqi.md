# P04 四凤三旗

## trigger
- 某宫有自化，且同宫无同类生年四化作为锚点。
- 能用该宫宫干继续飞出成组象：禄忌组或权科组。

## rc-sources
- RC-xu3-ch1-18
- RC-xu3-ch1-14
- RC-xu3-ch1-19

## 操作步骤
1. 以自化所在宫 A 为标地宫/发射宫，不先用生年宫改写 A 的重心。
2. 成组追象：自化禄再找同宫干飞忌，自化忌再找飞禄；自化权再找飞科，自化科再找飞权。
3. 计算追象落宫相对 A 的距离，并套入距离表判象变。
4. 将距离象义转成事项：邻宫为相欠债，第三宫为三旗有对待，第四宫为四凤主变/形变，三合宫为入宫显吉凶且另一三合宫指时间，第六宫主破耗，对宫主化且形质皆变。
5. 再做公转检查：若追象或自化与生年四化同类/成组，补充先天定数如何约束最终结果。

## 判读规则
| 飞入位置 | 象义 |
|---|---|
| 邻宫 | 两仪，相欠债 |
| 第三宫 | 三旗，有对待，必有吉凶 |
| 第四宫 | 四凤，主变、形变 |
| 三合宫 | 入宫显吉凶，另一三合宫指示时间 |
| 第六宫 | 主破耗，因奴仆为绝情宫 |
| 对宫 | 主化，形变加质变 |

- 本卡处理自化的自转关系，先看宫位自身气与气如何互动。
- 最终仍要回到生年四化定体，不能只凭距离定终局。

## 边界
- 同宫已有同类生年象时，优先走 P11 或 P06-P09。
- 不能只写“四凤/三旗”名称，必须交代 A、追象、落宫距离和事项翻译。
- 若无法确定追象落宫，本卡不输出结论。

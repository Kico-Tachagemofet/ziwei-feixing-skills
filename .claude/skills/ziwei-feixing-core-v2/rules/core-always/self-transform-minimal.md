# 自化最小规则

> Tier 1 core-always。凡遇自化/视同自化必读。
> 来源: RC-xu3-ch1-11/12/13/14/19, RC-xu2-ch3-14

## 三种基本类型

| 类型 | 定义 | 关键差异 |
|---|---|---|
| 本宫自化 | 宫干化星在本宫 | 直接，变化发生在本宫 |
| 视同自化 | 宫干化星在对宫 | 不断缘起，新缘随时间更替 |
| 对宫化入 | 对宫干化星入本宫 | 化出/化入分清方向 |

## 自化的三层含义

1. **一定有变化** — 该宫因人/地/物/时而动。
2. **旧去新来** — 无中生有、有中化无、旧结束新开始。
3. **一定有时间** — 可定十年、一年、一月的应验点。

## 自化交易规则

自化不孤立读。禄忌为因果，权科为互因：

| 看到 | 同时找 | 原因 |
|---|---|---|
| 自化忌 | 追化禄 | 因果组：忌是果，禄是因 |
| 自化禄 | 追化忌 | 因果组：禄是因，忌是果 |
| 自化权 | 看化科 | 互因组：权推动，科修饰 |
| 自化科 | 看化权 | 互因组 |

**先天组定取舍（三步）**：

1. 先列出生年四化组成的先天组：同宫、本对、邻宫、三合均可成组；双象盘至少要有一个忌，常见为禄忌、权忌、科忌，或以权科互因、禄忌因果成组。
2. 遇后天自化时，只取能平衡先天组的象：自化忌追化禄，自化禄追化忌；自化权看化科，自化科看化权。若先天为权忌组，自化忌就找权，自化权就找忌；若先天为科忌组，自化忌就找科，自化科就找忌。
3. 输出时写明“先天组是什么、后天自化取哪一象、为什么舍弃其他飞化”。不需要四个飞化全取；被取用的象再回到宫职、内外、同类生年宫与自化宫定吉凶和应期。

## 自转公转

1. **自转**（先）: 自化禄再飞忌 / 自化忌再飞禄 = 宫位自身气与气互动。
2. **公转**（后）: 自化再与生年四化互动 = 后天变数与先天定数互动。
3. 最终仍回到生年四化这四个”定数钉子”。

## 操作要点

- 有自化时加载 `source-cards/sihua/self-transform.md`。
- 有自化忌再加载 `source-cards/sihua/ji.md`。
- 自化重心在较小时间单位：大运是十年重心，本命宫职多作修饰。
- 无生年有自化 → 触发 pattern-audit P05，读 pattern-rules 卡。
- “忌随禄走”是例外，不能机械套一般飞宫。

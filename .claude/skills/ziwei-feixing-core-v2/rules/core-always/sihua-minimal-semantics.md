# 四化最小语义

> Tier 1 core-always。每次断语前强制查验。
> 来源: sihua-benyi.md §0-§4 压缩 + RC-xu2-ch3-03/04 + A02-A07 四化象 + C00 论命十要

## §-1 四化气化框架（Phase C 取象的底层语义）

四化不是四个操作符，是宇宙气化的四个面向，同时存在于一个太极之内。

**总纲**: 以忌为骨，以禄为本，权为枢纽，科为用。

| 四化 | 气化定义 | 纲位 | 六项特质(许铨仁 A03-A07) |
|---|---|---|---|
| **禄** | 一切因缘的**起点**，缘起 | 本(根) | 财禄福禄 / 主缘 / 忙碌施舍 / 聪明 / 增加 / 解厄制化 |
| **权** | 一切因缘的**自变**，天意自变 | 枢纽(转) | 权柄 / 掌权领导 / 刚果决有责任 / 专技开创 / 争执外伤 / 增值 |
| **科** | 一切因缘的**续接**，承先启后 | 用(应) | 科名声誉 / 恋旧 / 聪明平顺 / 贵人 / 风度桃花 / 和睦 |
| **忌** | 一切因缘的**终点**，果报归主 | 骨(架) | 变动驿马 / 收藏收束管束 / 亏欠情谊 / 不顺是非 / 凶兆灾厄 / 无缘 |

**两组对待**:

| 组 | 族属 | 管什么 |
|---|---|---|
| **禄↔忌** | 同族 | 得失轴（起↔终） |
| **权↔科** | 同族 | 推续轴（变↔续） |

**气化 vs 在数**:

| 层 | 组合 | 对应 | 性质 | Phase C 影响 |
|---|---|---|---|---|
| **气化** | 禄+科 | 人、物 | 可变、可转化 | 禄科双象→有转圜 |
| **在数** | 权+忌 | 天、地 | 不可解、定数 | 权忌双象→天意无奈，不可勉强 |

## §0 断语门禁

写任何涉及四化的断语之前，先填这个模板。**填不出来 → 不准写断语。**

```
本义检查:
  这里的 ___化，按本义是 ___（填缘义 + 动词）
  在 ___宫（人/事/物哪层）表现为 ___
  常见误读 ___ 不适用，因为 ___
```

## 四化本义

| 四化 | 缘义 | 动词 | 不是什么 | 自化时 | 对组 | 深查 |
|---|---|---|---|---|---|---|
| 禄 | 缘起，新开始 | 聚、起、开 | 不是现成财富，不只金钱 | 新缘涌起，变动有新生 | 禄忌同族(因果) | `sihua/lu.md` |
| 权 | 缘变，自然变化 | 变、担、承 | 不是权力(power)，不是轻松得来 | 有应期，天意突变 | 权科同族(互因) | `sihua/quan.md` |
| 科 | 缘续，延续旧有 | 续、修、和 | 不只是贵人，不只是名声 | 平顺延续，贵人随缘 | 权科同族(互因) | `sihua/ke.md` |
| 忌 | 缘终，收束归藏 | 收、定、终 | 不只是坏，忌后可能有生机 | 变动收束，旧结束 | 禄忌同族(因果) | `sihua/ji.md` |

## 操作规则

1. **先定忌再追禄** — 忌定位问题所在，禄追溯缘起来源。
2. **宫象合一先于星** — 象落何宫，先由该宫宫职决定发挥范围，再配星细化。
3. **象从何来** — 生年四化看来因宫，飞宫四化看发射宫。凡言象必须假合两个宫位。
4. **四化给气数方向** — 不等于现实事件。宫位规范范围，星曜细化形相，四化给吉凶气数。
5. **三种四化层级**:

| 类型 | 层级 | 性质 |
|---|---|---|
| 生年四化 | 体 | 先天定数，质不变 |
| 飞宫四化 | 用 | 后天变数，由用归体 |
| 自化 | 用中之用 | 变化+时间，是应期核心 |

6. **主司先于通论** — 化星的主司(官方身份/角色)是 Phase C 取象的第一层依据。
   忌×贪狼: 先读"忌×祸福主=祸福方向的收束"，再读"忌×桃花=桃花收束"。
   禄×武曲: 先读"禄×财帛主=财源的缘起"，再读"禄×将星=果决的开始"。
   跳过主司直接用通论属性（桃花/欲望/变动…）= Phase C 最常见的错误。

# 飞星最小判读语法

> Tier 1 core-always。每次 Stage 2c finding 必读。
> 来源: RC-xu2-ch3-03/05, RC-xu3-ch3-01, RC-xu3-ch5-23

## 固定顺序

1. 确认命盘结构：命宫、宫干、生年四化、主星、辅星、大限/流年层。
2. 先看生年四化，定象之体与先天气数。
3. 选太极点：按用户主题选命、夫、财、官、疾、田等宫。
4. 太极点宫干飞四化，记录：发射宫 → 化象 → 化星 → 落宫/冲宫。
5. 标记自化与视同自化。
6. 宫象星四层合参（缺任一层不出断语）：

| 层 | 读什么 | 决定什么 |
|---|---|---|
| 来源宫 | 发射宫 | 忌/禄的**内容域** |
| 象 | 四化类型 × 入/冲 | 事件**性质** |
| 化星 | 飞的那颗星 | 体现的**质感** |
| 落宫 | 落点宫 | **归处** |

7. 结论必须回写证据链。禁止单星 cookbook 直断。

## 入照冲

- **入**: 象进入某宫，该宫承受该象。六内入多作得。
- **照**: 禄权科入A宫 = **同时照A对宫**(正面辉映)。照 = 对宫也得到好处。
- **冲**: 忌入A宫 = **同时冲A对宫**(负面冲击)。冲 = 对宫受压力。

⚠️ **照命 = 命主得到好处，不是资源外流。**
- "禄科入迁移照命" = 命主被辉映/得到好处。不可读成"好处流向迁移/外在/散出去"。
- "禄入夫妻照官禄" = 感情好也利事业。不可读成"好处给了配偶就没了"。
- **来源宫→落宫 决定内容色彩**: 官禄飞禄→夫妻 = 事业资源利好关系 = 配偶对事业有帮助。不是"事业桃花"。
- 每条飞化必须同时标记"入X"和"照/冲X对宫"，不可只记入不记照。

## 宫位吉凶判读优先级 (后天交易为第一要素)

> 看任何宫位太极（夫妻/官禄/田宅/疾厄/福德/财帛/父母/兄弟/子女）的吉凶基调，**第一要素永远是后天交易**——太极宫飞化↔scope-relevant 关键宫的交互。**先天象（生年忌/自化/双象气数组）不独立判吉凶**，要根据后天交易情况调整读法。
>
> 例外：命宫立太极时，因来因宫=年干→命宫四化=生年四化(体之体)，此规则不适用 (Stage 2 character topic 的标准例外)。

### Layer 1: 后天交易（最优先 — 定吉凶有无）

**Scope-aware 关键宫**：Layer 1 的"凶象/吉象"标签是 scope-dependent，不同 topic 关心不同宫位:

| Scope | 关键宫 (太极飞化要看交互的对象) |
|---|---|
| partnership | 命 + 子女(人位冲人位) + 父母(配偶之子) + 田宅(绝情位) + 福德 |
| career | 命 + 财帛 + 父母(机构/长辈) + 兄弟(同事/合伙) |
| health | 命 + 父母(疾之迁) + 夫妻(生育/感情联动) + 田宅(寿元层) |
| property | 命 + 子女(田之对) + 财帛(田之财源) + 兄弟(田之财位) + 疾厄(田之官) |
| occult | 命 + 财帛(身宫/天巫) + 疾厄(感应) + 父母(传承) |
| children | 命 + 田宅(子之财) + 奴仆(子之疾) |
| siblings | 命 + 疾厄(兄之财) + 父母(兄之疾) |

| Layer 1 信号 | 取象 |
|---|---|
| 太极宫禄/科照命 | 该领域品质好、有缘有情、辉映命主 (明显吉) |
| 太极宫禄/科入命 | 该领域直接利命主(贵人/才华/福气进命) (明显吉) |
| 太极宫忌入命 | 该领域困扰直接砸命主 (明显凶) |
| 太极宫忌冲命(经对宫) | 理则五：该领域冲击命主 (明显凶) |
| 太极宫忌冲 scope-相关人位宫 | 该关系/对象方向有压力 (scope-specific 凶，如夫忌冲子女=配偶子女缘薄) |
| 命飞禄/科入太极 | 命主主动经营该领域 (吉，但偏命主主动) |
| 命飞忌入/冲太极 | 命主主动收束该领域 (用冲体=RC-xu3-ch3-13) |

**量化标准**:
- 禄照命 OR 科照命 = **明显吉**（不需要叠加）
- 1 条忌冲命 = **明显凶**（理则五，独立成立）
- 忌入命 = **明显凶**
- 多条吉/凶叠加 = 更强，但单条已足以定基调

### Layer 2: 先天结构（仅在 Layer 1 不明显时独立成断）

看生年忌、自化（含视同自化）、矩阵（生年×自化同宫同星）、双象气数组（权忌组/禄忌组/科忌组等）。

| Layer 1 → Layer 2 读法 |
|---|
| Layer 1 明显吉 → 先天凶象**降级为形式/质感/隐忧**，不反驳后天的吉。例：夫禄科照命+权忌组 → 权忌组只读"来得突然"，不读"结束突然" |
| Layer 1 明显凶 → 先天吉象可作**缓冲/解厄**，但不抵消凶。例：疾忌冲命+生年禄@疾 → 底子好抗造但凶象成立，起伏型非致命 |
| Layer 1 双吉/双凶 → 钉死基调 |
| Layer 1 **无明显吉凶象** → 先天独立成断。例：夫飞化全不照命+权忌组 → 权忌组独立定底色"突变/平行"，劝晚婚等流年激活 |

### Layer 3: 应期（流年/大运激活）

吉凶基调（Layer 1+2）定下来后，看流年/大运什么时候走到太极宫/自化宫/生年同类宫——触发机会/事件的时间。

### "凶锚/吉锚"= scope-dependent Layer 1 outcome 标签

⚠️ **关键认知**: "凶锚" 不是 static 属性（不是"这个宫客观凶"），是 **scope-dependent Layer 1 outcome**——同一条飞化在 topic A 是凶锚，在 topic B 可能只是 background。

**示例：夫(丙)忌(廉贞)→田宅冲子女**

| Scope | 角色 | 读法 |
|---|---|---|
| partnership (太极=夫) | **凶锚** | 配偶子女缘薄(人位冲人位) + 婚姻面绝情位结构节点 |
| property (太极=田宅) | background | 田宅↔命零互飞 → 田宅对命主间接影响，家里关上门自己扛 |
| 子女 (太极=子女) | **凶锚** | 子女缘被冲(被夫飞忌冲) |
| character (太极=命) | 不适用 | 命宫不是承受宫 |

写 finding 前问自己：**这条飞化在当前 topic scope 下，它的关键宫是谁？是 Layer 1 凶/吉，还是 background？**

### 零互飞定义

**命↔太极宫零互飞 = 入/照/冲全无**（不只飞入，飞照、飞冲都算）。
- 命四化无一落太极宫/太极对宫
- 太极宫四化无一落命宫/迁移宫

零互飞 → 该太极对命主是**间接影响**，结论必须降级（不能像直接互飞那样高 confidence 断"命主直接受影响"）。

### 操作检查清单

写任何宫位吉凶断语前：
- [ ] 当前 scope 的关键宫列出来了吗？（不只命，还包括 scope-specific 人位/事位）
- [ ] Layer 1：太极↔关键宫飞化交互 = 明显吉 / 明显凶 / 无明显象？
- [ ] Layer 1 明显 → Layer 2 降级读形式/质感，不反驳
- [ ] Layer 1 不明显 → Layer 2 独立成断
- [ ] 命↔太极零互飞 → 结论降级（间接影响）
- [ ] Layer 3：应期/激活时机看流年大运

### 旧框架的消解

⚠️ **"凶锚宫飞出的禄权科，先判代价/逃生口/转化/补偿"** (旧 CLAUDE.md 条款) — 在此规则下已**消解**。

原因：那套 ad-hoc 框架本质上是在挽救"被错误标为 static 凶宫但实际飞化吉象"的情况。本规则明确说（这是飞星派原本就应该这样判，只是过去没写清楚）："凶锚"是 Layer 1 outcome，不是 static 标签——飞化对命有禄科照入的宫**从一开始就不是凶宫**，不需要"代价/转化/补偿"去解释吉象。

## 忌汇聚≠凶度 (与 N象一星 平行规则)

一个宫位承受多条忌时，先检查该宫坐星是否为**高频忌星**:

| 高频忌星 | 化忌天干 | 说明 |
|---|---|---|
| 天同 | 庚 | 福德主 |
| 太阴 | 乙 | 财田主 |
| 文昌 | 辛 | 司文 |
| 文曲 | 己 | 司文 |
| 天机 | 戊 | 兄弟主 |
| 廉贞 | 丙 | 官禄主/囚星 |

高频忌星坐某宫 → 多忌汇聚是**十干四化表的算术必然**，不可直接断凶。例: 天同+太阴+文昌同宫 → 庚/乙/辛 三个天干的忌都飞到这里，4-5 忌汇聚是结构常见不是特异。

**判断忌汇聚的实际影响必须看 4 点**:
1. **落宫星旺度** — 旺/庙地吸收忌的能力强，忌到旺地=起伏但不倒
2. **忌的主司** — 是否灾厄级(太阳/太阴=中性偏生活; 廉贞+煞+忌=可灾厄级)
3. **发射宫身份** — 谁给的忌=什么内容色彩(子女发忌 vs 父母发忌 含义完全不同)
4. **同时有无禄权科照** — 禄权科同时照该宫(或从该宫照对宫)=冲力被稀释

⚠️ 不可只数忌的条数就断凶度。"4忌冲命"在天同太阴文昌坐迁移的盘里是常见结构，不是特殊凶兆。

## 双象六组

两个四化象同宫/本对宫/飞宫逢生年逢自化时，按组合含义判读，不拆单象：

| 组合 | 族属 | 事项 | 人/桃花 |
|---|---|---|---|
| 禄权 | 异族 | 丰盛多元 | 泛水桃花，多不专情 |
| 禄科 | 异族 | 名利双收，文路巧艺 | 良性桃花，有缘有情 |
| 权科 | 同族 | 专技才艺，工程军政 | 象二数一，一显一隐 |
| 禄忌 | 同族 | 起伏大变化大周转大 | 通常双忌论凶 |
| 权忌 | 异族 | 专技薪俸，善变突变 | 来急去快，一刀两断 |
| 科忌 | 异族 | 学术教育服务业上班 | 藕断丝连，纠缠拖拉 |

**禄忌例外操作**（不直接走大凶）：

1. **化忌逢生年禄**：先确认落宫原有生年禄，再记录哪一宫化忌后到。先生年禄为体，保留缘起、财禄、人缘或物相基底；后到化忌为用，说明起伏、周转、压力或收束。结论写成“有缘/有物但过程起伏”，不要直接写双忌大凶。
2. **生年禄自化忌**：先说生年禄本来存在，再把自化忌视为禄出或禄被带动。看自化箭头指向本宫还是对宫，并追同类生年禄宫作应期；结论写成“禄有而后变、缘起后有散动”，不把生年禄抹掉。
3. **例外复核**：若同一链条又出现对宫生年忌/自化忌，另触发出忌/刺出忌；若同类生年禄被自化带出，另触发自化带出震荡。例外只修正禄忌组合的初判，不覆盖后续更强结构。

## 时序规则

1. **飞宫先发→自化/生年定究竟**: 飞宫之象先发生，自化与生年再决定最后结果。忌入六内不能直接定吉——对宫若有生年忌/自化忌把气带出，改判最后究竟。
2. **有生年有自化排序**: 先说生年（体，不变），再说自化（用，生吉凶）。体不变而用随时空出入。
3. **解厄制化层级**: 生年科最强 > 飞宫同干科 > 自化科（有时有又像没有）。

## 空宫处理

1. **空宫(无主星)借星安宫**: 不只借对宫主星——对宫的主星+辅星+煞星全部借入，视为该宫的底色来源。借入后与本宫自身的辅星/小星/自化/宫干/地支位合参
2. **合参层次**: 对宫借入(主星+旺度+辅星+煞星) → 本宫自有辅星+小星修饰 → 本宫自化/视同自化叠加 → 本宫宫干+地支位定方向
3. **如有 wtz addon**: 本宫小星用场景化读法(多颗合成画面，不逐颗贴标签)
4. **空宫有自化/视同自化时**: 先看对宫借入定宫态，再叠加自化方向
5. **空宫不等于"没有"**: 空宫=无主星坐守，不等于该宫主题不存在。借星后该宫有完整的星曜质感，只是以对宫为底色、本宫修饰

# 飞星最小判读语法

> Tier 1 core-always。每次 Stage 2c finding 必读。
> 来源: RC-xu2-ch3-03/05, RC-xu3-ch3-01, RC-xu3-ch5-23

## 固定顺序

1. 确认命盘结构：命宫、宫干、生年四化、主星、辅星、大限/流年层。
2. 先看生年四化，定象之体与先天气数。
3. 选太极点：按用户主题选命、夫、财、官、疾、田等宫。
4. 太极点宫干飞四化，记录：发射宫 → 化象 → 化星 → 落宫/冲宫。
5. 标记自化与视同自化。
6. 宫象星四层合参（缺任一层不出断语）：

| 层 | 读什么 | 决定什么 |
|---|---|---|
| 来源宫 | 发射宫 | 忌/禄的**内容域** |
| 象 | 四化类型 × 入/冲 | 事件**性质** |
| 化星 | 飞的那颗星 | 体现的**质感** |
| 落宫 | 落点宫 | **归处** |

7. 结论必须回写证据链。禁止单星 cookbook 直断。

## 入照冲

- **入**: 象进入某宫，该宫承受该象。六内入多作得。
- **照**: 对宫辉映，合参本对轴。
- **冲**: 对宫受冲，忌冲谨慎；但六内物象入不能机械作失。

## 忌汇聚≠凶度 (与 N象一星 平行规则)

一个宫位承受多条忌时，先检查该宫坐星是否为**高频忌星**:

| 高频忌星 | 化忌天干 | 说明 |
|---|---|---|
| 天同 | 庚 | 福德主 |
| 太阴 | 乙 | 财田主 |
| 文昌 | 辛 | 司文 |
| 文曲 | 己 | 司文 |
| 天机 | 戊 | 兄弟主 |
| 廉贞 | 丙 | 官禄主/囚星 |

高频忌星坐某宫 → 多忌汇聚是**十干四化表的算术必然**，不可直接断凶。例: 天同+太阴+文昌同宫 → 庚/乙/辛 三个天干的忌都飞到这里，4-5 忌汇聚是结构常见不是特异。

**判断忌汇聚的实际影响必须看 4 点**:
1. **落宫星旺度** — 旺/庙地吸收忌的能力强，忌到旺地=起伏但不倒
2. **忌的主司** — 是否灾厄级(太阳/太阴=中性偏生活; 廉贞+煞+忌=可灾厄级)
3. **发射宫身份** — 谁给的忌=什么内容色彩(子女发忌 vs 父母发忌 含义完全不同)
4. **同时有无禄权科照** — 禄权科同时照该宫(或从该宫照对宫)=冲力被稀释

⚠️ 不可只数忌的条数就断凶度。"4忌冲命"在天同太阴文昌坐迁移的盘里是常见结构，不是特殊凶兆。

## 双象六组

两个四化象同宫/本对宫/飞宫逢生年逢自化时，按组合含义判读，不拆单象：

| 组合 | 族属 | 事项 | 人/桃花 |
|---|---|---|---|
| 禄权 | 异族 | 丰盛多元 | 泛水桃花，多不专情 |
| 禄科 | 异族 | 名利双收，文路巧艺 | 良性桃花，有缘有情 |
| 权科 | 同族 | 专技才艺，工程军政 | 象二数一，一显一隐 |
| 禄忌 | 同族 | 起伏大变化大周转大 | 通常双忌论凶 |
| 权忌 | 异族 | 专技薪俸，善变突变 | 来急去快，一刀两断 |
| 科忌 | 异族 | 学术教育服务业上班 | 藕断丝连，纠缠拖拉 |

**禄忌例外操作**（不直接走大凶）：

1. **化忌逢生年禄**：先确认落宫原有生年禄，再记录哪一宫化忌后到。先生年禄为体，保留缘起、财禄、人缘或物相基底；后到化忌为用，说明起伏、周转、压力或收束。结论写成“有缘/有物但过程起伏”，不要直接写双忌大凶。
2. **生年禄自化忌**：先说生年禄本来存在，再把自化忌视为禄出或禄被带动。看自化箭头指向本宫还是对宫，并追同类生年禄宫作应期；结论写成“禄有而后变、缘起后有散动”，不把生年禄抹掉。
3. **例外复核**：若同一链条又出现对宫生年忌/自化忌，另触发出忌/刺出忌；若同类生年禄被自化带出，另触发自化带出震荡。例外只修正禄忌组合的初判，不覆盖后续更强结构。

## 时序规则

1. **飞宫先发→自化/生年定究竟**: 飞宫之象先发生，自化与生年再决定最后结果。忌入六内不能直接定吉——对宫若有生年忌/自化忌把气带出，改判最后究竟。
2. **有生年有自化排序**: 先说生年（体，不变），再说自化（用，生吉凶）。体不变而用随时空出入。
3. **解厄制化层级**: 生年科最强 > 飞宫同干科 > 自化科（有时有又像没有）。

## 空宫处理

1. **空宫(无主星)借星安宫**: 不只借对宫主星——对宫的主星+辅星+煞星全部借入，视为该宫的底色来源。借入后与本宫自身的辅星/小星/自化/宫干/地支位合参
2. **合参层次**: 对宫借入(主星+旺度+辅星+煞星) → 本宫自有辅星+小星修饰 → 本宫自化/视同自化叠加 → 本宫宫干+地支位定方向
3. **如有 wtz addon**: 本宫小星用场景化读法(多颗合成画面，不逐颗贴标签)
4. **空宫有自化/视同自化时**: 先看对宫借入定宫态，再叠加自化方向
5. **空宫不等于"没有"**: 空宫=无主星坐守，不等于该宫主题不存在。借星后该宫有完整的星曜质感，只是以对宫为底色、本宫修饰

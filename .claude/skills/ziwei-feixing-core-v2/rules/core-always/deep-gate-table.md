# Deep Gate Table

> Phase B.5 查此表决定是否追读 deep 卡。
> 原分散在 88 张短卡的 needs_deep_when 段落，v3.0 集中于此。

## 通用 Gate（所有类别）

短卡摘录不足以支持当前判断链时，追读对应 deep。

**强制 deep**:

- combo/star 用于 Primary finding、人物画像、主断语、吉凶修正、关系机制判断。
- 需要判断星曜主司优先级（官禄主、祸福主、财星、父母星、荫星等）。
- 需要排除候选义（如亏欠、桃花、无缘、灾厄）。
- 短卡与 deep/原文可能存在优先级差异。

未满足强制 deep 时，finding 只能标 `insufficient-source`，不可 high confidence。

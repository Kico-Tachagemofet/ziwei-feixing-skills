# 化忌分读操作速查

> Tier 1 core-always。每次涉及化忌必读。
> 来源: sihua-benyi.md 精华 + bundle guardrails + 用户 历次校准 + RC-vA06/07/08 + A06-A07 note 查漏补缺(2026-05-22)
> 与 sihua-minimal-semantics.md 互补: 后者=门禁+本义, 本文件=分读操作

## §1 忌的本义（不可跳过）

忌 = **缘终/收束/归藏**。六项特质:

| 特质 | 含义 | 常见误读 |
|---|---|---|
| 变动驿马 | 多变动/多变迁/含驿马 | ≠天机的"机巧谋略"变动。**正面=数理细胞发达/思考逻辑活泼/创意**(忌在命宫个性论); 负面=爱钻牛角尖 |
| **收藏收束** | 归藏/入库/定位/收拢 | ≠一律坏 |
| 亏欠情谊 | 心理上的欠/放不下 | ≠欠债 |
| 不顺是非 | 困扰/卡住/不畅 | ≠灾 |
| 凶兆灾厄 | 最重一层，需理则三/五+煞+忌汇聚才走 | 不可轻易启用 |
| 无缘 | 缘薄/不对频 | 需看禄科是否照命才定(不可只看忌断缘薄) |

## §2 六内六外分判（最高优先级）

**先分六内六外，再定得失。不分则必错。**

| 忌落何处 | 读法 | 关键 |
|---|---|---|
| **六内宫**(命/财/疾/官/田/福) | **归藏/收藏/偏主得** | 财帛忌入六内=收财入库, 不是破 |
| **六外宫**(兄/夫/子/迁/交/父) | 偏失/偏破/偏散 | 六外=他人之事, 忌入六外=给出去/散出去 |

⚠️ **自化忌↓(离心)在六内宫**: 归藏力流向对宫。不是"破耗"。
⚠️ **离心禄↓才是破耗**: 禄=缘起, 离心=外泄 → 缘起外泄=钱/人/物出去。

| 自化类型 | 在六内宫 | 在六外宫 |
|---|---|---|
| 自化禄↓(离心) | **破耗**(缘起外泄=主出) | 新缘涌起但散 |
| 自化忌↓(离心) | **归藏力外泄**(收存方向转移, ≠破) | 困扰外泄(可能减压) |
| 自化科↓(离心) | 延续力外泄 | 贵人/风度外流 |
| 自化权↓(离心) | 掌控力外泄+含破耗 | 权变外放 |

### §2b 桃花收束三条件（特质二·场合二）

**三个条件缺一不可**:
1. 宫位 = **六亲宫**（人位）
2. 星辰 = **桃花星**（贪狼/廉贞/天姚/文昌等）
3. 变化 = **自化忌**

三者齐备 → 该宫之人桃花收束了（自我因素，非外力压制）。
缺任何一条 → 不走收束读法。

### §2c "有其象必有其人事物"

忌在某宫**也主"有"**——忌在财帛=有财(但不多/辛苦/守不住), 忌在田宅=有不动产(但过程不顺)。**忌≠没有。**

## §3 入/冲分判（在六内六外之后）

| 方向 | 读法 | 操作 |
|---|---|---|
| **入** | 忌落在该宫=该宫承受困扰/收束 | 先看六内六外(§2)定得失, 再看主司(§4)定质感 |
| **冲** | 忌落在对宫, 冲到本宫=与本宫无缘/冲击 | 理则二冲C=结构记录不独立断; 只有理则三/五冲命才独立成断。**人位忌冲人位=无缘(价值观差距), ❌ 绝对禁止写"刑克"** |

## §4 主司先行（入/冲之后）

忌×化星的主司(官方身份)是取象第一层。**跳过主司直接用通论属性 = 最常见的错误。**

| 化星主司 | 忌×该主司 | 常见误读 |
|---|---|---|
| 天同(**福德主**) | 福/享受方向的收束 | ≠"拖累"≠"懒" |
| 武曲(**财帛主**) | 财物方向的收束 | ≠"破财"(看六内六外) |
| 太阳(**官禄主/父夫男**) | 事业/权贵方向的收束; 或男性缘收束 | ≠"事业毁灭" |
| 太阴(**财田主/母妻女**) | 财田/女性方向的收束; 或感情收束 | ≠"克妻"(需更多结构) |
| 贪狼(**祸福主**) | 祸福方向的收束 | ≠"桃花断"(先读祸福主, 再读桃花) |
| 廉贞(**官禄主/囚星**) | 官禄/公权方向的收束; 或囚困 | ≠"牢狱"(需煞+忌汇聚) |
| 巨门(**主是非**) | 是非/口舌/暗面的收束 | ≠"被人害" |
| 天机(**兄弟主**) | 谋略/变动方向的收束 | 忌的变动(不安)≠天机的变动(谋略) |
| 文昌/文曲(**司文**) | 文书/科名方向的收束 | 文昌忌=注意文书官非/支票有价证券 |
| 破军(**祸福主/司夫子奴**) | 破耗方向的收束; 祸福面 | ≠"一定破" |

### §4b 亏欠条件（特质三）

**生年忌**落在**六亲宫** → 一律先读**亏欠**（潜意识的、讲不出的莫名亏欠→想弥补→始终不够）。

| 生年忌落宫 | 亏欠对象 | 行为倾向 |
|---|---|---|
| 兄弟宫 | 兄弟 | 想帮兄弟分忧解劳 |
| 夫妻宫 | 配偶 | 为对方付出过多, 相欠相还 |
| 子女宫 | 子女 | 省吃俭用供子女, **最怕溺爱** |
| 交友宫 | 朋友 | 对朋友有情有义 |
| 父母宫 | 父母 | 主孝顺, 只是不足以充分表达 |
| 命宫 | 自己 | 对自己不顺/固执/自尊与自卑并存 |

⚠️ 条件: **必须是生年忌**(先天层)。飞宫忌落六亲宫走管束/困扰/是非(§5), 不走亏欠。

## §5 人/事/物位分判（主司之后）

| 忌落何种宫位 | 条件义 | 禁止 |
|---|---|---|
| **人位**(六亲宫: 兄/夫/子/交/父 + 命) | 困扰/是非/纠缠/执念 | ❌ 不写"管束"(管束是权的概念) |
| **事位**(官/财/福 等六事宫) | 不顺/变动/收束 | — |
| **物位**(田宅/财帛 等) | 收藏/归库/入库 | 六内物位忌=偏得(归藏) |

## §6 三种忌的层级

| 类型 | 层级 | 性质 | scope |
|---|---|---|---|
| **生年忌** | 体 | 先天定数, 永久存在 | 终身 |
| **飞宫忌** | 用 | 后天变数, A宫交易B宫 | 受发射宫与太极点关系限定 |
| **自化忌** | 用中之用 | 变化+时间, 是应期核心 | 本宫变动信号 |

三种共存时: 先说生年(体不变), 再说自化(用, 生吉凶), 飞宫叠加方向。

## §7 禄忌交互规则

| 场景 | 读法 | 来源 |
|---|---|---|
| 飞忌碰生年禄(落宫已有禄) | **先有缘(禄), 后有起伏(忌)**。不直接双忌大凶。 | feixing-minimal-grammar 禄忌例外 |
| 生年禄+自化忌(同宫) | **先说禄在, 再看忌出方向。走生年×自化矩阵, 不走双象六组。** | shengnian-zihua-matrix |
| 禄随忌走 | 看到忌必须追禄(因果組); 看到禄必须找忌。**单边断都是不完整。** | CLAUDE.md 最高优先级 |

## §8 操作检查清单

写任何涉及化忌的断语前, 逐条过:

- [ ] **六内六外**分了吗? (§2)
- [ ] **桃花收束**三条件检查过了吗? (§2b, 如涉及桃花星自化忌)
- [ ] **"有其象必有其人事物"**记住了吗? 忌≠没有 (§2c)
- [ ] **入还是冲**标了吗? (§3)
- [ ] **主司**先于通论了吗? (§4)
- [ ] 如果是**生年忌落六亲宫**, 先读亏欠了吗? (§4b)
- [ ] **人/事/物位**分了吗? (§5)
- [ ] **生年/飞宫/自化**层级标了吗? (§6)
- [ ] **禄随忌走**追了吗? (§7)
- [ ] 过了 sihua-minimal-semantics §0 门禁了吗?
- [ ] 没写"管束"吧? 没写"刑克"吧?

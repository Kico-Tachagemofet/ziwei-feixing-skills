# 生年×自化组合矩阵

> Tier 1.5 session-warm。从 sihua-benyi.md §6 提取。
> v2.4 新增: 从深参考提升到 session-warm，每次断盘会话加载。
> 来源: RC-vC00-04, RC-vC00-11, RC-xu3-ch1-12

## 适用范围

此矩阵适用于**同一宫内**生年象与自化象**共存**。

## 与双象六组的 scope 区分

| 场景 | 两个象的来源 | 用哪套规则 |
|---|---|---|
| 飞化落 B 宫，B 宫已有生年/自化象 | 一个外来 + 一个宫内既有 | **双象六组** (feixing-minimal-grammar.md) |
| 同一宫有生年象 + 自化象 | 两个都是宫内既有 | **生年×自化组合** (本矩阵) |

**不可混用。** 两套规则结论可完全相反。
区分方法：先判断两个象的来源——一个从外飞入 vs 两个都是宫内先天配置。

## 生年禄 + 自化

| 组合 | 本义解读 | 本义关键 |
|---|---|---|
| 生年禄 + 自化禄 | 禄出，缘散 | 有过但在散 |
| 生年禄 + 自化权 | 有缘有物，天意使然变化 | 禄在，权推动 |
| 生年禄 + 自化科 | 有缘有续，念旧延续 | 禄在，科续 |
| 生年禄 + 自化忌 | 有而后变，缘起后有散动 | 禄在但忌出 |

## 生年权 + 自化

| 组合 | 本义解读 | 本义关键 |
|---|---|---|
| 生年权 + 自化禄 | 有物有变，新缘涌起 | 权在，禄新起 |
| 生年权 + 自化权 | 权出，权力/担承不稳 | 过犹不及 |
| 生年权 + 自化科 | 有物有续，平稳延续 | 权在，科续 |
| 生年权 + 自化忌 | 先有担承后收束，来急去快但非双象六组的一刀两断 | 权在，忌收 |

## 生年科 + 自化

| 组合 | 本义解读 | 本义关键 |
|---|---|---|
| 生年科 + 自化禄 | 旧缘上新发，良性桃花 | 科在，禄新 |
| 生年科 + 自化权 | 旧缘上变化，科权同组现象成局 | 科在，权变 |
| 生年科 + 自化科 | 科出，贵人/延续力不稳 | 过犹不及 |
| 生年科 + 自化忌 | 旧缘收束，藕断丝连感强 | 科在，忌收 |

## 生年忌 + 自化

| 组合 | 本义解读 | 本义关键 |
|---|---|---|
| 生年忌 + 自化禄 | 旧象终止后新生，物换星移再新开始 | 忌=旧底色，禄=变动方向(新) |
| 生年忌 + 自化权 | 未完全弃旧，在旧基础上突变创新。应期前强求不得，到了来得突然 | 权=权变/天意，不是"掌控" |
| 生年忌 + 自化科 | 偏向人的问题，也含健康。类似忌+权但更人、更柔 | 科=缘续/念旧 |
| **生年忌 + 自化忌** | **渐变，急不来。人事物皆为渐变** | **双忌不是双倍坏，是慢** |

## 四种变量总纲

| 有无生年 | 有无自化 | 性质 | 关键 |
|---|---|---|---|
| 无 | 无 | 纯飞宫（用） | 必须飞宫才显现象 |
| 无 | 有 | 飞宫 + 变动（用） | 先知用神何组 |
| 有 | 无 | 先天存在（体） | 大限流年坐照才有现象 |
| 有 | 有 | 由体入用 | **先有再变，不可直接说无** |

## 附: 不利论规则

RC-vC00-11: 凡生年四化宫位有自化忌，该宫**一律先作不利论**，再分人、事、物解释。
RC-xu2-ch8-03: 自化忌→同类生年忌震荡，为忌出。

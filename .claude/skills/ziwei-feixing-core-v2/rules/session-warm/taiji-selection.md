# 太极点选择

## 以宫定极（论人/论事）

- 泛问本人整体：命宫。
- 婚恋、配偶、关系责任：夫妻宫，合参官禄、命迁、子女。
- 财务、收入、现金流：财帛宫，合参福德、官禄、田宅。
- 事业、职位、学业、形象：官禄宫，合参夫妻、命宫、财帛。
- 健康、身体、灾厄：疾厄宫，合参父母、命迁、福德；高风险主题必须保守表达。
- 房产、家庭资产、居住：田宅宫，合参子女、财帛、父母。

## 以星定极（论物）

问具体的物时，先定哪颗星代表该物，再看该星坐哪宫，以该宫为太极。
- 物象锚点星由 用户指定，CC 不自动映射。
- 锚点星只决定太极在哪个宫，吉凶由飞化决定，不由星的静态标签决定。

选定太极点后，再飞该宫宫干四化。

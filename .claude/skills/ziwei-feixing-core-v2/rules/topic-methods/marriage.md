## Finding procedure

Step 1: 先立婚姻太极，检查命宫、夫妻宫、福德宫；再展开婚姻六内宫：命、夫妻、财帛、疾厄、奴仆、田宅。夫妻看缘分与配偶，财帛看婚后对待，田宅作夫妻绝情位。
Step 2: 查本夫与大夫飞化。第一次结婚必须同时看本夫与大夫；后续婚姻可重点看大夫。记录夫妻宫/大夫化禄、权、科、忌入何宫，尤其忌冲命、忌冲夫、忌入田宅或田宅对宫。
Step 3: 判成婚或离散结构。成婚看禄缘、忌名分、权媒介、科最后质；离婚/分居看同类忌冲、田宅绝情位、妻财变化与自化时间。
Step 4: 桃花另查兄子重叠、二四为肩、大子飞禄科权忌入本兄，以及命夫福桃花星/贵人星过多；用禄科、权忌、科忌、禄忌等组合定桃花性质。
Step 5: 应期取本夫/大夫自化宫、同类生年宫、媒介象宫，以及流命/流夫/流财走到的交会点；输出主应年份和备选年份。

### RC-xu3-ch3-01: 飞宫先发生，自化与生年决定最后究竟

> 来源：`extractions/xu-book3/ch3-marriage-taohua.md`
- 许铨仁在进入婚姻桃花前补充：飞宫之象先发生，自化与生年再发生，用来定最后究竟。比如大财化忌入田宅本主得财；若田宅对宫有生年忌、自化忌，或生年忌加自化忌，则形成“出忌”，归藏之气散掉，物上可变成破财，人上可变成生离死别。
- ### 解厄制化层级
- 1. 生年科最有解厄制化力。
- 2. 飞宫同宫干化科也有一定解厄力。
- 3. 自化科力量薄弱，有时“有又像没有”。
- ---

### RC-xu3-ch3-02: 婚姻桃花先以缘业理解

> 来源：`extractions/xu-book3/ch3-marriage-taohua.md`
- 婚姻夫妻是缘，父子是债；缘有良缘恶缘，缘业相生。有的缘浓业轻，表现为幸福快乐；有的缘薄业重，可能到生离死别。
- 婚姻桃花主题包括：
- - 几岁结婚。
- - 配偶个性、才华、家室。
- - 婚后对待与是否生离死别。
- - 桃花、外遇、出轨与感情困扰。
- ---

### RC-xu3-ch3-03: 婚姻先看命、夫、福，再看婚姻六内宫

> 来源：`extractions/xu-book3/ch3-marriage-taohua.md`
- 论先天姻缘，优先看命宫、夫妻宫、福德宫。进一步处理婚姻对待、生离死别、配偶身体与绝情位时，再用婚姻六内宫：命、夫妻、财帛、疾厄、奴仆、田宅。
| 宫位 | 婚姻用途 |
| 命宫 | 统管一切；熟练后可由大命直接论婚姻桃花 |
| 夫妻 | 异性缘、婚姻观、姻缘厚薄、配偶个性才华家室 |
| 福德 | 第二夫妻宫，尤其传统女命看先天姻缘与享福 |
| 财帛 | 夫之夫，婚后夫妻对待位 |
| 疾厄 | 躯体的我；身体层面的夫妻互动 |
| 奴仆 | 配偶身体位，也可辅助断配偶生肖 |

### RC-xu3-ch3-04: 田宅为夫妻绝情位，主生离死别断点

> 来源：`extractions/xu-book3/ch3-marriage-taohua.md`
- 以夫妻宫立太极，田宅为夫之奴仆，是夫妻宫的绝情位。田宅出问题，可表现为夫妻宫之人不在、离婚或生离死别。
- ### 应用
- - 夫妻与财帛合参看婚后对待。
- - 田宅参与“妻田交易”时，可断搬家、分居、离婚或丧偶。
- - 若后天气数形成的权忌组平衡先天权忌组，现象必然发生，再以流命、流夫、流财走到对应宫位定时间。
- ---

### RC-xu3-ch3-05: 夫妻宫四煞、天刑与未宫组合影响先天姻缘

> 来源：`extractions/xu-book3/ch3-marriage-taohua.md`
- 夫妻宫见四煞会影响婚姻和谐：
| 星曜 | 婚姻倾向 |
| 羊刃、陀罗 | 吵架加动手，破坏力较大 |
| 火星、铃星 | 争吵、口舌，较偏言语冲突 |
| 天刑 | 人生观、价值观、婚姻观差距，话不投机、精神孤寂 |
- 未宫属后天八卦坤卦，主婚姻。命宫或夫妻宫在未，遇武贪、天相、太阳太阴再加昌曲左右等，常主先天姻缘不美，女命尤重。
- ---

### RC-xu3-ch3-06: 命夫福桃花星与贵人星过多，先天姻缘有缺憾

> 来源：`extractions/xu-book3/ch3-marriage-taohua.md`
- 命宫、夫妻宫、福德宫为先天姻缘三宫。若桃花星或贵人星过多，婚前婚后都容易有感情困扰。
- ### 星曜分类
| 类型 | 星曜 |
| 桃花星 | 天姚、贪狼、廉贞、昌曲、红鸾、化禄、化科 |
| 贵人星 | 左右、昌曲、魁钺、化科 |
- 破坏力前三名：右弼 > 左辅 > 化科。右弼或化科在命、夫、福，容易出现三角、四角恋；若婚前未应，婚后也可能出现感情困扰或二婚象。
- ---

### RC-xu3-ch3-07: 夫妻宫是共业宫，再一次现象未必由命主本人应

> 来源：`extractions/xu-book3/ch3-marriage-taohua.md`
- 夫妻宫包含自己与配偶的共业，如父母宫同时含父母。夫妻宫有左右等“再一次”现象时，可能是命主本人再婚，也可能是配偶曾经再婚、丧偶或离婚后再与命主成婚。
- ---

### RC-xu3-ch3-08: 四化可用于形相细节，但只能作辅助校时

> 来源：`extractions/xu-book3/ch3-marriage-taohua.md`
- 案例中用形相辅助定寅时/卯时：武曲主鼻子、胸部，天同主丰满，文曲主嘴唇，巨门主嘴巴，太阴太阳主眼睛。四化修饰形相：
| 四化 | 形相倾向 |
| 禄 | 大、厚、扩散 |
| 权 | 隆挺、饱满 |
| 科 | 细、小、巧 |
| 忌 | 收敛、收束 |
- ---

### RC-xu3-ch3-09: 第一次结婚必须看本夫和大夫

> 来源：`extractions/xu-book3/ch3-marriage-taohua.md`
- 第一次结婚一定要看本夫与大夫；后续婚姻则可直接看大夫。本夫视同自化禄，常表姻缘早发但未必早婚；是否成婚还要看化忌名分、化权媒介与化科最后质。
- ### 成婚结构示例
- - 夫妻宫视同自化禄：有缘。
- - 夫妻宫化忌入父母：有名分。
- - 夫妻宫化权入命：作为禄忌媒介，事件发生。
- - 夫妻宫化科入田宅：最后质为不会离婚，若凶则可能是配偶死亡而非离婚。
- ---

### RC-xu3-ch3-10: 桃花组合要分良性、恶性、人缘与肉欲

> 来源：`extractions/xu-book3/ch3-marriage-taohua.md`
- 桃花可按四化组合分性质：
| 组合 | 桃花倾向 |
| 禄科 | 有缘有情，较良性 |
| 权忌 | 来得快、去得莫名，不期而遇，结束干脆 |
| 科忌 | 拖拉、藕断丝连、纠缠是非，分分合合 |
| 禄忌 | 起伏大、投入深，时间到可一刀两断 |
| 禄权 | 泛水桃花，不专情，花花公子型 |
| 权科 | 二数一，一先一后；或一显一隐 |

### RC-xu3-ch3-11: 兄子重叠与大子飞禄科入兄可作命带桃花

> 来源：`extractions/xu-book3/ch3-marriage-taohua.md`
- 洛书二四为肩，兄弟与子女重叠主桃花。命带桃花可由静态或动态形成：
- - 静态：大子与本兄重叠。
- - 动态：大子化禄、化科、化权、化忌入本兄。
- - 若形成大子 + 本兄 = 科，再逢自化禄，桃花必发。
- 若碰到生年忌，会带动生年干来因宫；来因为官禄时，异性桃花常来自工作或求学环境。
- ---

### RC-xu3-ch3-12: 来因宫决定命坐禄的聪明与缘分类型

> 来源：`extractions/xu-book3/ch3-marriage-taohua.md`
- 命坐生年禄可作聪明、人缘、财禄，但具体领域要看来因宫：
| 来因宫 | 命坐禄的倾向 |
| 子女 | 异性缘、桃花、交往异性的聪明 |
| 财帛 | 生意人的聪明 |
| 官禄 | 工作上的聪明 |
| 父母 | 读书、头脑聪明 |
| 田宅 | 家人缘佳，与家族成员相处好 |
- ---

### RC-xu3-ch3-13: 用冲体必凶，命冲夫与夫冲命要分责任来源

> 来源：`extractions/xu-book3/ch3-marriage-taohua.md`
- 命宫化忌冲夫妻，主命主因素造成先天姻缘不好；夫妻化忌冲命，则偏对方因素。夫妻为用、命为体，用冲体必见凶象，六亲宫可作缘薄到极点。
- ### 互为阴阳与各为阴阳
- - 互为阴阳：A+B 合为一太极，先看象如何成立。
- - 各为阴阳：A 化忌影响 B，也冲 B 对宫，开始有吉凶。
- ---

### RC-xu3-ch3-14: 离婚应期看同类忌冲、三象成立与自化时间

> 来源：`extractions/xu-book3/ch3-marriage-taohua.md`
- 离婚案例中，命宫或大夫化忌冲夫妻，再逢生年权、自化禄，形成三象决定一人一事一物；若来因被带动为子女宫，可解释为因桃花而成忌。因有自化，往下应时间，看本官/大夫/流命/流夫走到。
- ### 科忌姻缘
- 科忌组的姻缘往往纠缠不清；离婚后可能借故回来，藕断丝连。
- ---

### RC-xu3-ch3-15: 权忌组婚姻：先判有无，再用忌入内外定真假与成立

> 来源：`extractions/xu-book3/ch3-marriage-taohua.md`
- 权忌在同宫或本对，命盘可定为权忌组。权忌组命盘中，任何宫位化权常碰生年忌或自化忌；化忌也常碰生年权或自化权。后天象要平衡先天权忌组，现象才会成立。
- ### 成婚判断
- - 大夫化权逢自化权：姻缘有成立象。
- - 再看化忌飞往何处：入六内则成立，入六外则不稳或难成。
- - 应期取自化宫、生年权宫、流命/流夫走到的交会点。
- ---

### RC-xu3-ch3-16: 福德宫可看因缘果报与婚外缘分

> 来源：`extractions/xu-book3/ch3-marriage-taohua.md`
- 福德宫是大太极，含三世因果、福报、享福、应酬、人生观，也可看缘分。福德化禄曰缘，化科曰因；禄科落在哪个宫位，其宫位生肖之人与命主特别有缘，婚后也可能节外生枝。权忌用于定位。
- ### 因缘果报四组
| 组别 | 宫位 | 受伤/消耗对象 |
| 田宅组 | 田宅、兄弟、疾厄 | 不利福德，损福报或祖父母 |
| 福德组 | 福德、迁移、夫妻 | 不利父母 |
| 父母组 | 父母、子女、奴仆 | 不利自己 |
| 命宫组 | 命、财帛、官禄 | 不利兄弟 |
- ---

### RC-xu3-ch5-05: 初婚应期先看本夫，再以大夫、媒介、自化宫定年月至时

> 来源：`extractions/xu-book3/ch5-ten-essentials-marriage.md`
- 第一次结婚不能只看大夫，仍要以本夫为先。本夫若化禄权忌入夫妻对待位，本身表示有配偶；再逢自化，自化宫有时间观念，可取该宫变为大命或大夫的大运。若三象媒介为科，则用科来决定流年、月份乃至时辰。
- 另一婚姻案例中，本夫科忌入命，平衡先天科忌组；禄落田宅逢生年禄、自化禄，主缘分成局并带时间。大夫再以禄、权、科、忌分名分、占有、介绍、对待，最后以自化权、法象生年权定 27 岁八月结婚。
- ---

### RC-xu3-ch5-08: 科忌组常主上班格、为人作嫁，婚姻则有拖延纠缠

> 来源：`extractions/xu-book3/ch5-ten-essentials-marriage.md`
- 科忌组可由科忌同宫、本对、一命一官有科忌等方式成立。职业上常作上班格、为人作嫁；论财官时，科主人、忌主物，也可解释为人事与物质纠缠。婚姻上，科忌常表现为拖拖拉拉、纠缠不清。
- 健康或精神面上，科忌可作精神困扰、自刑、想不开。夫妻宫科忌若再与大运疾厄合参，可转为配偶身体、本人身体或婚姻精神压力。
- ---

### RC-xu3-ch5-09: 分居离婚看大夫垂象示凶、妻财变化与旧新交替

> 来源：`extractions/xu-book3/ch5-ten-essentials-marriage.md`
- 分居离婚案例中，本夫自化权表示配偶能干、自信，但若大夫化忌冲本夫，则大运婚姻垂象示凶。自化禄权法象生年禄权，可用于定结婚或分居的月份；大财本夫有自化，因妻财本有对待，有自化就有变化，表示旧的结束、新的开始。
- 若下一大运大夫继续垂象示凶，就可判断离婚节点。案例中，以大夫巨门忌、自化禄、视同自化权法象生年禄权，定分居月份和离婚年份。
- ---

### RC-xu3-ch5-19: 先天气数组取法须随主题立太极，婚姻可屈就夫妻宫组

> 来源：`extractions/xu-book3/ch5-ten-essentials-marriage.md`
- 先天气数组取法：
- 1. 同宫、本对优先。
- 2. 邻宫取象。
- 3. 四象分散时，先看六内宫，再看哪一宫有生年又有自化。
- 4. 论具体主题时，可改以主题宫立太极。
- 因此，论自己以命宫为原太极，可能是权忌组；论婚姻时以夫妻宫立太极，夫妻宫若有生年禄，就可改作禄忌组；夫妻宫生年科自化权，则论婚姻可屈就权科组。财官同理，应依财帛、官禄主题转太极。
- ---

### RC-xu3-ch5-20: 结婚预测要承认概率分布，不作绝对同年同月论

- 预测结婚应先认识在数与化气，再看夫妻宫、本夫、大夫、来因与先天气数组。若夫妻宫生年禄再自化科忌，表示缘浓、缘早现，但科忌拖延，要走过忌年、科年才结婚。若夫妻宫生年科自化权，为权科同组，时空俱在，可给 23、25、27、30 等不同可能点。
- 原文明确提醒，同年同月同日同时生的人不一定都同年结婚；可能 30 个同盘里，卯年、午年、申年各有人应。预测应给主应年份与备选年份，并说明年月日时通常集中在同一组宫位轴线。

## 技术边界（仅原 RC 明示时填写）

- RC-xu3-ch3-09: 第一次结婚一定要看本夫与大夫；后续婚姻则可直接看大夫。本夫视同自化禄，常表姻缘早发但未必早婚；是否成婚还要看化忌名分、化权媒介与化科最后质。
- RC-xu3-ch5-05: 第一次结婚不能只看大夫，仍要以本夫为先。本夫若化禄权忌入夫妻对待位，本身表示有配偶；再逢自化，自化宫有时间观念，可取该宫变为大命或大夫的大运。若三象媒介为科，则用科来决定流年、月份乃至时辰。

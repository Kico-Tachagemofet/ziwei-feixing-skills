## 使用时机

## 编译原则

- 只编译高级班明确给出的技术操作规则。

## Finding procedure

Step 1: 按问题分体用。运势、事业、求学、考运看命宫为体、官禄为用；财运、投资、创业、理财看命宫为体、财帛为用；合伙另以奴仆为体、子女为用。
Step 2: 先判先天财官：看生年忌在六内/六外定得失，再看生年禄权科落六阳/六阴定贵富，并以来因宫分自立格或他荫格。
Step 3: 再判后天大运：大命是否走到生年四化；大财、大官化忌是否入本命命、田、疾等体位；本命宫职飞宫主大运，大运宫职飞宫主流年。
Step 4: 财物优先用内外定得失、阴阳定多寡；事业调动优先用阴阳定动静、内外定吉凶；人事关系优先用阴阳定有无、内外定聚散。
Step 5: 行业/格局用组合收束：禄科偏文教巧艺企划，禄权偏生意开创，禄忌偏流通变化，科忌偏上班格；官运选运兼看官禄与奴仆。
Step 6: 应期先取大运空间，再看流命走生年、权/忌分流年流月，以及自化宫或同类生年宫交会。

### RC-xu3-ch2-02: 三合为用、四正为体，六阴富六阳贵

> 来源：`extractions/xu-book3/ch2-wealth-career.md`
- 财官论断中，洛书三合偏用，河图四正偏体：
| 体系 | 重点 |
| 三合为用 | 命财官，命官为主，用于运途、事业、财官发挥 |
| 四正为体 | 命迁子田，命田为主，用于承载、库藏、实体 |

| 六阴 | 兄、子、疾、奴、田、父，主富 |
- 命或官有生年权，可视为命财官一体都有权；财帛化忌入命或田宅，属用归体，主得。

### RC-xu3-ch2-03: 富贵贫贱要依来因宫与人事物注脚

> 来源：`extractions/xu-book3/ch2-wealth-career.md`
- 论先天富贵贫贱，以生年四化落宫为基础，并以来因宫作注脚。飞宫四化则以发射宫作注脚。注脚还要依人、事、物区别：
| 注脚对象 | 判断重点 |
| 人 | 散聚、有无、关系 |
| 物 | 得失、多寡 |
| 事 | 吉凶、兴衰、动静 |
- 例如来因为财帛而生年忌在子女，论财官时子女可不作桃花，而作合伙，表示合伙不利。
- ---

### RC-xu3-ch2-04: 财官最怕化忌冲父疾、迁命、田库等体位

> 来源：`extractions/xu-book3/ch2-wealth-career.md`
- 以财帛或官禄立太极时，忌冲体位或关键库位多主重大损失。
- ### 财帛忌
- - 财帛化忌入父母冲疾厄：大破财。
- - 财帛化忌入迁移冲命：大破财。
- - 财帛化忌入田宅：主大得财，因田宅为库且为六内六阴。
- ### 官禄忌
- - 官禄化忌入父母冲疾厄：事业倒闭或重大失败。
- - 忌冲三合可作破格；忌冲田疾可作损格。

### RC-xu3-ch2-05: 财官断法按物、事、人分别取内外阴阳

> 来源：`extractions/xu-book3/ch2-wealth-career.md`
- 内外与阴阳的优先级因对象而变：
| 对象 | 第一判断 | 第二判断 |
| 物（财） | 内外定得失：入六内得，入六外失 | 阴阳定多寡：六阴多实，六阳少虚 |
| 事（事业/调动） | 阴阳定动静：六阴主动，六阳主静 | 内外定吉凶：内吉外凶 |
| 人 | 阴阳定有无 | 内外定聚散 |
- ---

### RC-xu3-ch2-06: 格和局是两回事，生年忌是财官转变分水岭

> 来源：`extractions/xu-book3/ch2-wealth-career.md`
- 格如命，局如运。先天格局决定上限，后天努力可改善运局但不必然改变命格。财官一生的转变，以生年忌所落宫位为分水岭，分为忌前与忌后。
- ### 财官转变法
- 1. 未过生年忌前，即使大命、大财、大官好，也受先天限制。
- 2. 过生年忌后，重点看生年禄权科的大运/流年走到或照到。
- 3. 已过生年忌后，论财官以大命、大官为主，大财为辅。
- 4. 官禄以化权为主，忌为变通；财帛以化忌为主，权为转进。
- ---

### RC-xu3-ch2-07: 自立格与他荫格取来因六内六外

> 来源：`extractions/xu-book3/ch2-wealth-career.md`
- 定富贵贫贱后，要看来因宫在六内还是六外：
| 来因宫 | 格局 |
| 六内（命、财、疾、官、田、福） | 自立格 |
| 六外（兄、夫、子、迁、奴、父） | 他荫格 |
- 来因六外者，即便禄权科三方会照成贵格，也可能“人荫其成”，未必能自成君格。
- ---

### RC-xu3-ch2-08: 终身行业先取命财疾，后取命田疾

> 来源：`extractions/xu-book3/ch2-wealth-career.md`
- 论终身行业与性向，先看先天，再看后天：
- 1. 以命、财、疾为主体。若有生年四化落入，终身行业多与该象及星曜相关。
- 2. 再配来因宫与河图数，概括人生范围。
- 3. 若命财疾无生年四化，再看财帛宫四化是否飞入命、田、疾为主，官、福、财为次。
- 生年四化属先天，配河图；后天用财帛宫四化时，改以洛书应用，重命田疾。
- ---

### RC-xu3-ch2-09: 财官单象与双象有固定职业/得失倾向

> 来源：`extractions/xu-book3/ch2-wealth-career.md`
- ### 单象
| 象 | 财官倾向 |
| 禄 | 资金、投资、创业、巧艺、财的付出 |
| 权 | 升迁、变动、创业开创、专技 |
| 科 | 科甲功名、上班、企划设计、智慧付出 |
| 忌 | 得失、变迁、囚困、收藏、劳心付出 |
- ### 双象
| 组合 | 财官倾向 |

### RC-xu3-ch2-10: 化忌入与化忌冲必须分开

> 来源：`extractions/xu-book3/ch2-wealth-career.md`
- 忌入可主得，忌冲则多主凶。财帛化忌入田宅主得库财；财帛化忌冲田宅则为财冲破库，主失。财帛化忌入官禄可主赚钱但在周转中；财帛化忌冲官禄则事业财务不利。
- ### 体用方向
- - 由体宫化忌到用宫，纵然好也容易受囚。
- - 由用宫化忌到体宫，较好，得财较实，较不受囚。
- ---

### RC-xu3-ch2-11: 官禄宫看法：本命飞宫看大运，大运飞宫应流年

> 来源：`extractions/xu-book3/ch2-wealth-career.md`
- 本命宫职立太极飞宫，主要应大运；大运宫职立太极飞宫，主要应流年，并归本命盘分判吉凶。
- ### 飞宫铁律
- 1. 要了解大运吉凶得失，用本命宫职飞宫看。
- 2. 要求流年吉凶得失，用大运宫职飞宫归本命盘看。
- 3. 命理服务时，多以大运为枢纽，飞宫垂象归本命盘，以内外分判吉凶，再下应流年。
- ---

### RC-xu3-ch2-12: 求学、考运、退学与官非星象

> 来源：`extractions/xu-book3/ch2-wealth-career.md`
- 论官禄可细分求学、考运、事业、记过退学、官非牢狱。先了解命宫与生年四化，再看大官、大命与相关飞化。
- ### 关键点
- - 求学运：禄优于科，科优于权，权优于忌。
- - 大官或流官见左辅右弼，可能有退学、重修、留级、重考的象。
- - 天刑、文昌、文曲、廉贞化忌，或忌牵动天刑，常与官非、诉讼、记过、牢狱有关。
- - 官禄忌冲命、官、父，再逢昌曲廉忌或天刑，要特别注意官司是非。
- ---

### RC-xu3-ch2-13: 财帛宫立太极，先天为体，后天财帛为用

> 来源：`extractions/xu-book3/ch2-wealth-career.md`
- 论财帛仍以生年四化为体；若先天不明朗，再以本财、大财飞化为后天财帛气数。
- ### 财帛四化功能
| 财帛四化 | 功能 |
| 忌 | 去脉，钱用到何处；先定得失 |
| 禄 | 来龙，财从何处来；论行业与赚钱来源 |
| 权 | 时间、转进、变通 |
| 科 | 质，后续可归入四象一元 |
- ---

### RC-xu3-ch2-14: 财帛化忌入六内与六外的速查

> 来源：`extractions/xu-book3/ch2-wealth-career.md`
| 财帛化忌落宫 | 判断 |
| 命 | 主得，节俭或吝啬，较辛劳，积蓄之财 |
| 财帛 | 主得，用钱赚钱，周转中赚钱 |
| 疾厄 | 主得且多，但财多身弱，或钱花在身体医药 |
| 官禄 | 主得但周转中，投资事业 |
| 田宅 | 主得、实、多，常以置产论 |
| 福德 | 得会钱、利息钱 |
| 迁移 | 冲命，大破财 |

### RC-xu3-ch2-15: 偏财看父母宫

> 来源：`extractions/xu-book3/ch2-wealth-career.md`
- 正财为努力所得，偏财带投机与非努力所得。因奴仆宫为众生位，父母宫是奴仆之财帛宫，所以父母宫可论偏财。
- ### 判断流程
- 1. 本命父母宫看终身有无偏财运。
- 2. 大父化忌入本命六内，主此十年有偏财；入六外则无或不利。
- 3. 先看大父有无，再看流父应哪一年。
- 4. 流父可直接断流月；流父忌入大运内外论得失，最后归本命内外定究竟。
- ---

### RC-xu3-ch2-16: 合伙以子女为用、奴仆为体

> 来源：`extractions/xu-book3/ch2-wealth-career.md`
- 合伙主体宫位在子女宫。子女宫又称合伙位，因为它是奴仆宫的官禄位；以奴仆立太极，奴仆为合伙之体，子女为合伙之用。
- ### 合伙判断
- - 先天合伙：看本命子女宫、生年四化、宫飞本子。
- - 后天合伙：看大子垂象定吉凶。
- - 子女化忌冲奴仆：合伙凶。
- - 子女化忌冲本命：合伙凶。
- - 奴仆之财为父母宫，可用父母宫干四化看合伙事业财务状况。
- ---

### RC-xu3-ch2-17: 大财好不等于大官好，合伙可得财也可能不可自营

> 来源：`extractions/xu-book3/ch2-wealth-career.md`
- 财官必须分开：大财好、大官凶，可能投资赚钱但不宜自己经营；大官好、大财凶，可能帮公司或事业体系赚钱，但自己财运不佳。
- ### 合伙案例抽象
- - 大财忌冲官禄：可有合伙或赚钱机会，但不可自己经营生意。
- - 大官禄入父母或父母系注脚：可作批发、中介、公家机构、血缘地相关财。
- - 合伙宫位有科、禄、权等吉象，仍可能伴随口舌是非；要看是否冲本命、本奴或本官。
- ---

### RC-xu3-ch2-18: 工作异动看大官忌入六阴，吉凶再看法象生年

> 来源：`extractions/xu-book3/ch2-wealth-career.md`
- 工作异动以大官为主。大官化忌入六阴，主一定有异动；若入六阳，则多主静。异动吉凶再看内外与是否法象生年。
- ### 应用
- - 大官忌入六阴：有调动、驿动。
- - 大官忌入六外本来不佳，但若逢自化权，并法象生年权落六内，可转为升迁或有利变动。
- - 应期可取象落点、发射点、自化权宫、生年权宫，以及对应月份。
- - 权忌组的异动常来得快、去得突然。

### RC-xu3-ch5-10: 富贵贫贱先看生年四化，后天成就再看命宫四化

> 来源：`extractions/xu-book3/ch5-ten-essentials-marriage.md`
- 论先天富贵贫贱在生年四化；论后天富贵贫贱在命宫四化。先天若不主贵，后天主贵，多为副手命、二把手，或因后天努力而取得有限度的贵。来因宫提供格局注脚，如来因在夫妻属他力格，可含裙带、妻助、福报、外出机遇。
- 六阳宫主贵，六阴宫主富。禄权科在命财官最重，夫福迁次之；忌冲三合谓破格，忌冲田、疾谓损格。古代星性论富贵贫贱重六吉星，是因四化体系秘而不宣，可作参考但不应替代四化。
- ---

### RC-xu3-ch5-11: 官运与选运看官禄兼看奴仆，胜负取大官与大奴

> 来源：`extractions/xu-book3/ch5-ten-essentials-marriage.md`
- 看选运要看官禄并兼看奴仆。大官代表选情、事业场面、职位本身；大奴代表群众、人和、竞争关系。大官自化禄可见风光与好的一面；若化忌未逢自化，位阶不提升为大奴，未必构成严重不利。
- 大奴若化忌入命并逢自化忌，可能形成苦战；禄忌媒介权落财帛，则需大量开销。若兄弟/奴仆宫化科为质，仍可有群众或组织上的支撑。胜算常须综合大官、大奴与现实政治环境判断。
- ---

### RC-xu3-ch6-06: 财官先天以忌在内外分得失，再以禄权科定富贵格

> 来源：`extractions/xu-book3/ch6-property-cases.md`
- 论财官先看先天：
| 项目 | 用法 |
| 生年忌 | 入六内/六外定吉凶、得失；剖腹产择时也要把忌确定在六内 |
| 生年禄权科 | 分富格还是贵格；六阳主贵、六阴主富 |
| 来因宫 | 确立肯定有还是相对有；以六内六外定自立或他荫 |
- 六内表示自立、自得的程度较高；六外则偏他力、相对有，或受外部条件牵动。
- ---

### RC-xu3-ch6-07: 财官后天看大命是否走生年，再看大财大官入命田疾

> 来源：`extractions/xu-book3/ch6-property-cases.md`
- 后天财官首重大运命宫是否走到生年禄权科忌。生年四化是一定有的现象，大命走到，就进入该现象的十年应用。
- 再看大财、大官。论财官最重要的本命三宫为命、田、疾；大命、大财、大官化忌入本命命田疾，多主得、主有。若是大运六亲宫化忌入本命命田疾，则反作不利于该六亲。
- 论财物，以忌为主，定吉凶得失，以权为变通；论事业、职位、升迁，以权为主，忌为变通转进。忌可定时间，也可定空间；权与忌一者定流年，另一者可定流月。

### RC-xu3-ch6-08: 财官刺出忌主血本无归，人事刺出忌可主分手死别

> 来源：`extractions/xu-book3/ch6-property-cases.md`
- 站在人与物的不同语境，刺出忌转译不同。大财化忌入命、田、疾、福、官、财等宫，对宫若有生年忌或自化忌，构成刺出忌，谓血本无归；若对宫又有生年忌与自化忌，可能还要再拿钱出来。
- 大夫化忌入兄弟宫，兄弟对宫奴仆有生年忌或自化忌，同样构成刺出忌，常解释为分手，严重时可作死别。财官中化忌入福财官也可得，但多在周转、消费、消耗中。
- ---

### RC-xu3-ch6-09: 财官组合以禄科、禄权、禄忌、科忌分行业与格局

> 来源：`extractions/xu-book3/ch6-property-cases.md`
- 财官象组合速查：
| 组合 | 财官倾向 |
| 禄科 | 文教、巧艺、企划、智慧型事业或生意得财 |
| 禄权 | 一般生意、开创、实干经营 |
| 禄忌 | 变动性要快，项目轮换、流通、买卖变化 |
| 科忌 | 上班格、上班性质的命或格 |
- ---

### RC-xu3-ch6-10: 大财化忌入六内逢生年或自化，多仍以得财为主

> 来源：`extractions/xu-book3/ch6-property-cases.md`
- 大财化忌入六内，若逢生年忌，先物以类聚，得财必然发生；再类以群分，也会损财。两者合起来仍可总论赚钱，例如赚五十万但过程亏八万。飞宫忌逢生年忌也会带动来因宫同吉同凶。
- 若逢生年禄权科，则没有“物以类聚”的双忌效应，主要看来因宫跟着吉，并按双象属性解。若大财化忌入六内再逢自化，不论自化禄权科忌，仍以得、吉为主；自化另解钱用到哪里、花在何处、法象哪一生年宫与星。
- ---

### RC-xu3-ch6-11: 批流年先看流命走生年，再用大运飞宫与流命自化细判

> 来源：`extractions/xu-book3/ch6-property-cases.md`
- 批流年先看流命或流迁是否走到生年禄权科忌；走到就一定有现象，不论时间长短或缘浓缘薄。再合来因宫解释：例如流命走生年禄，来因本子，可解为新桃花，也可能是合伙；若再看宫干化禄在哪，就是在地成形。
- 传统批流年步骤：
- 1. 从大运十二宫飞宫。
- 2. 先论吉凶，再论时间。
- 3. 若流年命宫有自化，则该年有变化。
- 4. 看流命飞宫，先化忌，次禄权科。
- 例如大夫化忌冲本夫，若今年流命走到化忌，可把今年定为“婚姻安全年”，并进一步看二月、八月等月份。

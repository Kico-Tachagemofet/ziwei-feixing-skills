## Finding procedure

Step 1: 先看命迁轴定有无交通/出行风险：本命、本迁主此生，大命、大迁主十年；命迁飞化使禄忌交汇于命、疾、父、迁或子女时，列为交通线索。
Step 2: 再以疾厄宫定轻重和伤害类型。疾厄见凶主皮肉伤、骨折、残障等层次；若再合奴疾田凶象，风险等级上升。
Step 3: 查便捷禄忌组合：甲庚太阳禄忌、丙庚天同禄忌、丁辛巨门禄忌，以及己/辛文昌曲忌与对宫禄会合；高风险落点为父母、疾厄、命宫、迁移。
Step 4: 合星曜物相：廉贞忌七杀在迁移、七杀会羊陀、天机忌、巨门、破军、昌曲忌等，用来说明车辆、伤部或血光罚单性质。
Step 5: 应期取禄忌交汇宫、自化宫、同类生年宫，或流命/流疾/流迁/流父走到的年份月份。

## RC 证据摘录

### RC-xu3-ch4-06: 交通意外以命迁定有无，以疾厄定类型，以子女为意外宫

> 来源：`extractions/xu-book3/ch4-health-disasters.md`
- 交通意外先看命宫与迁移宫：命迁组合决定是否有交通意外。疾厄宫进一步判断灾厄类型，如被杀、生病、意外、外伤等。子女宫为意外宫，若与迁移或疾厄发生交易，或重叠宫职并见凶，也可作意外解。
- 本命、本迁主此生；大命、大迁主十年。命迁组合使化禄、化忌交汇于命、疾、父、迁，或交汇于子女宫，通常要特别注意交通意外。
- ---

### RC-xu3-ch4-07: 交通断命诀重在甲庚、丙庚、丁辛禄忌交汇

> 来源：`extractions/xu-book3/ch4-health-disasters.md`
- 交通意外的便捷断命诀，以特定天干组合造成主星禄忌交汇为核心：
| 干组合 | 星曜禄忌 | 高风险落点 |
| 甲庚 | 太阳忌、太阳禄 | 父母、疾厄、命宫、迁移 |
| 丙庚 | 天同禄、天同忌 | 父母、疾厄、命宫、迁移 |
| 丁辛 | 巨门忌、巨门禄 | 父母、疾厄、命宫、迁移 |
| 己/辛文昌曲忌 | 昌曲忌与对宫禄会合 | 可与主星同宫作血光、罚单、车辆象 |
- 若禄忌交汇只在大运父疾命迁，力量较本命盘稍弱，常见为可避免、可减轻，重则残障，未必死亡。若先天有象、后天也有象，再加奴疾田交易，则风险显著提高。
- ---

### RC-xu3-ch4-08: 交通事故轻重看疾厄垂象与星曜象

> 来源：`extractions/xu-book3/ch4-health-disasters.md`
- 交通事故的轻重，不只看命迁，还要看疾厄宫宫干垂象。疾厄见凶，轻则皮肉伤、骨折、扭伤；重则残障、死亡。
- 星曜速查：
| 星象 | 交通/外伤倾向 |
| 廉贞忌与七杀在迁移 | 古称“路旁埋尸”，疾厄再凶尤须保守 |
| 七杀会羊刃/陀罗 | 车祸、骨折、脱臼、残疾 |
| 天机忌 | 不宜骑机车或飙车，手脚骨折、扭伤象 |
| 巨门 | 微卡车、砂石车 |
| 破军 | 货柜车、联结车 |

### RC-xu3-ch4-09: 交通意外的象组合以禄忌、权忌、自化权区分性质

> 来源：`extractions/xu-book3/ch4-health-disasters.md`
- 交通与外伤主题中，象的性质很关键：
| 组合 | 交通/外伤含义 |
| 禄忌 | 起伏大、变化大，常以双忌论 |
| 权忌 | 来得快、去得急，有天意与无奈感，交通尤须注意 |
| 权 | 碰伤、撞伤、跌伤、烫伤等外来伤害 |
| 自化权 | 变化中带破耗，落疾厄/迁移时多指外力伤害 |
| 科 | 解厄制化，生年科最强，飞宫科次之，自化科较弱 |
| 忌 | 不顺、困扰、灾厄、凶兆 |

### RC-xu3-ch4-10: 交通应期多取禄忌交汇宫、自化宫或生年同类宫

> 来源：`extractions/xu-book3/ch4-health-disasters.md`
- 交通事故流年常落在禄忌交汇宫位，至少也常决定流月。若有自化，则自化宫有时间观念，可取自化忌宫、法象生年忌宫，或生年同类宫位变为流命、流疾、流迁、流父时应数。
- 案例中，20 岁车祸以大疾本迁、辛干所得自化忌法象生年忌，并以子午纳气定时；28 岁压断双腿以大命甲、大迁庚形成太阳禄忌交汇，大疾双忌冲命加生年忌，后天变数转为定数。

## 0. 断语门禁

写任何涉及四化的断语之前，先填这个模板：

```
本义检查:
  这里的 ___化，按本义是 ___（填缘义 + 动词）
  在 ___宫（人/事/物哪层）表现为 ___
  常见误读 ___ 不适用，因为 ___
```

**填不出来 → 不准写断语。** 回来查下面各节。

## 1. 禄 — 缘起

**本义一句话**: 缘起，新的开始。聚缘处，取财之源。禄落何宫，该宫即为聚缘处。

**动词**: 聚、起、开。禄是一切美好人事物的源起——有缘、有源头、有好处、有增长。

**不是什么**:
- 不是现成财富。禄是取财之**源**与**缘起点**，不等于手里有钱。
- 不是只有金钱。宫位不主财时，禄按该宫本义取象（人缘、新关系、新事业……）。

**自化禄**: 新缘涌起，变动中有新生。

**对组**: 禄忌同族（因果对立）。禄是因/起点，忌是果/终点。"禄随忌走"——忌定位在哪里，禄的缘就往那里收束。

[RC-vC00-03, RC-b3-01, RC-vA03-01]

## 2. 权 — 缘变

**本义一句话**: 缘变，自然变化。像物竞天择——含天意、无奈、责任、代价。许铨仁原话："无奈的变了，因时间空间原来熟络的变得客气了。"

**动词**: 变、担、承。权使人必须多元开展，不宜只寄托一个人一件事。独立、自立、逆来顺受。

**不是什么**:
- 不是权力（power）。"权"字在这里是权变、权宜、通权达变，不是统治力。
- 不是掌控（control）。权带来的同时也带来压力、摩擦、劳碌、伤害——是双刃剑。
- 不是轻松得来。权靠能力、责任、专业、开创**换来**，不是自然靠近。

**自化权**: 有应期。应期前强求不得，应期到来得突然。天意、突变。

**对组**: 权科同族（互因）。权偏天、偏实质主义；科偏人、偏精神主义。权科互因 = 天人合一。

[RC-vC00-05, RC-xu3-ch5-16, RC-b4-01, RC-vA04-01, xu-book3 ch1 缘分分类]

---

## 3. 科 — 缘续

**本义一句话**: 缘续，已有的人事物境。有恋旧感，承先启后。许铨仁原话："甚至梦境里都有，做梦到一个很熟悉却从没有到过的地方，像是上一辈子去过。"

**动词**: 续、念、回。旧识、念旧、重复、二度、放不下心。也代表荫人/人荫、寿、慧、健康。

**不是什么**:
- 不是暴富或强权。科的优势是稳、顺、名、贵人、细水长流。
- 不是单纯贵人。科深处是心理、思想、健康、旧缘与放不下。

**自化科**: 偏向人的问题，也含健康。

**对组**: 权科同族（互因）。科在人事物上都较能作贵人和解厄。禄的解厄偏健康寿元，科的解厄偏人事。

[RC-vC00-05, RC-xu3-ch5-16, RC-b5-01, RC-vA05-01, xu-book3 ch1 缘分分类]

## 4. 忌 — 缘灭

**本义一句话**: 缘灭、归主。对己有人事物的果报、终点、归藏、定位。许铨仁原话："灭不代表没有了，就像佛家说的因果轮回一样。"

**动词**: 归、收、定。收束、固结、入库。忌是果——再烂也是果，总比没有好。

**不是什么**:
- 不是单纯坏。结束之中可能含有新的生机空间。
- 不等于完全没有。忌代表该处有结果、有牵挂、有代价、有固执。
- 不能望字生义。忌的含义依条件分化——人上牵挂受累，事上不顺反复，物上入库收藏（物上反而可为好事）。

**自化忌**: 渐变，急不来。缘起即缘灭，短暂。

**忌义条件分读**: 必须按 人/事/物 × 入/冲 × 六亲/六事 选忌义。详见 feixing-framework.md §6。

[RC-vC00-04, RC-xu3-ch1-10, RC-b6-01, RC-vA06-01]

## 5. 两条脊柱

### 脊柱一：缘分生命周期

斗数排列（机遇人生）：

```
禄(缘起) → 权(缘变) → 科(缘续) → 忌(缘灭)
```

缘起之后经历缘变无奈，进入化科的怀缅，最后归于忌的尘埃落定。

易经排列（造命学）：

```
禄(缘起) → 科(缘续) → 权(缘变) → 忌(缘灭)
```

缘起之后珍惜其缘、随缘待之，再纵任缘变，终归其位。权科互因是天人合一的核心。

[RC-xu3-ch5-17]

### 脊柱二：四化司令

| 化 | 主类 | 核心动词 | 对组 |
|---|---|---|---|
| 禄 | 人 / 缘起 | 聚、起 | 禄忌同族（因果） |
| 权 | 事 / 权变 | 变、担 | 权科同族（互因） |
| 科 | 人 / 缘续 | 续、念 | 权科同族（互因） |
| 忌 | 物 / 归主 | 归、收 | 禄忌同族（因果） |

[RC-vC00-02]

## 6. 生年×自化 组合矩阵

### 适用范围

此矩阵适用于**同一宫内**生年象与自化象**共存**（RC-vC00-04, RC-xu3-ch1-12）。

### 与双象六组的 scope 区分

| 场景 | 两个象的来源 | 用哪套规则 |
|---|---|---|
| 飞化落 B 宫，B 宫已有生年/自化象 | 一个外来 + 一个宫内既有 | **双象六组** (RC-vA08-07/08) |
| 同一宫有生年象 + 自化象 | 两个都是宫内既有 | **生年×自化组合** (本节) |

**不可混用。** 两套规则结论可完全相反。例：

- 交易象权忌碰撞（双象六组）→ "来急去快 / 一刀两断"
- 生年忌 + 自化权同宫（本矩阵）→ "忌是先天底色，权是有应期的天意突变——应期前强求不得，到了来得突然"

区分方法：先判断两个象的来源——一个从外飞入 vs 两个都是宫内先天配置。

### 生年忌 + 自化

| 组合 | 本义解读 | 本义关键 |
|---|---|---|
| 生年忌 + 自化禄 | 旧象终止后新生，物换星移再新开始 | 忌=旧底色，禄=变动方向(新) |
| 生年忌 + 自化权 | 未完全弃旧，在旧基础上突变创新。应期前强求不得，到了来得突然 | 权=权变/天意，不是"掌控" |
| 生年忌 + 自化科 | 偏向人的问题，也含健康。类似忌+权但更人、更柔 | 科=缘续/念旧 |
| 生年忌 + 自化忌 | 渐变，急不来。人事物皆为渐变 | 双忌不是双倍坏，是慢 |

### 四种变量总纲

| 有无生年 | 有无自化 | 性质 | 关键 |
|---|---|---|---|
| 无 | 无 | 纯飞宫（用） | 必须飞宫才显现象 |
| 无 | 有 | 飞宫 + 变动（用） | 先知用神何组 |
| 有 | 无 | 先天存在（体） | 大限流年坐照才有现象 |
| 有 | 有 | 由体入用 | **先有再变，不可直接说无** |

[RC-vC00-04, RC-vC00-11]

## 7. 星性×四化 交叉提示

四化本义定了"什么在发生"，星性定"发生在谁/什么事上"。同一个化忌，星不同断事完全不同：

- 太阳忌：男性长辈/贵人缘薄，光明受损，眼睛
- 太阴忌：女性/母亲缘薄，情绪低落，妇科
- 天机忌：计划失误，思虑过度，神经系统
- 贪狼忌：欲望落空，嗜好成瘾，桃花劫

**操作**: 四化本义先定"缘起/缘变/缘续/缘灭在哪个层面"，再用星性落实"具体是什么人/事/物"。不可跳过本义直接从星性出断语。

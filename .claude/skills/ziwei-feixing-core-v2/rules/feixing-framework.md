## 1. 断盘总序

断盘分四个阶段。前两个确定性（可查表/读盘面），后两个需要判断力。

### Phase A: 结构读取（确定性，查表）

```
生年四化定象 (先天，全局，只做一次)
  ↓
选太极点 (问题驱动)
  ↓
飞宫四化 (太极点 ↔ 命，用理则一~八)
  ↓
自化标记 (后天变动，有自化才有应期)
```

Phase A 是机械操作：查十天干表、找星位、跑理则。Skill 在这个阶段正确率高。

### Phase B: 信号收集（确定性，读盘面）

每条飞化链完成后，收集该链涉及的**所有信号**。信号是盘面上确定性可读的事实，不是解读。

| 信号类型 | 读什么 | 举例 |
|---|---|---|
| 化星信号 | 飞的是什么星？带什么属性域？ | 天机=变动/计划/驿马；太阴=女性/情绪/妇科 |
| 落宫信号 | 落在什么宫？什么位？ | 夫妻=人位；寅申巳亥=驿马位 |
| 同宫信号 | 宫内还有什么星？组合是什么？ | 武杀组合、机阴组合（先查组合，不拆单星） |
| 结构信号 | 生年象？自化？入/冲？来源区分？ | 生年忌+自化权=先天共存(走组合规则，不走双象六组) |
| 辅星信号 | 天马？桃花星？煞星？ | 天马=驿马加强；桃花星=桃花义加强；煞=外力/伤 |

Phase B 也是确定性的——看盘面就知道。但这是 **义的激活的输入**，必须在判断之前做完。

#### C.0: 五域全列（不可跳过）

**入/冲优先**: 同一条链，入和冲的取义方向完全不同。
例: 忌×贪狼 入 田宅 = 祸福困扰落在家里；忌×贪狼 冲 田宅 = 与家宅无缘。

**主司优先**: 星的主司先于通论属性（sihua-minimal-semantics.md §6）。
例: 贪狼的主司="主祸福"，通论="桃花/欲望/才艺"。忌×贪狼先读"忌×祸福主"。

#### C.2a: 双象六组优先（有双象时走这条）

1. 确认双象组合类型（禄权/禄科/权科/禄忌/权忌/科忌）

#### C.2b: 五域共振（无双象时走这条）

五域的属性在某些方向上重叠——重叠越多，那个方向的权重越高。不重叠的方向权重自然低。这就是取象比类：画面从共振中浮现，不是从公式中算出。

**共振纪律**:

1. **化义×化星属性重叠 = 双重确认**。忌(收束/终止) + 天机(变动) → "终止导致的变动"被确认；忌的其他义(是非/亏欠等)因缺同方向信号而权重低。
2. **重叠是共振不是加法**。不是拼标签("收束+变动=收束的变动")，是两个象在同一个画面上产生共鸣。
3. **低权重 ≠ 排除**。没有共振支持的义自然低，但同 topic 另一条链、命主与太极点的互飞交易、忌冲带来的新信号，都可以把低权重义推回来。
4. **入/冲改变整个共振方向**。忌入的义域(困扰/是非/收藏/管束机制)和忌冲的义域(无缘/缘薄/破坏)几乎不重叠。先定入/冲，再看其他四域。

**忌义区分**: 变动/驿马是忌的固有特质（许铨仁 A06-A07 第一条："主多变动，多变迁，含驿马，变动不安"），不只是下游效果。但忌的变动是"变动不安"（多变动、多变迁），天机的变动是"机巧谋略"（策划、计算）。两者共振时看语境定画面，不能叠成一个标签。

**化×星共振参考**:

| 共振类型 | 举例 | 画面 |
|---|---|---|
| 化忌落囚星 | 忌→廉贞 | 收束方式是困住/不见光 |
| 化禄落桃花星 | 禄→贪狼/天姚 | 缘起方向是感情/欲望 |
| 化科落科星 | 科→文昌/文曲 | 缘续加强=再续前缘/旧识 |
| 化忌落驿马星 | 忌→天机/天马 | 困扰来自不安定/漂泊 |
| 化权落煞星 | 权→擎羊/陀罗 | 变动带外力冲击/摩擦 |

**四档输出**:

| 档位 | 含义 |
|---|---|
| **主取** | 五域共振最集中的方向（≥2 个域指向） |
| **次取** | 有共振但不是最强方向 |
| **低权重** | 该化的义域里有但五域没给共振支持，新信号可推回 |
| **不取** | 与底色/入冲方向矛盾，或该义需要的共振条件完全不具备 |


## 5. 飞宫四化（八大理则）

### 基本结构 [RC-vA08-01]

```
A宫(飞象宫) → 四化 → B宫(落点) → 对宫C
```

### 理则一二合用 [RC-vA08-02]

任何飞宫都要做两步：
1. **理则一（互为阴阳）**: A + B = 象
2. **理则二（各为阴阳）**: 禄权科**照**C，忌**冲**C

两步都做才完整。

### 理则三：本命十一宫忌冲命宫必凶 [RC-vA08-03]

六亲宫忌冲命 → 无缘、缘薄。六事宫忌冲命 → 破财、不利、凶。

### 理则四：同类用冲体必凶 [RC-vA08-04]

大运同名宫忌冲本命同名宫 → 该领域十年亮红灯。实务上最重要的理则。

### 理则五：元太极不可冲 [RC-vA08-05]

任何层级的宫位忌冲本命命宫 → 必凶。流年命宫忌冲本命命宫 = 真正的犯太岁。

### 理则六：小归大 [RC-vA08-06]

飞宫落点同时有本命宫和大运宫身份时，以时间区间大的为准。本命 > 大运 > 流年。

### 理则七八：双象 [RC-vA08-07]

飞宫四化落入B宫，B宫已有生年象(理则七)或自化象(理则八) → 产生双象。

### 双象六组 [RC-vA08-08]

| 双象 | 族 | 财官 | 婚恋 |
|---|---|---|---|
| 禄权 | 不同族 | 多元跨业 | 泛水桃花 |
| 禄科 | 不同族 | 因名得利 | 良性桃花 |
| 权科 | 同族 | 专门技术 | 象二数一 |
| 禄忌 | 同族 | 起伏大 | 通常大凶(有例外) |
| 权忌 | 不同族 | 技术谋生 | 来急去快 |
| 科忌 | 不同族 | 上班格 | 藕断丝连 |

## 6. 四化取义：义域参考表

> ⚠️ v3.0 降级声明：以下义域表为**参考性自检工具**，不是 Phase C 的推导来源。
> Phase C 推导走 C.0→C.6 取象比类流程（见 §1）。义域表只在 C.7 做可选 sanity check。
> 如果 C.6 结论与义域表条目不匹配，**以 C.6 为准**——义域表无法覆盖所有配置组合。

四化各有多个义。取义不是查条件表选一个，是通过 Phase C 取象比类推导。以下表格仅作参考。

### 6.0 总则

1. **信号决定取义**。不是"人位→亏欠，物位→收藏"的机械分配，是看盘面上有什么信号在共振什么义。
2. **多义共存**是常态。条件义 + 通论义并存（如"变动中有牵挂"），不必强选唯一。
3. **通论不主断**。变动、不顺、凶兆都是通论——无特别信号加强时仅作底色，不作主断。
4. **共振加强**。同一个义被多重信号指向时，从底色升为主断。
5. **本义约束**。取的义必须能回扣四化本义（sihua-benyi.md）。回扣不上 → 取义有误。
6. **飞化结构是第一证据**。判断取义时，飞化链结构（禄科照命、忌入/冲哪宫）是最硬的证据。义理文字推演（"忌的定义不含X"）只做辅助校验，不能替代结构证据。例：判断"不缘薄"的最强证据是夫妻飞禄科照命（结构），不是"忌的本义不含缘薄"（文字）。
7. **飞化关系才产 finding**。静态星性分布（某星坐某宫但没有飞化到太极点）不能产飞化级 finding。要用静态星性做证据，必须先找到飞化路径。[2026-05-20 用户校准]
8. **"正面结构"必须说清楚是什么正面**。禄科照命不能停在"正面结构"标签——照命=好处方向回到命主，具体好处是什么（缘起+缘续+旧识+念旧），必须从化×星推导出来说明白。

### 6.5 飞宫化忌操作流程 [RC-vA06-07]

1. 确定从哪个宫位下手
2. 看该宫宫干
3. 查十干四化表，确认哪颗星化忌
4. 找到该星在命盘的位置（星辰负责导气）
5. 陈述：某宫某干使某星化忌 → 忌入某宫
6. 判断入/冲
7. **Phase B 信号收集**（化星/落宫/同宫/辅星/结构信号）
8. **Phase C 信号→义 激活判断**（§6.1 表，看什么信号在共振什么义）
9. 忌冲时记得气讲往来，同时看所冲对宫

## 7. 自化与四种变量

### 自化 = 后天变动

自化 = 某宫有星自己化出去，代表该宫主动发生变动。有自化 → 有应期（配大运/流年定时间）。没有自化 → 静态、先天定好、不容易变。

### 生年忌 + 自化四种组合 [RC-vC00-04]

| 组合 | 解读 |
|---|---|
| 生年忌 + 自化禄 | 旧象终止后新生 |
| 生年忌 + 自化权 | 未完全弃旧，在旧基础上突变创新 |
| 生年忌 + 自化科 | 偏人的问题，含健康 |
| 生年忌 + 自化忌 | 渐变，急不来 |

### 四种变量 [RC-vC00-11]

| 有无生年 | 有无自化 | 性质 | 处理法 |
|---|---|---|---|
| 无 | 无 | 纯飞宫(用) | 必须飞宫才显现象 |
| 无 | 有 | 飞宫+变动(用) | 先知用神何组，看自化出入 |
| 有 | 无 | 先天存在(体) | 合来因宫解象，大限流年坐照才有现象 |
| 有 | 有 | 由体入用 | 先有存在，再看自化造成变化 |

## 8. 信号收集（Phase B 操作）

信号收集在飞化链完成后、义的激活之前。不是断盘的最后一步——是判断的**输入**。

### 信号收集模板

```
化星: [星名] — **主司: [官方身份/角色]** — 属性域: [核心取象面关键词]
落宫: [宫名] — 类型: [人位/事位/物位] — **地支位: [驿马位(寅申巳亥)/桃花位(子午卯酉)/库位(辰戌丑未)/普通]**（必填）
同宫: [双星组合名，或单星] — 组合取象: [关键词]（**落宫组合必读**，组合先于单星）
辅星: [列出天马/桃花星/煞星等] — 信号: [驿马/桃花/外力伤/...] — [wtz]小星(如有): [X]
结构: [生年X / 自化X / 入B / 冲C / 生年+自化共存 / 飞化碰既有象]
```

### 星性属性域

化星是信号的核心来源。"飞的是什么星"决定了这条链能激活什么义。

**组合优先**: 同宫双星必须先查组合取象，不可拆开只查单星。[用户纠正]

星性表来源: xu-book2 Ch4 / video-course B01-B13（ziyun star-meanings/ 尚未接入，接入后作为标注补充）

### 驿马信号

| 信号源 | 具体内容 |
|---|---|
| 驿马位 | 寅、申、巳、亥四宫 |
| 驿马星 | 天马（核心）、天机、破军、太阳、天同、廉贞、武曲（在驿马位时激发） |
| 驿马配置 | 禄存+月马=禄马交驰（出外发迹最强） |

### ⚠️ 星性深度不足

当前星性知识为**通论级**（"天机=变动"），缺**断事级**（"天机忌在夫妻，什么信号下取计划失误 vs 思虑过度 vs 神经系统？"）。

通论级足够做 Phase B 信号收集（"这颗星带变动属性"），但不够做 Phase C 激活判断的精细区分。断事级取象需要从 source 继续蒸馏。见 §11。

## 11. 知识缺口与判断力天花板

Phase A（飞化链）和 Phase B（信号收集）是确定性的，skill 可以做好。Phase C（义的激活判断）是 skill 当前最弱的环节。

### 缺口清单

| 缺口 | 现状 | 需要什么 | 影响 |
|---|---|---|---|
| **星性取象深度** | 通论级（"天机=变动"） | 断事级（"天机忌在夫妻+X信号→取计划失误 vs 思虑过度"） | Phase C 无法做精细激活判断 |
| **双星组合取象** | 有框架，不够深 | 每个组合的断事级取象面（武杀=?、机阴=?、廉贪=?…） | 同宫信号读不深 |
| **辅星信号域** | 散落在各处 | 系统化：天马/桃花星/煞星/空劫各自的信号功能 | Phase B 辅星信号收集不全 |
| **位的信号** | 只有驿马位（寅申巳亥） | 其他位域（辰戌丑未=? 子午卯酉=?）的信号功能 | 落宫信号读不全 |
| **四化×星性 激活矩阵** | 不存在 | 每颗星化忌/禄/权/科时的断事级激活信号表 | Phase C 的核心知识库 |
| **配置模式库** | 不存在 | 什么配置 pattern 激活什么义的 pattern library | Phase C 的判断模板 |

### 判断力天花板

缺口未补之前，Phase C 有天花板：

1. **能做的**: 信号收集 → 列出候选义 → 按通论级知识做粗筛
2. **做不好的**: 精细判断"在这个配置下，这个义比那个义更强"
3. **应该做的**: 遇到不确定时诚实标注 `confidence: low — Phase C 信号不足以区分候选义`，不假装能判断

### 建设路线

```
星性断事级蒸馏（从 source 提取）
  ↓
辅星/位域 信号系统化
  ↓
四化×星性 激活矩阵
  ↓
配置模式库
```

优先级：星性断事级蒸馏是地基，其他都依赖它。**当前来源**：许铨仁体系（xu-book2 Ch4 / video-course B01-B13）。紫云系列星性取象蒸馏尚未接入，接入后须标 [ziyun] 且冲突时许铨仁赢。分析框架（理则/四化表/飞化操作/应期）100% 许铨仁。

# 紫微斗数 Anti-Cliche 规则

> 文体层规则。写每段后过一遍。每条规则后附正例/反例。

## A. 禁 Cookbook 套话

Cookbook = 不经飞化推导直接用星性出断语。

✗ "贪狼在疾厄，所以桃花旺"
✗ "天机在夫妻，配偶聪明但不专一"
✗ "空宫代表没有方向"
✗ "身宫在财帛说明对赚钱上心"
✗ "天马代表跑动"

✓ 从飞化链推导出来的、经过四化本义校验的判断，用人话写
✓ 直接引用星性原书定性改成人话："你事业宫的组合，许铨仁的定性是老谋深算、府城深、动中求财——不靠嘴靠做"
✓ 小星具体角色："天月是病星=先天有点小毛病反复的体质"
✓ 飞化人话："你命宫把困扰的能量送去了朋友圈，同时把人缘送去了外面"

测试：把这句话的星性/宫位名去掉，还有没有信息量？"X在Y，所以Z"如果X换成别的星Z就不成立，才是真判断。

## A2. 禁旺度作正文判据（v1.3 新增）

许铨仁飞星派不主看旺度，旺度有争议。

✗ "贪狼旺在迁移=旺地扛忌，起伏但不倒"
✗ "擎羊庙地=执行力正面发挥"
✗ "天同巨门双不旺=身体底子偏弱"
✗ "太阴落陷=能量虚弱"

✓ 技术依据行里记录旺度作为事实数据（来自文墨天机排盘）
✓ 正文用star本义+宫位+飞化推导做判断，不用旺度修饰

来源：用户校准 2026-05-23 — "我们的体系实际上不主要看旺度，许铨仁是不认这个的，也有争议，别放到文本里"。

## B. 禁抽象属性作主语

主语必须是人/事/状态，不能是占星概念。

✗ "化忌的能量流向家宅方向"
✗ "权变的力量在外部环境中显现"
✗ "缘起的方向指向身体层面"
✗ "忌象的收束性为关系带来了定位效应"

✓ "你容易把感情的纠葛闷在心里消化"
✓ "到了时间点变化就来了，之前催不动"
✓ "赚钱的方式是先争后得，不是天上掉的"

测试：主语是不是一个真实存在的人或者一件真实发生的事？

## C. 禁机制层词汇进正文

这些词只能出现在 `<details>` 技术块：

- 管束、收束、归主、缘灭、缘起、缘变、缘续
- 飞化、飞宫、化气、导气
- 入六内、入六外、冲C
- 由体入用、先天共存
- 象、物、用神、太极点
- 信号激活、义域

## D. 禁模糊泛化

✗ "需要注意" / "值得关注" / "有一定的影响"
✗ "可能会有一些困扰"
✗ "感情方面比较复杂"
✗ "事业上会有挑战"

这些话说了等于没说。要么说具体是什么，要么不写。

✓ "感情里想太多，精神层的纠缠比外在事件更明显"
✓ "赚钱靠争、靠表达、靠嘴，不是坐着等来的"

## E. 禁刚性句式

✗ "……是一种……的表现"
✗ "……体现了……的特质"
✗ "……意味着……"
✗ "从命理角度来看……"
✗ "综上所述……"

这些是八股文句式，写出来读者会走神。

✓ 直接说事："你天生有一股力把注意力拽向感情方向"
✓ 带场景："每次感情有波折，心里都要翻来覆去消化一轮"

## F. 抗重复

同一个判断点在一个 topic 内只写一次。如果多条 finding 指向同一个判断，合并成一段，不要在第1段写一次第3段又换个说法写一次。

跨 topic 可以从不同角度触及同一结构（如迁移宫忌汇聚），但措辞必须不同。

## G. 句子纪律

- 一句话一个判断，不要塞两三个判断进一个长句
- 避免"虽然A但是B同时C而且D"的四段连环句
- 可以用短句（"变化来了。之前催不动。"），但不能全是短句碎片
- 短句和长句交替，保持节奏

## H. 禁"不是A而是B"过度使用

"不是X而是Y"句式偶尔用有力量，连续用就成了套路。一个 topic 最多用一次。

✗ "不是感情差，而是想太多。不是计划失败，而是纠缠放不下。不是外在冲突，而是内心反复。"

✓ 第一次用"不是A而是B"点明核心区分，后面直接写B不再对比A。

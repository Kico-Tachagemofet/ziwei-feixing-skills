# Composition Grammar

## 一句话

小星只做“修饰语”，主星/组合/宫位/四化/流限才是句子的主干。

## 禁止句型

| 错误句型 | 问题 |
|---|---|
| 红鸾 = 幽默有才 | 把组合语境误存成单星标签 |
| 文曲 = 说话绕弯 | 脱离巨门/口舌星语境 |
| 火铃 = 一定有灾 | 脱离宫位、忌煞、流限 |
| 天喜 = 一定结婚 | 脱离夫妻宫和流年承接 |

## 正确句型

```text
[主星/组合基础象] + [小星修饰] + [宫位/四化/流限承接] -> [可用断语]
```

## 检查流程

1. 先写主星/组合基础象：如巨门=口舌、暗处、是非、表达遮蔽；天机巨门=机巧思辨、口才、争执。
2. 再写小星修饰：如文曲修饰表达为文雅、曲折、文艺气；红鸾修饰表达为讨喜、幽默、带情感吸引。
3. 再写宫位承接：命、福、官、财、夫、迁等决定这段表达落在哪个生活域。
4. 再写四化/流限：禄权科忌与大限/流年决定吉凶和时间层。
5. 若任何一层缺失，降级为 background 或待核。

## 场景合成 (v2 新增)

当满足以下**任一条件**时，在逐颗检查之后，必须做场景合成:
- 同宫/强关联内有 **≥2 个小星或功能星**（如三台八座+解神=2组信号）
- 同宫/强关联内有 **1 个小星组 + 1 个功能星或主星仪仗场景**（如三台八座+紫微=小星组+主星）
- 跨轴成对的小星组（如三台在A宫+八座在对宫B宫）视为同组，合并后再判断是否触发

1. **全列**: 把同宫所有小星+主星+辅星列出
2. **查古义**: 每颗星的古义/典故/物象是什么？(如三台=三公, 八座=八抬轿, 华盖=帝王伞盖)
3. **合成画面**: 这些古义放在一起画的是什么场景？(如三台八座+华盖+紫微=帝王加冕排场)
4. **功能星标注**: 年解/天解等"功能型"小星标注场景的功能(解厄/喜庆/变动等)
5. **与主星合参**: 场景+主星四化=事件性质(如+破军禄=破旧立新的加冕)

### 组合场景速查

| 组合 | 场景 | 条件 |
|---|---|---|
| 三台八座 + 解厄星(年解等) | **仪式** (有排场的解厄仪式) | 排场+解厄=仪式 |
| 三台八座 + 华盖 + 紫微 | **帝王加冕/出巡排场** | 帝王+伞盖+仪仗 |
| 三台八座 + 华盖 + 解厄星 + 破军禄 | **破旧立新的加冕仪式，解开束缚** | 完整画面 |
| 龙池凤阁 + 三台八座 | **有舞台/排场的才艺展演** | 才艺+排场 |

此表随 case 校准持续扩展。

## 校准样例

| 组合语境 | 可用取象 | 禁止误存 |
|---|---|---|
| 巨门 + 红鸾同宫/强关联 | 巨门口舌象被红鸾修饰，可出现幽默、有才、讨喜表达 | 红鸾单独=幽默有才 |
| 巨门 + 文曲同宫/强关联 | 巨门表达象被文曲修饰，可出现文雅、绕弯、文艺化表达 | 文曲单独=绕弯 |
| 天机 + 巨门对宫/同宫语境 | 思辨、奔波、争执、口才清晰，需看权忌和宫位 | 只看巨门单星停住 |

## 输出要求

每条小星信号必须带：

- `base_combo`: 主星或组合基础象。
- `modifier`: 小星修饰了什么。
- `carrier`: 宫位/四化/流限承接。
- `claim`: 最终断语。
- `boundary`: 缺少哪层时不能推出什么。

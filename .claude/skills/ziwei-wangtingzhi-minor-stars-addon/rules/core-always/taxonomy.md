# 小星 Taxonomy

## 核心分类

| 类别 | 星曜 | 速用信号 |
|---|---|---|
| 辅曜 | 左辅、右弼、天魁、天钺 | 扶助、贵人、旁力、成对更稳 |
| 佐曜 | 文昌、文曲、禄存、天马 | 文书才艺、财禄、移动、远方 |
| 四煞 | 火星、铃星、擎羊、陀罗 | 外力、冲突、伤损、急烈、阻滞 |
| 六煞扩展 | 四煞 + 地空、地劫 | 煞性叠加空耗、破散 |
| 空曜 | 地空、地劫、天空、截空、旬空 | 空耗、落空、虚位；截空/旬空较弱 |
| 刑曜 | 擎羊、天刑 | 刑伤、手术、官非、规则压迫 |
| 忌曜 | 化忌、陀罗 | 牵滞、拖延、困住、旧债 |
| 桃花诸曜 | 红鸾、天喜、咸池、大耗、天姚、沐浴 | 人缘、欲望、情感牵动、享乐消耗 |
| 文曜 | 化科、文昌、文曲、天才、龙池、凤阁 | 文书、名声、才艺、表达 |
| 科名诸曜 | 文曜 + 三台、八座、恩光、天贵、台辅、封诰、天官、天福 | 名位、扶抬、正式认可、贵显装饰 |
| 孤寡类 | 孤辰、寡宿 | 孤独、六亲疏离、情感独行 |
| 杂曜 | 蜚廉、阴煞、天哭、天虚、破碎等 | 细节修饰，须依组合使用 |

## 操作原则

- 先看主链：宫位、四化、飞化、自化、流限。
- 再看主星/组合：小星必须修饰某个主星、组合、宫位或飞化链，不能独立贴标签。
- 再看小星：它们说明事件的质地、方式、人物、形相。
- 成对星优先看“是否成对”；单星尤其要小心父母宫、夫妻宫等六亲宫。
- 三方四正、对宫、借星安宫会改变小星是否可用。
- 流曜只在对应流限层使用，不能写死为本命固定事实。
- 若一个取象只在“某小星 + 某主星/组合”时成立，必须存为 combo/card，不可塞回单星卡。

## 例证类型

本索引只记录可迁移的结构，不搬运长段案例。

| 模式 | 可迁移结构 | 读法 |
|---|---|---|
| 百官朝拱 | 多组辅佐/科名星成对拱照 | 可增强名位、社会承认、被扶抬 |
| 见星寻对 | 遇成对星先找另一颗 | 确认完整助力还是单星缺配 |
| 父母/夫妻单星 | 辅佐成对星只见一颗 | 六亲宫关系可能不完整、两重或偏侧 |
| 天梁羊刑 | 天梁与擎羊/天刑相会 | 疾病、手术、官非题的高敏信号 |
| 昌曲叠忌 | 昌曲化忌或流昌流曲叠忌 | 文书、票据、合同、名誉、破财风险 |
| 桃花耗星叠会 | 红鸾天喜、咸池、大耗、天姚叠会 | 人缘/情欲/精神享受与消耗并见 |
| 禄马相会 | 禄存与天马会合 | 动中求财、远方资源、迁移财机 |
| 巨门表达被小星修饰 | 巨门与文曲/红鸾强关联 | 巨门口舌象被文雅、讨喜、幽默、曲折表达修饰 |

## 使用方式

- 先从小卡判断是否命中。
- 若命中且要入断，读对应 deep combo。
- 输出时标 `[wtz-example-pattern]`，并说明是结构类比，不是照抄案例。
- 用户校准的组合语境标 `[kico-calibration]`，不要塞回单星卡。

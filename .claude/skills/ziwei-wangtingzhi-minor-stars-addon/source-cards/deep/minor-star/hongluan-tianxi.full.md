# 红鸾天喜 Deep

## 核心定位

- 红鸾天喜是**年支系星**，安星法决定**必定在对宫成对**（不是飞星派小星，不参与四化，不走 wtz composition-grammar）。
- 鸾喜偏喜庆、人缘、礼仪性关系事件、感情触发。
- 流年见之**不等于必婚**，要看夫妻宫、福德宫、命宫、流限与主链是否承接。

## 按所在宫位断姻缘形态（三合派经典技法）

⚠️ **常见误区**: 以为"红鸾天喜要在夫妻/福德这条姻缘主线上才管用"——**错**。鸾喜在任何宫位都成立, **不同位置断的是"姻缘在什么情境下发生 / 这段关系是什么形态"**, 不是"鸾喜不在主线就不算"。

|---|---|---|
| **命宫** | 命主本人有桃花气场/招感性吸引; 自身就是吸引人的状态 | 命宫主星+辅星色彩 |
| **夫妻宫** | 直接姻缘发生位; 流年走到大概率有姻缘事件 | 夫妻宫飞化结构 |

| **子女宫** | 参加娱乐活动/共同创造类活动认识(派对/兴趣班/工作坊/合伙项目); 通过 fun/玩 触发 | 桃花星(贪狼/廉贞/咸池等)合参 |
| **疾厄宫** | 性吸引力强的关系/肉体上的喜欢/身体感应型连结; 关系带强烈生理面 | 疾厄主星+桃花星 |

### 使用规则

1. **必合参主星+桃花星**: 鸾喜只是"形态/情境"的标签, 具体性质取主星(命/夫主星) + 桃花星(贪狼/廉贞/咸池/天姚) 决定。例: 红鸾在子女+贪狼=娱乐+欲望型; 红鸾在子女+紫微=共同创造的高格活动。
2. **对宫必有另一颗**: 红鸾↔天喜必成对(年支系星), 所以"鸾喜所在宫"实际是**对轴**——读时两宫一起看。
3. **跟夫妻宫合参看完整姻缘画面**: 鸾喜定"情境/形态", 夫妻宫定"配偶质感+缘分深浅"。两层一起读才完整。
4. **不替代夫妻宫主结构**: 鸾喜是 modifier 不是主轴。婚姻 topic 主结构仍在夫妻宫飞化↔命的 Layer 1 交互。

### 常见配置示例

- **红鸾在子女+贪狼+咸池**: 通过娱乐/玩乐场合, 带欲望/桃花质的认识方式
- **红鸾在父疾线+天梁老人星@夫妻+父科飞夫妻**: 三线汇聚 → 长辈介绍 / 配偶年长 / 传承绑定配偶 (Z03-kico 案例)
- **红鸾在官禄+太阳巨门**: 工作场合/公开场合相识, 可能带表达/争论性质
- **红鸾在迁移+天马**: 出外/远行/异地缘, 驿马型

## 与桃花叠会规则

- 与**咸池、大耗、天姚**叠会时, 桃花性质增强 → 走 `minor-combo/taohua-stack` 处理
- 与**忌煞**同会时, 喜中带耗或关系麻烦 → 礼仪性场合伴有破耗/纠纷信号

## 输出提示

- `joy_signal`: celebration / relationship_trigger / social_attention / background
- `relationship_form`: (按所在宫填) social_party / work_intro / family_intro / overseas / etc
- `marriage_claim_allowed`: only with spouse-palace and timing support
- **不可单凭鸾喜位置断婚姻成败**, 只断"情境/形态"; 婚姻吉凶看夫妻宫飞化↔命主结构

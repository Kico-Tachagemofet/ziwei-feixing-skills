# Topic Lens Reference: 路人缘 / 粉丝缘

> Calibrated: 2026-05-20
> Source: 首案例 v1 人工校对 + 校准

## Topic Definition

命主与不认识的人（路人、粉丝、网民、一面之交）之间的缘分、互动方式、以及这层关系对命主的影响。

## Main Taiji

- 宫位: **交友宫**
- 理由: 交友宫=十二宫中人数最多的宫（路人/众生/粉丝/网民/一面之交），论人为"不熟的人"。[RC-b2 宫位语义 + taiji-point-selection]
- lookup keywords: `交友宫`, `奴仆宫`, `众生`, `路人`, `粉丝`

**关键区分**：
- 交友 = 路人/一面之交（不认识或只见过一面）
- 兄弟 = 至亲好友/发小/闺蜜（熟的）
- 迁移 = 外出环境/场域（不是人群）

## Secondary Taiji

| 宫位 | 用途 | 权重 | 限制 |
|---|---|---|---|
| 命宫 | 看命主与路人的互飞关系（命↔交友互飞忌是常见核心结构） | high | 只看命与交友之间的飞化互动，不扩展到命宫本身的通论 |
| 子女宫 | 作品、娱乐、角色、桃花式投射——粉丝缘的表达出口 | medium | 只看交友→子女方向的飞化，不主断子女宫本身的事 |
| 财帛宫 | 看粉丝缘是否变现（交友→财帛方向） | low | 只看变现可能性，不主断"赚钱方式"整体 |
| 官禄宫 | 看粉丝缘是否推动事业/名声（交友→官禄方向） | low | 只看推动方向，不主断事业格局整体 |

## Supporting Palaces

| 宫位 | 只能怎么用 | 不能怎么用 |
|---|---|---|
| 迁移宫 | 理则二结构记录中的 C 宫（交友忌入命冲迁移时，记录气的走向） | 不能独立断"出外不利"或"路人影响外出" |
| 兄弟宫 | 交友宫的对宫，做对宫参照；理则二结构记录中的 C 宫 | 不能独立断"因路人与熟人疏远" |
| 福德宫 | 交友禄照福德时记录照射方向 | 不能主断精神状态 |

## Banned Misreads

- **禁止**: 交友忌入命冲迁移 → 直接断"外出不利" / "路人影响出外"。理则二冲 C 是结构记录不是断语，只有理则三/五冲命才独立成断。[校准 2026-05-20 + eight-rules.md]
- **禁止**: 命忌入交友冲兄弟 → 直接断"因路人与至亲好友疏远"。同上，冲兄弟不独立成断。
- **禁止**: 把"人位忌入人位"断为"管束"。忌入的机制是管束（A干涉B），但断语层写困扰、是非、纠缠、执念。"管束"听起来像化权。[RC-vA06-03 + 校准]
- **禁止**: 单星 cookbook——不看武曲七杀同宫的武杀组合，直接拿武曲单星的"财损"断。组合取象必须先于单星。[star-meanings-skeleton §4]
- **禁止**: 副链（财帛/官禄方向）抢主链。路人缘的核心矛盾在交友↔命的互飞忌，不在"路人给命主带来多少钱"。
- **禁止**: Finding 层做综合一句话。综合叙事是 render 层的事。

## Chain Priority

### Primary Chains
- 交友宫静态读取（主星组合、辅星、生年/自化状态）
- 交友干飞忌 → 命宫（路人对命主的忌象）
- 命干飞忌 → 交友宫（命主对路人的忌象）
- 命 ↔ 交友互飞忌综合结构

### Secondary Chains
- 交友干飞禄 → 财帛（路人带来的财缘方向，只看变现可能性）
- 交友干飞权 → 官禄（路人对事业的推动方向，只看推力性质）
- 交友干飞科 → 子女（路人缘的贵人/续缘方向，看作品/角色投射）

### Background Chains
- 生年四化全局定象（先天格局，不主断路人缘但提供底色）
- 理则三/五冲命检查（通论凶相，非交友专属）
- 双象检查（各落点的生年/自化碰撞）

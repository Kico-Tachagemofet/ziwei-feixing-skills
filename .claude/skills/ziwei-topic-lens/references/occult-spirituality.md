# Topic Lens Reference: 神秘学 / 玄学 / 宗教 / 命理 / 供奉

> Calibrated: 2026-05-22
> Source: 用户校准 (多信号 checklist 型)

## Topic Definition

命主与神秘学/玄学/宗教/命理/灵性修行的关系——是否有体质、是否有兴趣、是否适合从事、是否需要供奉、供奉什么方向。

论: **事**（不是论人。论的是"命主与神秘学/宗教的关系"这件事。）

## Scope 注意

本 topic 涵盖：
- 宗教信仰倾向
- 命理/五术学习能力和兴趣
- 灵性体质/第六感
- 供奉/设坛需求
- 神秘学从业可能性

**不涵盖**：心理咨询（走 career）、哲学学术研究（走 learning，如有此 topic）。

## 操作优先级

**先有物再找象。** 实际操作中，通常是命主已经有宗教/供奉/玄学行为，命理师在盘上找对应信号。不是"盘上有信号→你应该供奉"，是"你已经在供奉→盘上哪里呼应"。

找象的顺序：
1. 命主已有的玄学/宗教行为是什么（供奉/学命理/修行/灵性感应）
2. 在盘上找信号群对应——**不是单个信号下结论，是信号叠加定方向**
3. 信号落在哪条轴/哪个宫群 → 玄学活动在生活中落实的方向

## 太极点选择

本 topic **不是单太极型**。没有一个宫位能完整覆盖"神秘学/宗教"。应走**多信号 checklist**：多个宫位和信号共同构成判断，不以某一宫为 Main Taiji。

如果必须选一个主入口：

- 宫位: **福德宫**（精神面/内在世界/宗教倾向的第一落点）
- 理由: 宗教/玄学首先是精神面的事，福德宫是精神面的太极点。
- 但: 福德宫只看精神倾向，不看体质（疾厄）、传承（父母）、外在表现（命迁）。

## 格局 Checklist

### 天巫赶马格

- **条件**: 天巫 + 天马 + 禄存 三星会齐（同宫或三方会照）
- **位置**: 必须在四马地（寅申巳亥）
- **约束**: 不能有空劫冲破（否则禄逢冲破，吉处藏凶）
- **效果**: 加速做法事/宗教活动/奔波供奉; 有为宗教奔走的缘
- **飞星重构**: 看天巫所在宫的宫干飞化方向 → 宗教活动对命主哪个面有影响

#### 实例: 校准案例盘供奉信号 (校准案例)

信号群集中在夫官线:
- 官禄(壬戌): 三台[旺] + 解神[庙] + 阴煞 + 华盖[平] — 四颗小星堆一宫
- 夫妻(丙辰): 天梁[庙](宗教星) + 八座[旺] — 三台八座跨轴成对
- 父母(己)化科=天梁→夫妻 — 传承的延续/旧缘飞向夫官线，化星=宗教星
- 福德(庚申)生年科(右弼) — 精神面先天有延续
- 天巫在财帛/身宫(甲寅) — 身宫近五术
- 迁移: 天同+太阴 — 太阴(命理之星)被天同(多/福)放大，命理涉猎多，在迁移=外在面呈现灵性/命理气质

→ 供奉落在夫官线, 落实到生活=供奉对象与夫妻宫方向相关(灵体 Landlord)。阴煞信号=供奉对象为阴性灵体。

## Banned Misreads

- **禁止**: "福德宫化科=一定要供奉" — 化科落福德是信号，不是结论。要看化星是不是宗教星+来源宫+其他信号叠加。单一信号不成立。
- **禁止**: "华盖在命=一定是修行人" — 华盖是方外/孤高信号，不自动等于修行。看主星承载和其他信号叠加。
- **禁止**: "天巫在命=必须供神" — 天巫近宗教/五术，但"必须供"要多信号叠加。
- **禁止**: 只用三合（位置→结论）判断。飞星派必须追飞化链：化科是谁飞的、忌指向哪里、禄随忌走方向。
- **禁止**: 把"孤辰寡宿=出家"当 cookbook 断 — 孤辰寡宿在现代语境更多是独处/独行偏好，不自动等于出家。

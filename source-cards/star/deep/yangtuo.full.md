## RC-xu3-ch6-12: 火灾必先田宅见凶，并合火铃羊陀与大田本田禄忌/权忌

**provenance**: [许-高级班]

断火灾，必然先田宅宫见凶象。火灾一定与火星有关，多在本田或大田；火星、铃星、羊刃、陀罗虽不参与四化，但有实质妙用。案例中，大田权忌入本田，平衡先天权忌，又田宅有火星，判断当年须注意火灾。

火灾口诀：大田与本田各有宫干，若大田宫干与本田宫干造成禄忌交汇于某宫或某星，同时大田/本田有火星，或禄忌交汇宫有火星，就要注意火灾。大田本田禄忌交汇于同星，比同宫更显著。

**guardrail**: 火灾判断应转化为现实检查：电路、瓦斯、厨房、香烛、办公室消防，不要只停留在断语。

## RC-xu3-ch4-08: 交通事故轻重看疾厄垂象与星曜象
**use-when**: 命迁已有交通象，需判定血光、残障或事故性质
**source**: PDF pp257, 260 / printed pp253, 256
**provenance**: [许-高级班]

交通事故的轻重，不只看命迁，还要看疾厄宫宫干垂象。疾厄见凶，轻则皮肉伤、骨折、扭伤；重则残障、死亡。

星曜速查：

|---|---|

**guardrail**: 星曜只给“象”，事故现实形态还要由环境、行为、安全措施决定。

## RC-xu3-ch4-11: 症状论有生年有自化时，优先取生年星为主症
**use-when**: 健康象有生年星、自化星并见，需要判断症状部位或系统
**source**: PDF pp264, 269-270, 280, 291 / printed pp260, 265-266, 276, 287

| 巨门 | 脾胃，以胃为主；也可指器官系统、哮喘等 |
| 天相 | 皮肤、膀胱系统 |
| 天机 | 肝胆，以胆为主 |
| 天同 | 肝胆，以肝为主；也可作气衰、疝气等语境 |
| 太阳 | 脑疾、高低血压、头晕头痛、眼疾 |
| 廉贞 | 疑难杂症、肿瘤癌症、糜烂性疾病 |
| 右弼/左辅/昌曲 | 附加、并发症 |

## RC-xu3-ch4-12: 手术瘫痪案例显示生年科可保命，自化权可恶化
**use-when**: 手术、瘫痪、治疗失败、残障但未死亡的案例分析
**source**: PDF pp268-270 / printed pp264-266
**provenance**: [许-高级班]

李佩青案例中，大疾本奴宫有生年科、自化权，并会紫微、天相、文昌、陀罗。动态以大疾宫干壬四化论，大疾忌冲本疾而逢生年科。原文判断为身体运逢凶，但生年科可解厄制化，所以手术失败、残废，却未死亡。

飞宫已定凶，再逢自化权，解释为恶化；自化权与忌形成权忌，来得急去得快。应期取自化权宫或生年权宫变为流命、流疾时。

**guardrail**: 这类卡片只抽取命例推理链，不可外推为医疗预后判断。

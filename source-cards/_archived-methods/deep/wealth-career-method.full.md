| 路径 | 解义 |
| 大运夫妻化忌冲本命夫妻 | 这十年成婚不易，或夫妻姻缘悔吝，严重则生离死别 |
| 大运官禄化忌冲本命官禄 | 这十年学业、事业、工作、官运不利，升迁难，可能变动失业 |
| 大运疾厄化忌冲本命疾厄 | 这十年健康或交通意外见凶象 |
- ### 理则五：任何宫职化忌冲本命命宫必见凶象
- 任何宫职包括本命十一宫、大运十二宫、流年命宫。命宫为宫职原体，任何宫职化忌冲原体必见凶象。许铨仁称此为入门重要断命诀，学员戏称“一忌走天涯”。

### RC-xu2-ch3-12: 飞宫理则六：宫职由小归大碰撞

### RC-b3-03: “有其象必有其物”是生年禄落宫的落地法

> 来源：`extractions/b-series/b3-lu-detailed.md`
- 生年四化落宫不是虚象。原文强调“有其象必有其物”：生年禄落在哪个宫，该宫所代表的人、事、物一定会以某种方式进入命主生命。
- ### 读取步骤
- 1. 确定禄是否为生年禄。
- 2. 确定禄落入的本宫语义。
- 3. 先按“该宫一定有缘、有源、有增益”取象。
- 4. 再按宫位细分：人、事、财、身体、工作、场所、精神状态。
- ### 示例逻辑
- - 禄在夫妻宫：不只代表婚姻好，也代表配偶、异性缘、外出工作位有资源或有形/无形之财。

### RC-b3-05: 禄的应用边界与交叉取象

> 来源：`extractions/b-series/b3-lu-detailed.md`
- ### 三个稳定关键词

- 2. **美好**：该宫领域较容易呈现顺、善、舒服、受欢迎。

### RC-b6-02: 忌要分人、事、物三层取象

> 来源：`extractions/b-series/b6-ji-detailed.md`
- ### 论人
- 忌在人上主缘薄、牵挂、受累、固执、三观不合。落六亲宫时，往往不是完全无缘，而是“很挂心、很难轻松相处、越想帮越容易好心办坏事”。

### RC-b6-05: 忌的三条核心法则与职业财务边界

> 来源：`extractions/b-series/b6-ji-detailed.md`
- ### 法则一：忌是果
- 忌代表该处一定有结果，只是结果带代价、固结、不顺或牵挂。不能把忌等同于空无。
- ### 法则二：忌在人牵挂受累
- 忌落六亲宫，多是操心、受累、缘薄、关系不轻松。好心容易办坏事。
- ### 法则三：忌在物入库兜底

### RC-b7-06: 理则二——大运宫位化忌冲本命命宫 = 凶

> 来源：`extractions/b-series/b7-flying-palace-liqi.md`
- ### 规则
- 理则一的时间延伸：大运的任何宫位化忌入本命迁移宫冲本命命宫 → 同样是凶。
- ### 核心原理
- **气由大向小走。**

- ### Worked Example
- - 42-51岁大运

### RC-b7-07: 理则三——大运化忌冲本命同名宫位 = 凶（后天冲先天）

> 来源：`extractions/b-series/b7-flying-palace-liqi.md`
- ### 规则
- 大运某宫化忌入本命对宫、冲本命同名宫位 → 该宫位这十年为凶。后天（运）冲先天（命），即用冲体。
- ### 操作
- 1. 确定要看的事项 → 找到对应的大运宫位
- 2. 看大运该宫位天干飞出的忌落在哪个**本命**宫位
- 3. 若忌落在该宫位的**本命对宫** → 构成"后天冲先天"
- ### Worked Example 1: 大运夫妻冲本命夫妻
- - 35-44岁大运，大运夫妻宫天干壬

### RC-b7-09: 多线索锁定法

> 来源：`extractions/b-series/b7-flying-palace-liqi.md`
- ### 规则
- 判断一件事至少要有**两到三条线索**同时指向，才可锁定。
- ### Worked Example: 26-35岁大运问婚姻
| 线索 | 路径 | 结论 |
| 线索一 | 大运交友宫（庚）化天同忌入本命迁移 → **冲命宫** | 交友运凶，易交损友/被骗财 |
| 线索二 | 大运命宫（丁）化巨门忌入本命迁移 → **冲命宫**；且大运命宫与本命夫妻宫重叠 | 这十年整体不顺，尤其夫妻不顺 |
| 线索三 | 大运夫妻化忌入本命官禄 → **冲本命夫妻** | 后天夫妻冲先天夫妻，婚姻凶象确认 |
- 两条以上线索指向婚姻不利 → **锁定：这十年结婚难，或婚后易出问题。**

## RC-xu3-ch1-08: 化权的核心是掌控、开创、外伤与升迁

化权主权柄、威严、占有、掌控、领导、性刚、才华专技、开创斗志。人际宫见权，多见争执摩擦；身体/人位见权，也有外来碰撞伤害的意味。

## RC-xu3-ch1-22: 阴阳事项有不同表达伦理
**use-when**: 面对寿元、婚姻、桃花、福德等敏感事项
**source**: PDF pp44-46 / printed pp40-42
**provenance**: [许-高级班]

许铨仁把事项分阴阳：阳为生不带来、死不带走，如财官，可较直断；阴为生会带来、死会带走，如婚姻、桃花、寿元、福德，可讲细，但不宜明断到刺激或误导当事人的程度。

### 咨询原则

- 敏感事项要先确认来问者与当事人的关系、动机和事实背景。
- 命理是相对论，不是绝对论；基准点不同，同一气数对应的现实量级不同。
- 问得越清楚，断语越能落地；答非所问时，只能保持在较泛的层级。

**guardrail**: 寿元、重病、婚姻生离死别等事项，不应脱离来问动机与伦理直接明断。

## RC-xu3-ch2-06: 格和局是两回事，生年忌是财官转变分水岭
**use-when**: 判断一生命运转折、忌前忌后、先天后天限度
**source**: PDF pp151-154 / printed pp147-150
**provenance**: [许-高级班]

格如命，局如运。先天格局决定上限，后天努力可改善运局但不必然改变命格。财官一生的转变，以生年忌所落宫位为分水岭，分为忌前与忌后。

### 财官转变法

1. 未过生年忌前，即使大命、大财、大官好，也受先天限制。
2. 过生年忌后，重点看生年禄权科的大运/流年走到或照到。
3. 已过生年忌后，论财官以大命、大官为主，大财为辅。
4. 官禄以化权为主，忌为变通；财帛以化忌为主，权为转进。

**guardrail**: 先天不贵而后天主贵，往往晚成且不易登顶。不要只凭一段好运断终身大格。

## RC-xu3-ch2-10: 化忌入与化忌冲必须分开

**source**: PDF pp159-160 / printed pp155-156
**provenance**: [许-高级班]

忌入可主得，忌冲则多主凶。财帛化忌入田宅主得库财；财帛化忌冲田宅则为财冲破库，主失。财帛化忌入官禄可主赚钱但在周转中；财帛化忌冲官禄则事业财务不利。

## RC-xu3-ch5-16: 禄为缘起、忌为终点、权为天意无奈、科为延续旧缘
**use-when**: 解释四化的哲学含义与人事转译
**source**: PDF pp322-324 / printed pp318-320
**provenance**: [许-高级班]

四化哲学定义：

| 四化 | 高级班定义 |
|---|---|
| 禄 | 缘起、起点、新欲望、新事业、新人事物 |
| 忌 | 因缘终点、结束、尘埃落定 |
| 权 | 独立、实质主义、责任、天意与无奈，处变、逆来顺受 |
| 科 | 延续、旧缘、念旧、重复、二度、承先启后、贵人、解厄制化 |

科在婚姻中含二度、重复，辅弼和化科对婚姻有“再一次”意味；科也有恋旧、放不下、隐衷。科在六内偏荫人，科在六外偏人荫；若科再自化忌，可能出现荫人不得善果或得人福荫但心态不同。

**guardrail**: 四化不是吉凶标签。禄也会缘尽，科也会障碍，权也会无奈，忌也有归结与成形。

## RC-xu3-ch6-02: 单象自化单象要分本宫自化与视同自化的箭头重心

## RC-xu3-ch6-07: 财官后天看大命是否走生年，再看大财大官入命田疾

| 禄忌 | 变动性要快，项目轮换、流通、买卖变化 |
| 科忌 | 上班格、上班性质的命或格 |

星曜辅助：武曲为大财星，吉可大富，凶可大败；破军财为不露白、不可说、典当流通之财；天同为服务性质之财。破军禄在命迁，主外出顺利；在大命大迁则十年外出顺利。

**guardrail**: 组合给行业“气质”，不是具体行业清单；仍需合宫位、来因、星曜和时代环境。

## RC-xu3-ch6-11: 批流年先看流命走生年，再用大运飞宫与流命自化细判
**use-when**: 年度咨询、财官婚姻安全年、当年重点提示
**source**: PDF pp358-359 / printed pp354-355
**provenance**: [许-高级班]

批流年先看流命或流迁是否走到生年禄权科忌；走到就一定有现象，不论时间长短或缘浓缘薄。再合来因宫解释：例如流命走生年禄，来因本子，可解为新桃花，也可能是合伙；若再看宫干化禄在哪，就是在地成形。

传统批流年步骤：

1. 从大运十二宫飞宫。
2. 先论吉凶，再论时间。
3. 若流年命宫有自化，则该年有变化。
4. 看流命飞宫，先化忌，次禄权科。

例如大夫化忌冲本夫，若今年流命走到化忌，可把今年定为“婚姻安全年”，并进一步看二月、八月等月份。

**guardrail**: 流年提示宜转为安全月、修身养性月、风险管理建议，而非恐吓。

## RC-xu3-ch6-15: 买卖房子以田宅为体，兄弟与疾厄为用
**use-when**: 判断买房、卖房、不动产交易、产权纠纷
**source**: PDF pp365-366 / printed pp361-362
**provenance**: [许-高级班]

买卖房子以田宅宫为体，以兄弟宫与疾厄宫为用。兄弟宫是田宅之财帛位，疾厄宫是田宅之官禄位。买房子多发生在兄奴线；卖房子多发生在父疾线。若田宅本体有象，就先看体；体没有象，才看用。

田宅宫自化忌可解释为会有房子，但过程不顺，如钱不够、权状不清、产权纠纷；也可表示买房前常搬家。田宅、兄弟的象落在寅申巳亥，常应买房。自化禄主过程较顺利、轻松；科出也可触发买卖，走到对应流田、流月时应。

**guardrail**: 不动产规则应配现实尽调：产权、贷款、合同、登记、税费、验屋和法律文件。

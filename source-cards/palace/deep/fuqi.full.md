## RC-xu3-ch1-22: 阴阳事项有不同表达伦理
**use-when**: 面对寿元、婚姻、桃花、福德等敏感事项
**source**: PDF pp44-46 / printed pp40-42
**provenance**: [许-高级班]

许铨仁把事项分阴阳：阳为生不带来、死不带走，如财官，可较直断；阴为生会带来、死会带走，如婚姻、桃花、寿元、福德，可讲细，但不宜明断到刺激或误导当事人的程度。

### 咨询原则

- 敏感事项要先确认来问者与当事人的关系、动机和事实背景。
- 命理是相对论，不是绝对论；基准点不同，同一气数对应的现实量级不同。
- 问得越清楚，断语越能落地；答非所问时，只能保持在较泛的层级。

**guardrail**: 寿元、重病、婚姻生离死别等事项，不应脱离来问动机与伦理直接明断。

## RC-xu3-ch3-09: 第一次结婚必须看本夫和大夫
**use-when**: 判断第一次结婚时点
**source**: PDF pp244-245 / printed pp240-241
**provenance**: [许-高级班]

第一次结婚一定要看本夫与大夫；后续婚姻则可直接看大夫。本夫视同自化禄，常表姻缘早发但未必早婚；是否成婚还要看化忌名分、化权媒介与化科最后质。

### 成婚结构示例

- 夫妻宫视同自化禄：有缘。
- 夫妻宫化忌入父母：有名分。
- 夫妻宫化权入命：作为禄忌媒介，事件发生。
- 夫妻宫化科入田宅：最后质为不会离婚，若凶则可能是配偶死亡而非离婚。

**guardrail**: 只见视同自化禄，只能说姻缘早发或有缘；三象/四象齐备才可进入结婚定事与定时。

## RC-xu3-ch3-13: 用冲体必凶，命冲夫与夫冲命要分责任来源
**use-when**: 判断离婚、生离、缘薄的主因
**source**: PDF pp246-247 / printed pp242-243
**provenance**: [许-高级班]

**guardrail**: 少用“刑克”直接吓人，可解释为缘薄、价值观差距、关系难以常驻。

### 科忌姻缘

科忌组的姻缘往往纠缠不清；离婚后可能借故回来，藕断丝连。

**guardrail**: 分居不必然离婚。自化科造成的变化，可能因科的性质而保持风度、不恶言相向，不可一概断绝。

## RC-xu3-ch3-15: 权忌组婚姻：先判有无，再用忌入内外定真假与成立

- 大夫化权逢自化权：姻缘有成立象。
- 再看化忌飞往何处：入六内则成立，入六外则不稳或难成。

## RC-xu3-ch3-16: 福德宫可看因缘果报与婚外缘分

## RC-xu3-ch5-10: 富贵贫贱先看生年四化，后天成就再看命宫四化

**source**: PDF pp301-303, 313-314 / printed pp297-299, 309-310

## RC-xu3-ch5-13: 共用宫与活盘：某宫被六亲借用后不再只属于命主
**use-when**: 六亲宫飞化入命、疾、夫等宫位，需要判断“谁”的命或身体
**source**: PDF pp314-315 / printed pp310-311
**provenance**: [许-高级班]

宫位可被六亲借用，称为共用宫或活盘应用。例如夫妻宫化象入命，夫妻与命宫交易，命宫就成了配偶可借用的命宫，表示“有夫妻的命”。同理，子女宫自化权若以疾厄为权科互因的科位，疾厄宫可成为子女的疾厄，用来解释子女身体。

## RC-xu3-ch5-14: 自化权含破耗，权科互因决定破耗落在哪一类人事物
**use-when**: 某宫有自化权或视同自化权，需要解释破耗与对象
**source**: PDF pp314-315 / printed pp310-311
**provenance**: [许-高级班]

任何宫位有自化都代表变化，唯独自化权特别含破耗，因为权本身带自变性，不必自化就会变。破耗在哪一类人事物，要看化科所在，因为权科互因；权为果则科为因，科为果则权为因。

例如子女宫自化权，可论子女、性生活、合伙、学生缘。若太阴科在疾厄，则破耗可能落在身体；若文曲科入夫妻，则可能落在异性缘、夫妻桃花或关系破耗。

**guardrail**: 自化权不等于一定灾凶。它先表示变化中有破耗，需按宫位主题与化科落点定位。

## RC-xu3-ch5-16: 禄为缘起、忌为终点、权为天意无奈、科为延续旧缘
**use-when**: 解释四化的哲学含义与人事转译
**source**: PDF pp322-324 / printed pp318-320
**provenance**: [许-高级班]

四化哲学定义：

| 四化 | 高级班定义 |
|---|---|
| 禄 | 缘起、起点、新欲望、新事业、新人事物 |
| 忌 | 因缘终点、结束、尘埃落定 |
| 权 | 独立、实质主义、责任、天意与无奈，处变、逆来顺受 |
| 科 | 延续、旧缘、念旧、重复、二度、承先启后、贵人、解厄制化 |

科在婚姻中含二度、重复，辅弼和化科对婚姻有“再一次”意味；科也有恋旧、放不下、隐衷。科在六内偏荫人，科在六外偏人荫；若科再自化忌，可能出现荫人不得善果或得人福荫但心态不同。

**guardrail**: 四化不是吉凶标签。禄也会缘尽，科也会障碍，权也会无奈，忌也有归结与成形。

## RC-xu3-ch5-19: 先天气数组取法须随主题立太极，婚姻可屈就夫妻宫组
**use-when**: 生年四化不明显成组，或论婚姻/财官需转太极
**source**: PDF pp329-330, 339-340 / printed pp325-326, 335-336
**provenance**: [许-高级班]

先天气数组取法：

1. 同宫、本对优先。
2. 邻宫取象。
3. 四象分散时，先看六内宫，再看哪一宫有生年又有自化。
4. 论具体主题时，可改以主题宫立太极。

因此，论自己以命宫为原太极，可能是权忌组；论婚姻时以夫妻宫立太极，夫妻宫若有生年禄，就可改作禄忌组；夫妻宫生年科自化权，则论婚姻可屈就权科组。财官同理，应依财帛、官禄主题转太极。

**guardrail**: “屈就”不是任意换组，而是主题宫有生年、自化或明显宫职主导时才成立。

## RC-xu3-ch5-20: 结婚预测要承认概率分布，不作绝对同年同月论
**use-when**: 多个可能结婚年同时成立，需要输出预测范围
**source**: PDF pp328-331, 338-340 / printed pp324-327, 334-336
**provenance**: [许-高级班]

预测结婚应先认识在数与化气，再看夫妻宫、本夫、大夫、来因与先天气数组。若夫妻宫生年禄再自化科忌，表示缘浓、缘早现，但科忌拖延，要走过忌年、科年才结婚。若夫妻宫生年科自化权，为权科同组，时空俱在，可给 23、25、27、30 等不同可能点。

**guardrail**: 结婚预测不是单点绝对化。更稳妥的输出是主应年、备选年、对应月份与成立条件。

## RC-xu3-ch5-23: 有生年有自化时，先生年后自化，体不变而用生吉凶
**use-when**: 同一宫有生年四化又有自化，需排序解释
**source**: PDF pp348-350 / printed pp344-346
**provenance**: [许-高级班]

有生年有自化的特征是：现象一定存在，且体不变；配合大限、流年时，由体入用，出现先后、次序、时空出入与吉凶。解释顺序为先说生年四化所拥有的质，再说自化造成的变化。若生年宫再有自化忌，该宫一律有不利面；若生年忌所在宫再自化禄权科忌，须法象生年同类象，再以内外判吉凶。

同组同类象即使是单象，也作双象解，如科自化科，为象一数二。双象自化双象，尤其生年科忌再自化科忌，必有变故或刑伤；在夫妻宫可指姻缘变故，在疾厄语境可指身体或精神困扰。

**guardrail**: 不要先看自化就改写生年本质。生年是体，自化是后来的用与变化。

## RC-xu3-ch6-02: 单象自化单象要分本宫自化与视同自化的箭头重心
**use-when**: 夫妻、官禄、命宫等出现单一生年象与单一自化象
**source**: PDF pp351-352 / printed pp347-348
**provenance**: [许-高级班]

单象自化单象要看箭头重心。夫妻宫权自化科在本宫，夫妻宫是人的问题；若夫妻宫生年权，化科到对宫视同自化科，箭头指向官禄，则重心转为事的问题，兼有人。

同组同类象如生年权又自化权，单象作双象解，象一数二。落夫妻宫时，可解配偶能干、性刚、果决；再自化权为加强实质，但也有反其意，过刚易折。

**guardrail**: 本宫自化与视同自化都可成立时间观念，但解释重心不同；要看箭头指向本宫还是对宫。

## RC-xu3-ch6-08: 财官刺出忌主血本无归，人事刺出忌可主分手死别
**use-when**: 化忌入宫冲对宫，对宫有生年忌或自化忌
**source**: PDF pp356-357 / printed pp352-353
**provenance**: [许-高级班]

站在人与物的不同语境，刺出忌转译不同。大财化忌入命、田、疾、福、官、财等宫，对宫若有生年忌或自化忌，构成刺出忌，谓血本无归；若对宫又有生年忌与自化忌，可能还要再拿钱出来。

大夫化忌入兄弟宫，兄弟对宫奴仆有生年忌或自化忌，同样构成刺出忌，常解释为分手，严重时可作死别。财官中化忌入福财官也可得，但多在周转、消费、消耗中。

**guardrail**: 刺出忌的“血本无归/死别”是高强度断语，必须有完整结构才可使用，并宜转为风险提示。

## RC-xu3-ch6-11: 批流年先看流命走生年，再用大运飞宫与流命自化细判
**use-when**: 年度咨询、财官婚姻安全年、当年重点提示
**source**: PDF pp358-359 / printed pp354-355
**provenance**: [许-高级班]

批流年先看流命或流迁是否走到生年禄权科忌；走到就一定有现象，不论时间长短或缘浓缘薄。再合来因宫解释：例如流命走生年禄，来因本子，可解为新桃花，也可能是合伙；若再看宫干化禄在哪，就是在地成形。

传统批流年步骤：

1. 从大运十二宫飞宫。
2. 先论吉凶，再论时间。
3. 若流年命宫有自化，则该年有变化。
4. 看流命飞宫，先化忌，次禄权科。

例如大夫化忌冲本夫，若今年流命走到化忌，可把今年定为“婚姻安全年”，并进一步看二月、八月等月份。

**guardrail**: 流年提示宜转为安全月、修身养性月、风险管理建议，而非恐吓。

## A01 骨架：生年禄落宫

> 来源：`E:\My Grimoire\紫微斗数\视频课收录\Note\A01_十二宫职的含义_note.md`

| 角度 | 解象 |
| :--- | :--- |
| 象一宫诀 | 配偶人缘好，异性缘不错，含桃花；有福禄、食禄、财禄；聪明有才艺 |
| 中太极（缘） | 我与配偶**伉俪情深** |
| 中太极（财） | 借配偶之力，结婚后财禄食禄福禄彰显——俗话「**妻带财来**」 |
| 姻缘类型 | 自由恋爱而结合 |
| 姻缘早花 | 懂两性关系之后，青春不留白，日子都是彩色的 |

> [!note] 妻带财来的两种形式
> - **无形之财**：结婚后生意一直好、升迁很快（升官发财）
> - **有形之财**：别墅4000多万 + 现金800万零用钱

> [!warning] 逻辑前提
> 「借配偶之力」需要**有配偶可借**。未婚者此吉象尚未应验，结婚之后方显。

## RC-xu3-ch1-10: 化忌七义与六内六外得失
**use-when**: 解释忌象的具体含义
**source**: PDF pp57-59 / printed pp53-55
**provenance**: [许-高级班]

化忌可分为七类含义：

| 忌义 | 操作含义 |
|---|---|
| 收束 | 人位自化忌可为收敛，疾厄可见瘤，命宫可见痣 |
| 管束 | 人位化忌到人位，常为管束对方 |
| 收藏 | 财帛忌入六内主得，入六外主失 |
| 亏欠 | 生年忌坐六亲宫，有亏欠感 |
| 不顺 | 事与愿违、坎坷、纠缠 |
| 缘薄 | 人位忌冲人位，聚少离多或价值观差距 |
| 凶兆 | 严重时可到疾病、灾厄、死亡 |

### 财帛用法

财帛忌入六内主得；命、田、疾为实得或多，财、官、福为周转中。怎么得财看禄，时间看权。

**guardrail**: 忌入不是一律凶。财帛、田宅、疾厄等物事宫必须先判六内六外和得失，再论吉凶质量。

## RC-xu3-ch2-10: 化忌入与化忌冲必须分开
**use-when**: 判断忌入六内是否仍吉，或为何忌冲凶
**source**: PDF pp159-160 / printed pp155-156
**provenance**: [许-高级班]

忌入可主得，忌冲则多主凶。财帛化忌入田宅主得库财；财帛化忌冲田宅则为财冲破库，主失。财帛化忌入官禄可主赚钱但在周转中；财帛化忌冲官禄则事业财务不利。

### 体用方向

- 由体宫化忌到用宫，纵然好也容易受囚。
- 由用宫化忌到体宫，较好，得财较实，较不受囚。

**guardrail**: 不要把“忌入六内主得”扩展到“忌冲六内也主得”。冲体位、库位时，凶性明显。

## RC-xu3-ch5-02: 批流年有两条主线：流命四化与大运十二宫归本命

## RC-xu3-ch5-09: 分居离婚看大夫垂象示凶、妻财变化与旧新交替

**source**: PDF pp300-301 / printed pp296-297

分居离婚案例中，本夫自化权表示配偶能干、自信，但若大夫化忌冲本夫，则大运婚姻垂象示凶。自化禄权法象生年禄权，可用于定结婚或分居的月份；大财本夫有自化，因妻财本有对待，有自化就有变化，表示旧的结束、新的开始。

若下一大运大夫继续垂象示凶，就可判断离婚节点。案例中，以大夫巨门忌、自化禄、视同自化权法象生年禄权，定分居月份和离婚年份。

**guardrail**: 分居不必然离婚。要看大夫、大田、妻财、绝情位和后续大运是否继续示凶。

## RC-xu3-ch5-17: 科是健康代号，健康寿元最怕禄出科出
**use-when**: 健康、寿元、命疾田中出现科、自化科或禄科出
**source**: PDF pp324-325 / printed pp320-321
**provenance**: [许-高级班]

科对已有之人事物才产生作用；生命为已有之体，因此科代表寿与慧，也可作健康代号。化科与化禄同在福禄体系中，论健康寿元最怕禄出、科出。

命宫、疾厄、田宅有科自化科或科自化禄，最怕走生年禄限。若某宫自化禄又化忌到生年科，或自化忌又化禄到生年科，落在大命、大疾可指命主健康；落在六亲宫则可能指该六亲健康或往生，至少为健康问题。

**guardrail**: 科在一般语境有解厄、风度、智慧；只有进入健康寿元语境，禄出科出才变为高风险。

## RC-xu3-ch6-03: 单象自化双象要分开合象，常有两件事或两类物相
**use-when**: 一宫有生年单象，又自化或飞出两个象
**source**: PDF pp352 / printed p348
**provenance**: [许-高级班]

田宅生年禄自化科忌，重点先站在物与家运；若箭头指向子女，则重心可转向子女、桃花、驿马、合作。解释时要把禄+科、禄+忌分开看：禄+科是一人一事一物，禄+忌也是一人一事一物，合起来就是两件事、两个人或两样物。

例：田宅坐禄，化科忌到子女，合伙可能不错，但合伙后易有桃花或精神面困扰。

**guardrail**: 不要把单象自化双象压扁成一个结论。需要分组合象后再综合。

## RC-xu3-ch6-12: 火灾必先田宅见凶，并合火铃羊陀与大田本田禄忌/权忌
**use-when**: 判断住宅、办公室、家宅火灾风险
**source**: PDF pp361-362 / printed pp357-358
**provenance**: [许-高级班]

断火灾，必然先田宅宫见凶象。火灾一定与火星有关，多在本田或大田；火星、铃星、羊刃、陀罗虽不参与四化，但有实质妙用。案例中，大田权忌入本田，平衡先天权忌，又田宅有火星，判断当年须注意火灾。

火灾口诀：大田与本田各有宫干，若大田宫干与本田宫干造成禄忌交汇于某宫或某星，同时大田/本田有火星，或禄忌交汇宫有火星，就要注意火灾。大田本田禄忌交汇于同星，比同宫更显著。

**guardrail**: 火灾判断应转化为现实检查：电路、瓦斯、厨房、香烛、办公室消防，不要只停留在断语。

## RC-xu3-ch6-13: 盗窃也先田宅见凶，巨门煞辅弼组合为传统偷盗象
**use-when**: 判断家宅、仓库、财物被盗
**source**: PDF pp362-363 / printed pp358-359
**provenance**: [许-高级班]

盗窃与火灾一样，先要田宅宫见凶象。飞宫上可见大田本田、大田大命或田宅相关禄忌交汇；星辰上常与巨门有关，巨门为遗弃之星。古法称，本田或大田，或大命走到本田，有巨门、煞星羊陀，再有左辅右弼，容易遭偷盗。

象义组合：巨门主遗弃、暗处；羊陀有强行进入之意；左辅右弼在田宅主忙忙碌碌。合起来即有人在家中忙忙碌碌偷盗。被盗两车猪案例中，大田巨门忌入本田，逢生年权自化禄，禄权忌三象成局，并以自化禄应流田。

**guardrail**: 盗窃判断用于提醒防盗、保险、监控、门锁与库存管理；不可据此指认具体嫌疑人。

## RC-xu3-ch6-14: 子女被害、绑架、离家须合参子女宫与奴仆宫凶象
**use-when**: 判断子女安全、被带走、绑架、强暴、离家出走等高风险主题
**source**: PDF pp363-364 / printed pp359-360
**provenance**: [许-高级班]

白冰冰女儿案例中，大子化忌冲本子，十年子女运不好；化禄在兄弟宫，三合指示流年。大子化忌入本田，本田有生年科又自化科，若只有生年科可有解厄，但再自化科形成科出，随科出而凶。

原文总结：绑架要看命主本人的奴仆宫与子女宫合参见凶；被强暴同理，是坏的奴仆宫加坏的子女宫。相同命盘未必都应极端事件，也可能表现为子女被带走后返回、夜不归宿、离家出走等不同层级。

**guardrail**: 这类内容必须以安全为先。现实中应转为子女安全、出行、联络、报警与心理支持，不可把高风险断语说成命定。

## RC-xu3-ch6-15: 买卖房子以田宅为体，兄弟与疾厄为用
**use-when**: 判断买房、卖房、不动产交易、产权纠纷
**source**: PDF pp365-366 / printed pp361-362
**provenance**: [许-高级班]

买卖房子以田宅宫为体，以兄弟宫与疾厄宫为用。兄弟宫是田宅之财帛位，疾厄宫是田宅之官禄位。买房子多发生在兄奴线；卖房子多发生在父疾线。若田宅本体有象，就先看体；体没有象，才看用。

田宅宫自化忌可解释为会有房子，但过程不顺，如钱不够、权状不清、产权纠纷；也可表示买房前常搬家。田宅、兄弟的象落在寅申巳亥，常应买房。自化禄主过程较顺利、轻松；科出也可触发买卖，走到对应流田、流月时应。

**guardrail**: 不动产规则应配现实尽调：产权、贷款、合同、登记、税费、验屋和法律文件。

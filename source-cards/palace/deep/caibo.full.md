## RC-xu3-ch1-06: 禄随忌走，但单象自化忌例外
**use-when**: 处理禄忌组合、职业适性、事项因果
**source**: PDF pp18, 57, 65, 82 / printed pp14, 53, 61, 78
**provenance**: [许-高级班]

禄忌为因果组。一般情况下，忌代表结果或收束，禄代表原因、行业、缘起，因此“禄随忌走”：见忌要追禄，以说明因何而忌、靠何而得或失。

### 例外

单象自化忌时，许铨仁提示有时要倒过来作“忌随禄走”，因为自化本身还有时间与变化的结构，不能机械套一般飞宫。

**guardrail**: 财帛断得失常以忌为主，但“怎么得/怎么失”仍要追禄。职业适性也常由忌的宫位与禄的星曜、宫职共同决定。

## RC-xu3-ch1-08: 化权的核心是掌控、开创、外伤与升迁
**use-when**: 解释权象或官禄化权
**source**: PDF pp40-50 / printed pp36-46
**provenance**: [许-高级班]

化权主权柄、威严、占有、掌控、领导、性刚、才华专技、开创斗志。人际宫见权，多见争执摩擦；身体/人位见权，也有外来碰撞伤害的意味。

### 应用重点

- 断升迁、掌权、事业开创，重点看权。

**guardrail**: 只要是人位，权常归该宫位的人所有；不要自动把权都归命主。

## RC-xu3-ch1-10: 化忌七义与六内六外得失
**use-when**: 解释忌象的具体含义
**source**: PDF pp57-59 / printed pp53-55
**provenance**: [许-高级班]

化忌可分为七类含义：

| 忌义 | 操作含义 |
|---|---|
| 收束 | 人位自化忌可为收敛，疾厄可见瘤，命宫可见痣 |
| 管束 | 人位化忌到人位，常为管束对方 |
| 收藏 | 财帛忌入六内主得，入六外主失 |
| 亏欠 | 生年忌坐六亲宫，有亏欠感 |
| 不顺 | 事与愿违、坎坷、纠缠 |
| 缘薄 | 人位忌冲人位，聚少离多或价值观差距 |
| 凶兆 | 严重时可到疾病、灾厄、死亡 |

### 财帛用法

财帛忌入六内主得；命、田、疾为实得或多，财、官、福为周转中。怎么得财看禄，时间看权。

**guardrail**: 忌入不是一律凶。财帛、田宅、疾厄等物事宫必须先判六内六外和得失，再论吉凶质量。

## RC-xu3-ch1-16: 万象归一元分“入”与“出”
**use-when**: 生年四化与同类自化互动
**source**: PDF pp78-80, 90 / printed pp74-76, 86
**provenance**: [许-高级班]

当命宫、大运命宫或流年命宫走到某个生年四化时，其余十一宫的同类自化都归入该宫，称为“入”。例如大命逢生年忌，大财自化忌，可解为忌入而主得财，仍须按忌的属性说明财型。

若反过来大命逢自化忌，则生年忌所在宫位被带动而不稳定，可称为“出”，常表现为该生年宫职的震荡或损失。

**guardrail**: 入多作有、成局、主得；出多作不稳定、带出、震荡。不能把同类自化一律视为凶。

## RC-xu3-ch1-22: 阴阳事项有不同表达伦理
**use-when**: 面对寿元、婚姻、桃花、福德等敏感事项
**source**: PDF pp44-46 / printed pp40-42
**provenance**: [许-高级班]

许铨仁把事项分阴阳：阳为生不带来、死不带走，如财官，可较直断；阴为生会带来、死会带走，如婚姻、桃花、寿元、福德，可讲细，但不宜明断到刺激或误导当事人的程度。

### 咨询原则

- 敏感事项要先确认来问者与当事人的关系、动机和事实背景。
- 命理是相对论，不是绝对论；基准点不同，同一气数对应的现实量级不同。
- 问得越清楚，断语越能落地；答非所问时，只能保持在较泛的层级。

**guardrail**: 寿元、重病、婚姻生离死别等事项，不应脱离来问动机与伦理直接明断。

## RC-xu3-ch2-06: 格和局是两回事，生年忌是财官转变分水岭
**use-when**: 判断一生命运转折、忌前忌后、先天后天限度
**source**: PDF pp151-154 / printed pp147-150
**provenance**: [许-高级班]

格如命，局如运。先天格局决定上限，后天努力可改善运局但不必然改变命格。财官一生的转变，以生年忌所落宫位为分水岭，分为忌前与忌后。

### 财官转变法

1. 未过生年忌前，即使大命、大财、大官好，也受先天限制。
2. 过生年忌后，重点看生年禄权科的大运/流年走到或照到。
3. 已过生年忌后，论财官以大命、大官为主，大财为辅。
4. 官禄以化权为主，忌为变通；财帛以化忌为主，权为转进。

**guardrail**: 先天不贵而后天主贵，往往晚成且不易登顶。不要只凭一段好运断终身大格。

## RC-xu3-ch2-10: 化忌入与化忌冲必须分开
**use-when**: 判断忌入六内是否仍吉，或为何忌冲凶
**source**: PDF pp159-160 / printed pp155-156
**provenance**: [许-高级班]

忌入可主得，忌冲则多主凶。财帛化忌入田宅主得库财；财帛化忌冲田宅则为财冲破库，主失。财帛化忌入官禄可主赚钱但在周转中；财帛化忌冲官禄则事业财务不利。

### 体用方向

- 由体宫化忌到用宫，纵然好也容易受囚。
- 由用宫化忌到体宫，较好，得财较实，较不受囚。

## RC-xu3-ch2-15: 偏财看父母宫

## RC-xu3-ch5-19: 先天气数组取法须随主题立太极，婚姻可屈就夫妻宫组

## RC-xu3-ch6-06: 财官先天以忌在内外分得失，再以禄权科定富贵格
**use-when**: 建立财官论断的先天层
**source**: PDF pp355 / printed p351
**provenance**: [许-高级班]

论财官先看先天：

| 项目 | 用法 |
|---|---|
| 生年忌 | 入六内/六外定吉凶、得失；剖腹产择时也要把忌确定在六内 |
| 生年禄权科 | 分富格还是贵格；六阳主贵、六阴主富 |
| 来因宫 | 确立肯定有还是相对有；以六内六外定自立或他荫 |

六内表示自立、自得的程度较高；六外则偏他力、相对有，或受外部条件牵动。

**guardrail**: 财官格局不能只看禄权科，忌在内外与来因宫是格局的底层注脚。

## RC-xu3-ch6-07: 财官后天看大命是否走生年，再看大财大官入命田疾
**use-when**: 判断某大运财官成败、赚钱、升迁或事业变化
**source**: PDF pp356 / printed p352
**provenance**: [许-高级班]

后天财官首重大运命宫是否走到生年禄权科忌。生年四化是一定有的现象，大命走到，就进入该现象的十年应用。

再看大财、大官。论财官最重要的本命三宫为命、田、疾；大命、大财、大官化忌入本命命田疾，多主得、主有。若是大运六亲宫化忌入本命命田疾，则反作不利于该六亲。

论财物，以忌为主，定吉凶得失，以权为变通；论事业、职位、升迁，以权为主，忌为变通转进。忌可定时间，也可定空间；权与忌一者定流年，另一者可定流月。

**guardrail**: 大命兼有财官与六亲面向，需按大命重叠的本命宫职判断哪一面好、哪一面受损。

## RC-xu3-ch6-08: 财官刺出忌主血本无归，人事刺出忌可主分手死别
**use-when**: 化忌入宫冲对宫，对宫有生年忌或自化忌
**source**: PDF pp356-357 / printed pp352-353
**provenance**: [许-高级班]

站在人与物的不同语境，刺出忌转译不同。大财化忌入命、田、疾、福、官、财等宫，对宫若有生年忌或自化忌，构成刺出忌，谓血本无归；若对宫又有生年忌与自化忌，可能还要再拿钱出来。

大夫化忌入兄弟宫，兄弟对宫奴仆有生年忌或自化忌，同样构成刺出忌，常解释为分手，严重时可作死别。财官中化忌入福财官也可得，但多在周转、消费、消耗中。

**guardrail**: 刺出忌的“血本无归/死别”是高强度断语，必须有完整结构才可使用，并宜转为风险提示。

## RC-xu3-ch6-09: 财官组合以禄科、禄权、禄忌、科忌分行业与格局
**use-when**: 财官飞化成组，需要判行业性质与工作模式
**source**: PDF pp357 / printed p353
**provenance**: [许-高级班]

财官象组合速查：

| 组合 | 财官倾向 |
|---|---|
| 禄科 | 文教、巧艺、企划、智慧型事业或生意得财 |
| 禄权 | 一般生意、开创、实干经营 |
| 禄忌 | 变动性要快，项目轮换、流通、买卖变化 |
| 科忌 | 上班格、上班性质的命或格 |

星曜辅助：武曲为大财星，吉可大富，凶可大败；破军财为不露白、不可说、典当流通之财；天同为服务性质之财。破军禄在命迁，主外出顺利；在大命大迁则十年外出顺利。

**guardrail**: 组合给行业“气质”，不是具体行业清单；仍需合宫位、来因、星曜和时代环境。

## RC-xu3-ch6-10: 大财化忌入六内逢生年或自化，多仍以得财为主
**use-when**: 大财化忌入命田疾等六内，并遇生年或自化
**source**: PDF pp358 / printed p354
**provenance**: [许-高级班]

大财化忌入六内，若逢生年忌，先物以类聚，得财必然发生；再类以群分，也会损财。两者合起来仍可总论赚钱，例如赚五十万但过程亏八万。飞宫忌逢生年忌也会带动来因宫同吉同凶。

若逢生年禄权科，则没有“物以类聚”的双忌效应，主要看来因宫跟着吉，并按双象属性解。若大财化忌入六内再逢自化，不论自化禄权科忌，仍以得、吉为主；自化另解钱用到哪里、花在何处、法象哪一生年宫与星。

**guardrail**: 财务中“忌入六内主得”不适用于六亲或精神面；同一结构在六亲宫可能不利人。

## RC-xu3-ch6-11: 批流年先看流命走生年，再用大运飞宫与流命自化细判
**use-when**: 年度咨询、财官婚姻安全年、当年重点提示
**source**: PDF pp358-359 / printed pp354-355
**provenance**: [许-高级班]

批流年先看流命或流迁是否走到生年禄权科忌；走到就一定有现象，不论时间长短或缘浓缘薄。再合来因宫解释：例如流命走生年禄，来因本子，可解为新桃花，也可能是合伙；若再看宫干化禄在哪，就是在地成形。

传统批流年步骤：

1. 从大运十二宫飞宫。
2. 先论吉凶，再论时间。
3. 若流年命宫有自化，则该年有变化。
4. 看流命飞宫，先化忌，次禄权科。

例如大夫化忌冲本夫，若今年流命走到化忌，可把今年定为“婚姻安全年”，并进一步看二月、八月等月份。

**guardrail**: 流年提示宜转为安全月、修身养性月、风险管理建议，而非恐吓。

## RC-xu3-ch6-15: 买卖房子以田宅为体，兄弟与疾厄为用
**use-when**: 判断买房、卖房、不动产交易、产权纠纷
**source**: PDF pp365-366 / printed pp361-362
**provenance**: [许-高级班]

买卖房子以田宅宫为体，以兄弟宫与疾厄宫为用。兄弟宫是田宅之财帛位，疾厄宫是田宅之官禄位。买房子多发生在兄奴线；卖房子多发生在父疾线。若田宅本体有象，就先看体；体没有象，才看用。

田宅宫自化忌可解释为会有房子，但过程不顺，如钱不够、权状不清、产权纠纷；也可表示买房前常搬家。田宅、兄弟的象落在寅申巳亥，常应买房。自化禄主过程较顺利、轻松；科出也可触发买卖，走到对应流田、流月时应。

**guardrail**: 不动产规则应配现实尽调：产权、贷款、合同、登记、税费、验屋和法律文件。

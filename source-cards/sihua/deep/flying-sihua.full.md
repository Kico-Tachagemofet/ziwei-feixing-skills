### RC-xu2-ch3-08: 飞宫四化的“宫-象-宫”公式

> 来源：`extractions/xu-book2/ch3-sihua-rules.md`
- ### 定义
- 飞宫四化为象之用，表后天变数，由用归体，也归生年。飞为显象，伏为生年四化。飞宫用于明当下之机：本命十二宫、大运十二宫、流年命宫都可作为发射宫。
- ### 公式
- `A 宫四化入 B 宫`，等价于：
- - `A 宫 + B 宫 = 禄/权/科/忌`
- - A 宫为发射宫、注脚宫。
- - B 宫为象所在宫。
- - 禄权科入 B 宫影响对宫，称照。

### RC-xu2-ch3-09: 飞宫理则一、二：互为阴阳与各为阴阳

> 来源：`extractions/xu-book2/ch3-sihua-rules.md`
- ### 理则一：互为阴阳说
- 交易的两宫互为阴阳，凭借象假合为一太极，作 `A 宫 + B 宫 = 象` 的解说。
- ### 理则二：各为阴阳说
- 象言往来；当化忌入 B 宫时，会冲 B 的对宫，此时 A 宫与 B 对宫发生对待关系，见吉凶。若是禄权科，则为照，也要看 A 宫与 B 对宫的关系。

### RC-xu2-ch3-11: 飞宫理则四、五：同类宫职与“一忌走天涯”

> 来源：`extractions/xu-book2/ch3-sihua-rules.md`

### RC-xu2-ch3-12: 飞宫理则六：宫职由小归大碰撞

> 来源：`extractions/xu-book2/ch3-sihua-rules.md`
- ### 规则
- 飞宫四化在大运、流年层使用时，宫职要由小归大碰撞。大运宫职与本命宫职同处一宫时，若飞化落入该宫，应优先用较大层级的本命宫职命名，不可只按同层大运宫职解。
- ### 示例
- 女命 16-25 岁大运，以 `|夫|` 看姻缘。`|夫|` 宫干丙使天同化禄，天同在本命官禄宫，同时该宫也是大运 `|奴|`。应表述为：
- - `|夫| 化禄入本官`
- - 或 `|夫| + 本官 = 禄`
- 不可表述为 `|夫| 化禄入 |奴|`，因为 `|夫|` 与 `|奴|` 同为大运层级，本官统管一生事业，是较大层级。

### RC-xu2-ch3-13: 飞宫理则七、八：交易象逢伏象/自化象须碰撞

> 来源：`extractions/xu-book2/ch3-sihua-rules.md`
- ### 理则七：交易象逢伏象须碰撞
- 飞宫四化为交易象、象之用；生年四化为伏象、象之体。飞宫落入有生年四化之宫，用象与体象发生碰撞，构成命与运的交错。
- 示例：`|夫| 化禄入本官`，本官已有生年忌，则 `禄 + 生年忌 = 双忌`，可出现姻缘到来但种下多起伏、多变化的恶缘。
- ### 理则八：交易象逢自化象须碰撞
- 飞宫落入有自化之宫，交易象与自化象碰撞。示例中 `|夫| 化权入本财`，本财自化禄，可解婚后对待易争执摩擦，且与桃花泛滥相关；自化重心在自化宫，而非被重叠的本命宫。

### RC-b1-01: 三种四化象分类框架

### RC-b1-03: 飞宫四化象（后天象/飞星）

> 来源：`extractions/b-series/b1-three-transformation-types.md`
- ### 定义
- 每个宫位都有天干，该天干飞出禄权科忌四道气到其他宫位，据此判断吉凶走向。
- ### 性质
- - 属**运**的范畴
- - 是推断运势、判断吉凶最重要的方法
- - 大运、流年均以飞宫来看
- ### 关键规则
- **空宫不影响飞宫**：空宫虽无星星，但一定有天干。有天干 → 一定有四化 → 一定可以看出吉凶。

### RC-b1-04: 自化象

> 来源：`extractions/b-series/b1-three-transformation-types.md`
- ### 定义
- 一个宫位的天干飞出的四化中，若所需星曜恰好落在**本宫**或**对宫**（同一条线 = 同一公司），即为自化。
- ### 分类
| 类型 | 条件 | 特征 |
| **纯自化** | 四化星落在本宫 | 一次性事件 |
| **视同自化** | 四化星落在对宫（同一条线） | 多了**涌动力**——一波刚平一波又起 |
- ### 六条线（六组对宫）
- 命迁线、兄友线、夫官线、子田线、财福线、父疾线。

### RC-b1-05: 三象层级总纲

> 来源：`extractions/b-series/b1-three-transformation-types.md`
- ### 层级
- 1. **生年四化象（命）**：出生年干决定，能量最强，格局高低已定，永久不变
- 2. **飞宫四化象（运）**：各宫天干飞出四化到其他宫位，推断运势吉凶
- 3. **自化象（运）**：四化落回本宫或对宫，纯自化为单次变化，视同自化多涌动力
- ### 判读顺序
- 先读生年四化定格局 → 再读飞宫四化看运势 → 自化作为变化/触发的补充层。

### RC-b7-04: 禄权科忌吉凶二分

> 来源：`extractions/b-series/b7-flying-palace-liqi.md`
| 四化 | 性质 | 飞入某宫 | 术语 |
| 禄・权・科 | 吉 | 飞入 B 宫 | **照** B 宫 |
| 忌 | 凶 | 飞入 B 宫 | **入** B 宫，**冲** B 的对宫 |
- **铁律: 只要一冲——破、散、离。没有例外，没有好事。**
- ### "入"与"冲"的区分
- - 化忌**入**迁移宫 → 冲的是**命宫**（命宫最惨）
- - 化忌**入**命宫 → 冲的是**迁移宫**（对人不好，对事尚可）
- 命宫是十二宫之体、核心中的核心。核心被冲坏则全盘皆凶。

### RC-b7-05: 理则一——任何宫位化忌冲命宫 = 凶

> 来源：`extractions/b-series/b7-flying-palace-liqi.md`
- ### 规则
- 谁化忌冲命宫，那个宫位就是凶。
- ### 操作
- 1. 看迁移宫里有哪些星
- 2. 逐一查哪个天干会使该星化忌（反查十干四化表）
- 3. 找到那个天干所在的宫位 → 该宫位化忌入迁移宫，冲命宫
- ### Worked Example
- - 迁移宫有廉贞星

### RC-b7-06: 理则二——大运宫位化忌冲本命命宫 = 凶

> 来源：`extractions/b-series/b7-flying-palace-liqi.md`
- ### 规则
- 理则一的时间延伸：大运的任何宫位化忌入本命迁移宫冲本命命宫 → 同样是凶。
- ### 核心原理
- **气由大向小走。**
- - 大运宫位飞出的忌，落点**先看本命宫位**，不看大运宫位
- - 本命层级 > 大运层级，能量由高层次向低层次行进
- ### Worked Example
- - 42-51岁大运

### RC-b7-07: 理则三——大运化忌冲本命同名宫位 = 凶（后天冲先天）

> 来源：`extractions/b-series/b7-flying-palace-liqi.md`
- ### 规则
- 大运某宫化忌入本命对宫、冲本命同名宫位 → 该宫位这十年为凶。后天（运）冲先天（命），即用冲体。
- ### 操作
- 1. 确定要看的事项 → 找到对应的大运宫位
- 2. 看大运该宫位天干飞出的忌落在哪个**本命**宫位
- 3. 若忌落在该宫位的**本命对宫** → 构成"后天冲先天"
- ### Worked Example 1: 大运夫妻冲本命夫妻
- - 35-44岁大运，大运夫妻宫天干壬

### RC-b7-08: 三大理则精简总纲

> 来源：`extractions/b-series/b7-flying-palace-liqi.md`
- ### 总纲
- 1. **理则一（冲命宫）**: 本命盘中，任何宫位化忌入迁移宫冲命宫 → 该宫位为凶。命宫是体、是核心，核心被冲则全盘出问题。
- 2. **理则二（大运冲命宫）**: 理则一的延伸——大运任何宫位化忌入本命迁移宫冲本命命宫 → 同凶。气由大向小走，忌的落点先看本命层级。
- 3. **理则三（后天冲先天）**: 大运某宫化忌入本命对宫、冲本命同名宫位 → 该宫位这十年为凶。后天冲先天 = 用冲体。
- **本质上是两条：一、不能冲命宫；二、后天不能冲先天同名宫位。**

## RC-xu3-ch1-04: 宫象宫公式有三种解法，发射宫决定主动方
**use-when**: 解释 `A宫化禄/权/科/忌到B宫`
**source**: PDF pp20-29 / printed pp16-25
**provenance**: [许-高级班]

`A宫四化入B宫` 可用三种角度解释，取最合象义者：

1. A 把禄权科忌给了 B。
2. 因 B 宫的缘故，使 A 宫禄权科忌。
3. `A宫 + B宫 = 禄/权/科/忌`。

命宫化禄到兄弟，与兄弟化禄到命，结果可能同为兄弟圆融、互得助力；差别在过程。发射宫是谁，谁就是主动方。

**guardrail**: 不要只背“化到某宫”的固定断语。要先认发射宫、受象宫，再判断是主动给出、因对方而成，还是两宫合成一象。

## RC-xu3-ch1-15: 大运是断命断运的关键枢纽
**use-when**: 区分本命、大运、流年的职责
**source**: PDF pp68-72 / printed pp64-68
**provenance**: [许-高级班]

大运决定十年空间象，是断命断运的枢纽。由“大”往上归本命宫职，决定空间象的吉凶、得失；流年主要负责应在哪一年发生，流月再下推。

### 两种批流年

| 方法 | 取法 |
|---|---|
| 大运十二宫飞归本命 | 先定十年吉凶得失，再看流年感应 |
| 以流命为太岁 | 流命宫干飞化，先归大运宫职，再归本命宫职 |

**guardrail**: 流年不宜拿来重新论有无与大吉凶；它更多是空间象已经成立后的时间触发层。

## RC-xu3-ch1-20: 洛书宫职重叠是宫职累进循环法则
**use-when**: 用宫职重叠判断桃花、健康、寿元、运势
**source**: PDF pp129-131 / printed pp125-127
**provenance**: [许-高级班]

洛书应用的核心是宫职重叠。大运宫位自动移动，飞宫也制造宫职重叠；象成立后，重叠宫职决定事项类别。

| 洛书关系 | 重点事项 |
|---|---|
| 戴九履一 / 命财官一体 | 命、财、官合参，官禄看运势荣枯、求学事业 |
| 二四为肩 / 兄子一体 | 桃花、学生缘、异性缘 |
| 左三右七 | 驿马、外勤、迁动 |
| 六八为足 / 疾奴重叠 | 身体健康 |
| 奴疾田重叠 | 健康兼寿元，凶象成立时可论生死风险 |

**guardrail**: 寿元最怕奴、疾、田重叠再遇凶象。只有象学凶象成立且宫职重叠到位时，才可进入寿元判断。

## RC-xu3-ch1-21: 宫星象气数都讲往来
**use-when**: 本宫与对宫互相解释，或宫职互借
**source**: PDF pp131-132 / printed pp127-128
**provenance**: [许-高级班]

气象宫星一律言往来。本宫与对宫不是孤立两点，而是同一轴线的互相说明。

### 宫职往来示例

- 命宫与迁移宫都是我，迁移是在外的我。
- 兄弟与奴仆可互言，奴仆是广义兄弟。
- 夫妻与官禄互通，夫妻宫也可作官禄，官禄也可作夫妻。
- 财帛与福德互通：财帛是今生因缘果报，福德是三世因果；福德亦可作交际应酬财。
- 父母与疾厄互通：父母可论头面、遗传，疾厄亦可论遗传。

**guardrail**: 往来不是随意替换宫位，而是在本对轴线或既定宫职关系中取互相注脚。

## RC-xu3-ch3-01: 飞宫先发生，自化与生年决定最后究竟
**use-when**: 飞宫化忌入内但逢自化/生年，需判断得失终局
**source**: PDF pp195-196 / printed pp191-192
**provenance**: [许-高级班]

许铨仁在进入婚姻桃花前补充：飞宫之象先发生，自化与生年再发生，用来定最后究竟。比如大财化忌入田宅本主得财；若田宅对宫有生年忌、自化忌，或生年忌加自化忌，则形成“出忌”，归藏之气散掉，物上可变成破财，人上可变成生离死别。

### 解厄制化层级

1. 生年科最有解厄制化力。
2. 飞宫同宫干化科也有一定解厄力。
3. 自化科力量薄弱，有时“有又像没有”。

**guardrail**: 忌入六内不能直接定吉。若对宫有离心忌把气带出，要改判最后究竟。

## RC-xu3-ch5-02: 批流年有两条主线：流命四化与大运十二宫归本命
**use-when**: 批流年、定一年吉凶与事件主题
**source**: PDF pp293-294 / printed pp289-290
**provenance**: [许-高级班]

批流年有两个方法：

1. 流年命宫四化：流命四化先与大运宫职发生宫对宫碰撞，再归本命盘决定吉凶。流命为人，大命为地，本命宫为天；人、地、天合一见吉凶，而大运宫职是解释重心。
2. 大运十二宫四化归本命盘：如大财、大疾、大田、大父等四化归入本命盘，决定该年所应事件，再下应流年。

例：大父归本疾，忌冲本父，流年走到时既要看父母健康，也要看文书问题；两者合起来才是完整批流年。

**guardrail**: 只看流命四化会漏掉大运十二宫职垂象；只看大运飞化也会少了流年人事入口。

## RC-xu3-ch5-21: 在数因缘以体用分：来因或大限有生年/对宫可直接定物相
**use-when**: 判断某段姻缘、职业、桃花或事件是否属时节因缘
**source**: PDF pp331-334, 346-349 / printed pp327-330, 342-345
**provenance**: [许-高级班]

“在数因缘”可分体与用：来因宫有生年或对宫为体，可直接定象、定物相；大限有生年或对宫为用，也可定时节因缘。时节因缘有长有短，只说明现象存在，不先论吉凶与久暂。

有生年无自化时，要来因宫合生年四化所落宫，作宫象宫解说；大命或流命坐照，一定出现“有”的现象。无生年有自化时，生年宫位震荡不安；当大命、流命走到生年忌，十一宫自化忌可作忌入，主得、主有、主吉，但忌的性质仍含冷门、变动、不顺。

**guardrail**: “有”不等于“好”。有其现象后，还要按六内六外、宫职与星曜判吉凶得失。

## RC-xu3-ch5-22: 洛书应用以生年四化不动、大限宫职重叠为动态空间
**use-when**: 用洛书二四、六八、三合三角判断桃花、灾厄、在数
**source**: PDF pp341-345 / printed pp337-341
**provenance**: [许-高级班]

河图决定生态空间，偏本体格局；洛书决定运势走势，偏现象在数。斗数应用中，生年四化不动，大限随时间而动，通过宫职重叠形成洛书组合，呈现动态生活空间。

洛书速查：

| 洛书组合 | 用途 |
|---|---|
| 一五九 | 命财官一体，能力、气数、专技、赚钱 |
| 二四为肩 | 桃花、异性缘、肩位发生 |
| 六八为足 | 身体、灾厄、奴疾田、寿元风险 |

例：大子坐生年科，等同本兄化科到大子，合二四为肩，十年有桃花；若对宫自化科形成科出，可解释太太因桃花离家。大疾化忌到本奴，合六八为足，十年身体须特别注意。

**guardrail**: 洛书不是脱离飞宫的另一套系统。它以生年四化为重心，借大限宫职重叠来取动态现象。

## RC-xu3-ch6-11: 批流年先看流命走生年，再用大运飞宫与流命自化细判
**use-when**: 年度咨询、财官婚姻安全年、当年重点提示
**source**: PDF pp358-359 / printed pp354-355
**provenance**: [许-高级班]

批流年先看流命或流迁是否走到生年禄权科忌；走到就一定有现象，不论时间长短或缘浓缘薄。再合来因宫解释：例如流命走生年禄，来因本子，可解为新桃花，也可能是合伙；若再看宫干化禄在哪，就是在地成形。

传统批流年步骤：

1. 从大运十二宫飞宫。
2. 先论吉凶，再论时间。
3. 若流年命宫有自化，则该年有变化。
4. 看流命飞宫，先化忌，次禄权科。

例如大夫化忌冲本夫，若今年流命走到化忌，可把今年定为“婚姻安全年”，并进一步看二月、八月等月份。

**guardrail**: 流年提示宜转为安全月、修身养性月、风险管理建议，而非恐吓。

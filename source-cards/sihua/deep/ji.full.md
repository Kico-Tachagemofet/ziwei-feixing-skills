### RC-xu2-ch3-08: 飞宫四化的“宫-象-宫”公式

> 来源：`extractions/xu-book2/ch3-sihua-rules.md`
- ### 定义
- 飞宫四化为象之用，表后天变数，由用归体，也归生年。飞为显象，伏为生年四化。飞宫用于明当下之机：本命十二宫、大运十二宫、流年命宫都可作为发射宫。
- ### 公式
- `A 宫四化入 B 宫`，等价于：
- - `A 宫 + B 宫 = 禄/权/科/忌`
- - A 宫为发射宫、注脚宫。
- - B 宫为象所在宫。
- - 禄权科入 B 宫影响对宫，称照。

### RC-xu2-ch3-09: 飞宫理则一、二：互为阴阳与各为阴阳

> 来源：`extractions/xu-book2/ch3-sihua-rules.md`
- ### 理则一：互为阴阳说
- 交易的两宫互为阴阳，凭借象假合为一太极，作 `A 宫 + B 宫 = 象` 的解说。
- ### 理则二：各为阴阳说
- 象言往来；当化忌入 B 宫时，会冲 B 的对宫，此时 A 宫与 B 对宫发生对待关系，见吉凶。若是禄权科，则为照，也要看 A 宫与 B 对宫的关系。

### RC-xu2-ch3-11: 飞宫理则四、五：同类宫职与“一忌走天涯”

> 来源：`extractions/xu-book2/ch3-sihua-rules.md`

| 路径 | 解义 |

### RC-xu2-ch3-13: 飞宫理则七、八：交易象逢伏象/自化象须碰撞

> 来源：`extractions/xu-book2/ch3-sihua-rules.md`
- ### 理则七：交易象逢伏象须碰撞
- 飞宫四化为交易象、象之用；生年四化为伏象、象之体。飞宫落入有生年四化之宫，用象与体象发生碰撞，构成命与运的交错。
- 示例：`|夫| 化禄入本官`，本官已有生年忌，则 `禄 + 生年忌 = 双忌`，可出现姻缘到来但种下多起伏、多变化的恶缘。
- ### 理则八：交易象逢自化象须碰撞
- 飞宫落入有自化之宫，交易象与自化象碰撞。示例中 `|夫| 化权入本财`，本财自化禄，可解婚后对待易争执摩擦，且与桃花泛滥相关；自化重心在自化宫，而非被重叠的本命宫。

### RC-vA08-01: 飞宫基本理念是“宫随象走，象以宫利”

> 来源：`extractions/video-course/A08-A09-eight-rules.md`
- 飞宫学用于补足静态宫象无法回答的动态问题。静态能说配偶特质，飞宫才能判断何年何月结婚、婚后能否白首。

| 记号 | 名称 | 含义 | provenance |
| A宫 | 飞象宫 | 我们要研究的标的宫位 | [video-course] |
| B宫 | 象所落宫 | A 宫飞出的禄权科忌落入之处 | [video-course] |
| C宫 | B 宫对宫 | 被照或被冲的宫位 | [video-course] |
- “宫随象走”表示 A 宫随着四化象进入 B 宫；“象以宫利”表示四化象假合 A 与 B，形成 `A宫 + B宫 = 象` 的解读。

### RC-b7-04: 禄权科忌吉凶二分

> 来源：`extractions/b-series/b7-flying-palace-liqi.md`
| 四化 | 性质 | 飞入某宫 | 术语 |
| 禄・权・科 | 吉 | 飞入 B 宫 | **照** B 宫 |
| 忌 | 凶 | 飞入 B 宫 | **入** B 宫，**冲** B 的对宫 |

### RC-b7-06: 理则二——大运宫位化忌冲本命命宫 = 凶

> 来源：`extractions/b-series/b7-flying-palace-liqi.md`
- ### 规则
- 理则一的时间延伸：大运的任何宫位化忌入本命迁移宫冲本命命宫 → 同样是凶。

- **气由大向小走。**

### RC-b7-07: 理则三——大运化忌冲本命同名宫位 = 凶（后天冲先天）

> 来源：`extractions/b-series/b7-flying-palace-liqi.md`
- ### 规则
- 大运某宫化忌入本命对宫、冲本命同名宫位 → 该宫位这十年为凶。后天（运）冲先天（命），即用冲体。

### RC-b7-08: 三大理则精简总纲

> 来源：`extractions/b-series/b7-flying-palace-liqi.md`
- ### 总纲
- 1. **理则一（冲命宫）**: 本命盘中，任何宫位化忌入迁移宫冲命宫 → 该宫位为凶。命宫是体、是核心，核心被冲则全盘出问题。
- 2. **理则二（大运冲命宫）**: 理则一的延伸——大运任何宫位化忌入本命迁移宫冲本命命宫 → 同凶。气由大向小走，忌的落点先看本命层级。
- 3. **理则三（后天冲先天）**: 大运某宫化忌入本命对宫、冲本命同名宫位 → 该宫位这十年为凶。后天冲先天 = 用冲体。

### RC-b7-09: 多线索锁定法

> 来源：`extractions/b-series/b7-flying-palace-liqi.md`
- ### 规则
- 判断一件事至少要有**两到三条线索**同时指向，才可锁定。
- ### Worked Example: 26-35岁大运问婚姻
| 线索 | 路径 | 结论 |
| 线索一 | 大运交友宫（庚）化天同忌入本命迁移 → **冲命宫** | 交友运凶，易交损友/被骗财 |
| 线索二 | 大运命宫（丁）化巨门忌入本命迁移 → **冲命宫**；且大运命宫与本命夫妻宫重叠 | 这十年整体不顺，尤其夫妻不顺 |
| 线索三 | 大运夫妻化忌入本命官禄 → **冲本命夫妻** | 后天夫妻冲先天夫妻，婚姻凶象确认 |
- 两条以上线索指向婚姻不利 → **锁定：这十年结婚难，或婚后易出问题。**

——属人 天 人，即构成天 人合一，天 人合一，天 人合一的关系，
故确立庚干是天同化忌星。
说明四： 廉贞、天机、天同的化禄星成等级的排列，而其化忌星
却成等比例的排列，如上表的指示，更确立庚干是天同化忌星。
说明五：庚干的太阴化科星与血光有关，癸干的太阴化科星与桃花
有关。
综上所述，既知四化为斗数用神，而能明确其理，可达“用之不疑，
疑之不用”之功，为初学斗数之朋友扎下入门根基。

第二节 四(化)象的基本含义
本节阐释四（化）象的基本含义之前，必先提示四(化)象研究之要
领，以达事半功倍之效，俾使初学者直通正确的捷径，实则包括已开馆

回归正题，四(化)象的研究要领：（一）象要分人、物、事之别称
作解说；（二）象依宫位而立，言宫象合一说；唯须知象从何处来而作

因生年四(化)象为象之体，其质不变(第三节专述)；（四）象以组合结
构而分析其三象一物说，四象一元质，以便知其格与来因、互因、结果
论。(另出书专述)
※化禄星的基本含义
（1）代表财禄、食禄、福禄(含福报、慧报)。
（2）主人缘佳，异性缘佳——含桃花的意味。
（3）表忙碌、随和、施舍。
（4）主聪明、悟性高、禀赋佳，主有才艺。
（5）表增加、增多——偏向财方面。
（6）有解厄制化之功，与健康、寿元有关。
以上是化禄星的基本含义，初学者应熟稔之；因四(化)象为斗数占
命论运之錀(lún，金)，兹为加强读者再剖析如下：
禄——宇宙万有一切缘起的开端，犹如生命在于一系列缘起的无尽
洪流中；所以象征新的开始，如新认识的人、新的知识、新的事业。而
生命在时间的长流之下，万物万象是不断的更替换新，故缘起的化禄就
犹如新的展望、寄托或期待，相当于一颗希望的种子，不论是人、是物、

[[PDF p55 / printed p46]]
是事；因此，化禄在人而言，是缘字的起点，故坐禄于何宫，其宫就是
聚缘处。
例（一）生年禄在迁移宫
表聚缘之所在是迁移宫，谓之“外地发迹”之命，同时参看星性，
指示何种财禄；另言禄照命，表自立、或自求多福、或福自天申。
例（二）贪狼星生年禄在命宫
表聚缘之所在是命宫，于人谓之靠人缘得财禄，亦表自立；然需以
贪狼星的特性发挥其才华，如艺术、美工、巧艺、或五术得财；唯贪狼
星是一颗可福可祸之星，若贪欲之心过重或桃花太浮滥，表祸星坐命，
损及福泽、寿元。
例（三）破军星生年禄在官禄宫
表聚缘之所在是官禄宫，表事业心重，忙碌中得其财禄；唯破军星
表偏财、典当之财、或流动之财、机器维修之财，故可从事股票、仲介、
六合彩等。
※化权星的基本含义
（1）表征权势、权柄、威严。
（2）占权、掌权、具领导的实质。
（3）任性、霸道、性刚。
（4）主能干、有才华、含专技、表开创。
（5）表争执、摩擦，亦表外来伤害如跌伤、撞伤、烫伤。
（6）增加、增值——倾向权力、地位或不动产。
※化权星的剖析
权——宇宙万有一切因缘之自变体数，意指本来就会变，而非有原
因，纵是有因，亦感无奈，即莫名而至，故似无常。换言之，化权是对

[[PDF p56 / printed p47]]
已有的人、事、物作一种自然的变化，无为于天理流行，无奈于人自默
受。相对的，化权的特性更显得自立感、责任感，不向环境屈服的坚韧
性，对事也不喜欢假手他人，有应变力、负责到底，故对实质的拥有，
也须付出相当的代价而有，其现象有似“不耐天磨不成大器”之“磨”
字，以致老天对化权在命宫有不太公平的对待，于人就有无奈感的怨声，
如“我是如何的待人，而人为何如此待我？真不公平！”然这不公平，
就会去找化科或化禄，求其寄托的平衡点。于是生命的长流，不再孤寂
（三点平衡）。因此，化权是有孤独的独来独往、我行我素的潜在性，
好比有些作家、科学家、生物学家，哲学家......因孤独而成就自己，
所以化权教人“逆来顺受，处变于自然而度过”正如佛家言：应劫须逢，
天意也。
※化科星的基本含义
（1）表科甲、科名、功名。
（2）主声名、声望、名望(惜面子)、清白之宿。
（3）主聪明(一学即知)、博学多闻、主才艺。
（4）主平顺，表擅于计划。
（5）主贵人、解厄(与健康有关)。
（6）主风度、斯文、风情、珍惜、和睦（与桃花有关）。
注：风度——行为、举止的风采表现于外，通指男性。
风情——情愫蕴含于内，而呼之于风雅，通指女性。
※化科星的剖析
科——宇宙万有一切因缘之延续，与禄所不同的差异是：禄表新的
展望，而科表旧有的回顾。故万有因缘即有缘起，就有缘续，犹如新旧
接替，方能绵延于后，而成恒古不灭的岁月长流，继起承先启后的天职，
故禄、科是宇宙气化之象，象征生生不息。

[[PDF p57 / printed p48]]
故科的特征——旧识之人，念旧往昔（包括人、物、景）重复、二
度.......归究即是恋眷之星，有一种放不下心的烦恼，即一种难以外
人所知的隐衷。所以对人言，是很念旧之人，亦表荫人或人荫之象征；
对物而言，除上班或智慧的企划，作为谋财之外，尚代表喜好古老的东
西、古董、文物......换言之，对陈年之物有独钟之情；对事而言，倾
向于智慧、文笔、才华，且对学识有偏好、专攻、研究，最好有温故知
新的习性，即温故而突破的本能。
※化忌星的基本含义
（1）主变动、变迁、驿马、动荡不安。
（2）收藏、收束、管束。
（3）亏欠、固执、情义。
（4）不顺、困扰、是非。
（5）凶兆、灾厄、死亡。
（6）无缘、缘薄（人生观、价值观有差距，未必表刑克）。
※化忌星的剖析
忌——宇宙万有一切因缘缘起的终点，相当于一个结束，但所谓结
束，并不表示万缘全绝，是指万有万象之机变数，是生化作用，是物化
原理。(注：权——自变数，忌——机变数)。
故忌如果是一种果报，它必履经禄、权、科的过程，而产生应得之
果；相当于一种气自然流行的终点，犹如经过春、夏、秋而有冬，此乃
属循环之果。
因此，忌是对已有的人、物、事之果报相应而得，故不论是占有（有
形）或拥有(无形)，均同样的变化，意指忌的本能是机变数，其所得含
有机或无机；明确地说即是缘结束后，是有生机的转进，亦或无生机矣。
若属有机之生命变化，即终止的刹那间，同时又隐藏了另一生机，让生

[[PDF p58 / printed p49]]
命再存有另一新生的空间，等待展开而适其生存，犹如易曰：“卦终无
卦”。
※双象组合的基本含义
所谓双象组合，有下列五种情况：
①飞宫双象落同宫或本——对宫
②飞宫逢自化
③飞宫逢生年
④生年逢自化
⑤生年双象同宫或本——对宫。
故须对双象组合的基本含义有所认识，共有六种组合，介绍如下：
（一）禄权组合的含义
（1）主丰盛、财官兴旺，偏向一般投资、生产的生意。
（2）论人表二个人以上，言桃花属泛水桃花。
（二）禄科组合的含义
（1）名利双收，因名得利，偏向巧艺的生意或文路之人，如会计
师、律师、医师、老师......
（2）论人表二个人以上，论桃花主有缘、有情。
（三）权科组合的含义
（1）以专技、才艺为主，如技师、工程师、建筑师......，或军
政、或上班。
（2）论人或桃花，象二数一，即一显一隐。
（四）禄忌组合的含义

[[PDF p59 / printed p50]]
（1）表起伏大、变化大、周转大。
（2）往往以双忌论，主凶。(除①化忌逢生年禄②生年禄自化忌外)
（五）权忌组合的含义
（1）以专技、技能为主，如技工、机工......，或警、或俸薪者。
（2）主善变、突变，来得莫名，去也莫名，含无奈或天意。
（六）科忌组合的含义
（1）以学术、才艺为主，或上班为宜，或教育、著作或服务业。
（2）感情上表藕断丝连，主纠缠、是非、唠叨、拖拖拉拉。

## RC-xu3-ch1-05: 有其象必有其人事物，再依象性区分
**use-when**: 判断某类事项“有没有”时
**source**: PDF pp21-24 / printed pp17-20
**provenance**: [许-高级班]

禄权科忌都主“有其象”，因此先言有无，再言吉凶。比如命宫四化到子女宫，或子女宫四化到命宫，都可以先说有桃花，再依象性区分质量。

| 象 | 桃花/性生活倾向 |
|---|---|
| 禄 | 热情、次数多、圆融 |
| 权 | 占有、强势、性欲或能力强，低格可见骚扰 |
| 科 | 情调、气氛、前奏与风度 |
| 忌 | 缘薄、草率、将开始也将结束 |

**guardrail**: 忌也要先说“有”，再说缘薄或结束。不可因见忌就直接断“没有”。

## RC-xu3-ch1-06: 禄随忌走，但单象自化忌例外
**use-when**: 处理禄忌组合、职业适性、事项因果
**source**: PDF pp18, 57, 65, 82 / printed pp14, 53, 61, 78
**provenance**: [许-高级班]

禄忌为因果组。一般情况下，忌代表结果或收束，禄代表原因、行业、缘起，因此“禄随忌走”：见忌要追禄，以说明因何而忌、靠何而得或失。

### 例外

单象自化忌时，许铨仁提示有时要倒过来作“忌随禄走”，因为自化本身还有时间与变化的结构，不能机械套一般飞宫。

**guardrail**: 财帛断得失常以忌为主，但“怎么得/怎么失”仍要追禄。职业适性也常由忌的宫位与禄的星曜、宫职共同决定。

## RC-xu3-ch1-08: 化权的核心是掌控、开创、外伤与升迁
**use-when**: 解释权象或官禄化权
**source**: PDF pp40-50 / printed pp36-46
**provenance**: [许-高级班]

化权主权柄、威严、占有、掌控、领导、性刚、才华专技、开创斗志。人际宫见权，多见争执摩擦；身体/人位见权，也有外来碰撞伤害的意味。

### 应用重点

- 断升迁、掌权、事业开创，重点看权。
- 官禄化权到命：靠自己开创事业或提升身份。
- 官禄化权到迁移：在外开创、外地发展事业。
- 官禄化权到财帛：事业扩大是为了赚钱，且越扩大越赚钱。
- 官禄化权到子女：可合伙做事业，但经营权多在合伙人。

**guardrail**: 只要是人位，权常归该宫位的人所有；不要自动把权都归命主。

## RC-xu3-ch1-09: 化科兼具功名、贵人、解厄、缘续
**use-when**: 判断功名考运、贵人、因果桃花
**source**: PDF pp51-55 / printed pp47-51
**provenance**: [许-高级班]

化科主科名、功名、声誉、聪明才艺、计划性、贵人、解厄、风度、和睦。许铨仁把四化缘分过程区分为：禄为缘起，科为缘续，权为缘变，忌为缘灭。

### 功名条件

- 断考运首重化科，其次化禄。
- 断升迁首重化权。
- 生科落命、父母、官禄，同时来因宫为父母或官禄时，可作功名、公职、服务体系之象。

**guardrail**: 禄的解厄偏健康寿元，科在人事物上都较能作贵人和解厄。功名不能只看单颗化科，还要看落宫与来因。

## RC-xu3-ch1-10: 化忌七义与六内六外得失
**use-when**: 解释忌象的具体含义
**source**: PDF pp57-59 / printed pp53-55
**provenance**: [许-高级班]

化忌可分为七类含义：

| 忌义 | 操作含义 |
|---|---|
| 收束 | 人位自化忌可为收敛，疾厄可见瘤，命宫可见痣 |
| 管束 | 人位化忌到人位，常为管束对方 |
| 收藏 | 财帛忌入六内主得，入六外主失 |
| 亏欠 | 生年忌坐六亲宫，有亏欠感 |
| 不顺 | 事与愿违、坎坷、纠缠 |
| 缘薄 | 人位忌冲人位，聚少离多或价值观差距 |
| 凶兆 | 严重时可到疾病、灾厄、死亡 |

### 财帛用法

财帛忌入六内主得；命、田、疾为实得或多，财、官、福为周转中。怎么得财看禄，时间看权。

**guardrail**: 忌入不是一律凶。财帛、田宅、疾厄等物事宫必须先判六内六外和得失，再论吉凶质量。

## RC-xu3-ch5-16: 禄为缘起、忌为终点、权为天意无奈、科为延续旧缘
**use-when**: 解释四化的哲学含义与人事转译
**source**: PDF pp322-324 / printed pp318-320
**provenance**: [许-高级班]

四化哲学定义：

| 四化 | 高级班定义 |
|---|---|
| 禄 | 缘起、起点、新欲望、新事业、新人事物 |
| 忌 | 因缘终点、结束、尘埃落定 |
| 权 | 独立、实质主义、责任、天意与无奈，处变、逆来顺受 |
| 科 | 延续、旧缘、念旧、重复、二度、承先启后、贵人、解厄制化 |

科在婚姻中含二度、重复，辅弼和化科对婚姻有“再一次”意味；科也有恋旧、放不下、隐衷。科在六内偏荫人，科在六外偏人荫；若科再自化忌，可能出现荫人不得善果或得人福荫但心态不同。

**guardrail**: 四化不是吉凶标签。禄也会缘尽，科也会障碍，权也会无奈，忌也有归结与成形。

## RC-xu3-ch5-18: 斗数禄权科忌是机遇人生，易经禄科权忌是造命学
**use-when**: 区分一般命运链与修心/造命链
**source**: PDF pp325-327 / printed pp321-323
**provenance**: [许-高级班]

斗数排列为禄、权、科、忌，象征芸芸众生的机遇人生：缘起之后经历缘变无奈，进入化科的怀缅，最后归于忌的尘埃落定。重心落在科，若恋而不放，就变为命运锁链。

易经排列为禄、科、权、忌，象征造命学：缘起之后珍惜其缘、随缘待之，再纵任缘变，终归其位。权科互因是天人合一的核心；权偏天、科偏人，若能以心境傲数，由有为转无为，即是造命。

**guardrail**: 这是哲学层的抽取，不可用来替代具体飞宫断事；它用于解释命理咨询的价值观。

### RC-vA10-04: 第一段婚姻案例：自化权主变化中带括号

> 来源：`extractions/video-course/A10-practical-application.md`
- 戊子女命 25 岁结婚、29 岁离婚，均在 22-31 大限。大运夫妻落本命财帛，宫干壬使紫微化权，紫微在本宫，形成自化权。
- ### 结婚链
| 层次 | 判读 |
| 自化第一层 | 夫妻宫一定有变化 |
| 自化第二层 | 单身结束，为人之妻开始 |
| 自化第三层 | 25 岁流年走到大运夫妻所在位置，飞化位成立 |
- ### 离婚链
- 自化权、视同自化权特殊：变化中带括号，表示变化不是一次性的，还会再变。就近说再看大运夫妻壬干武曲忌，忌冲本命夫妻，理则四成立，这十年婚姻亮红灯；飞化位推至 29 岁离婚。

### RC-b1-04: 自化象

> 来源：`extractions/b-series/b1-three-transformation-types.md`
- ### 定义
- 一个宫位的天干飞出的四化中，若所需星曜恰好落在**本宫**或**对宫**（同一条线 = 同一公司），即为自化。
- ### 分类
| 类型 | 条件 | 特征 |
| **纯自化** | 四化星落在本宫 | 一次性事件 |
| **视同自化** | 四化星落在对宫（同一条线） | 多了**涌动力**——一波刚平一波又起 |
- ### 六条线（六组对宫）
- 命迁线、兄友线、夫官线、子田线、财福线、父疾线。

### RC-b1-05: 三象层级总纲

> 来源：`extractions/b-series/b1-three-transformation-types.md`
- ### 层级
- 1. **生年四化象（命）**：出生年干决定，能量最强，格局高低已定，永久不变
- 2. **飞宫四化象（运）**：各宫天干飞出四化到其他宫位，推断运势吉凶
- 3. **自化象（运）**：四化落回本宫或对宫，纯自化为单次变化，视同自化多涌动力
- ### 判读顺序
- 先读生年四化定格局 → 再读飞宫四化看运势 → 自化作为变化/触发的补充层。

## RC-xu3-ch1-01: 四化象是先天无为法，星曜五行是后天指示
**use-when**: 区分庙旺利陷、星曜五行与四化象的优先级
**source**: PDF pp11-13 / printed pp7-9
**provenance**: [许-高级班]

许铨仁把星曜五行、宫位五行、庙旺利陷归为后天有为法；四化象归为先天无为法。五行生克能作指示，但断吉凶得失要以四化象言气数。

### 操作原则

- 宫与星的五行生克产生庙旺利陷，是后天层面的指示。
- 四化象决定气数；星曜受四化后成为指事、指物、指人的标识。
- 以紫微为太极、日月为旋转、四化象为斗数用神。

**guardrail**: 不要先用庙旺利陷定终局，再把四化当补充。许氏体系中四化象优先给出气数，星曜负责落实形相与事项。

## RC-xu3-ch1-02: 宫星象三合一，象先言吉凶，星再成形
**use-when**: 解读宫位、星曜、四化同时出现的结构
**source**: PDF pp13-18 / printed pp9-14
**provenance**: [许-高级班]

完整解义要同时处理宫、星、象：

| 层级 | 功能 |
|---|---|
| 宫象合一 | 宫位加四化象，先言吉凶、气数、得失、悔吝 |
| 宫星合一 | 宫位加星曜，说明人事物形相与性情 |
| 宫象星合一 | 四化定气数，星曜负责指示具体形态 |

### 生年、飞宫、自化

- 生年四化是象之体，与生俱有，其质不变。
- 飞宫四化是象之用，显现后天交易行为。
- 自化也是用，但带时间、变化、无中生有与有中化无。

**guardrail**: 星曜不能单独定吉凶。必须先问象是什么，再用星曜说明“在天成象，在地成形”的具体形。

## RC-xu3-ch1-03: 先天气数不透过大限，命宫与生年四化先定本质
**use-when**: 问“先看什么”或要先判断命主本质时
**source**: PDF pp16-18 / printed pp12-14
**provenance**: [许-高级班]

论先天气数时，不必先透过大限。先看命宫，再看生年四化；命宫代表此生所有本质，生年四化代表先天定数。大限、飞宫与自化用于后续解释运与具体事项。

### 看盘顺序

1. 先看命宫：个性、性向、职业适性、命的基本质地。
2. 再看生年四化：象之体，定先天气数。
3. 当生年四化不足以组合成具体人事物时，再用飞宫四化。

**guardrail**: 大限用于动盘和应期，不应在还没看清命宫与生年四化前抢先定义命主本质。

## RC-xu3-ch1-04: 宫象宫公式有三种解法，发射宫决定主动方
**use-when**: 解释 `A宫化禄/权/科/忌到B宫`
**source**: PDF pp20-29 / printed pp16-25
**provenance**: [许-高级班]

`A宫四化入B宫` 可用三种角度解释，取最合象义者：

1. A 把禄权科忌给了 B。
2. 因 B 宫的缘故，使 A 宫禄权科忌。
3. `A宫 + B宫 = 禄/权/科/忌`。

命宫化禄到兄弟，与兄弟化禄到命，结果可能同为兄弟圆融、互得助力；差别在过程。发射宫是谁，谁就是主动方。

**guardrail**: 不要只背“化到某宫”的固定断语。要先认发射宫、受象宫，再判断是主动给出、因对方而成，还是两宫合成一象。

## RC-xu3-ch1-05: 有其象必有其人事物，再依象性区分
**use-when**: 判断某类事项“有没有”时
**source**: PDF pp21-24 / printed pp17-20
**provenance**: [许-高级班]

禄权科忌都主“有其象”，因此先言有无，再言吉凶。比如命宫四化到子女宫，或子女宫四化到命宫，都可以先说有桃花，再依象性区分质量。

| 象 | 桃花/性生活倾向 |
|---|---|
| 禄 | 热情、次数多、圆融 |
| 权 | 占有、强势、性欲或能力强，低格可见骚扰 |
| 科 | 情调、气氛、前奏与风度 |
| 忌 | 缘薄、草率、将开始也将结束 |

**guardrail**: 忌也要先说“有”，再说缘薄或结束。不可因见忌就直接断“没有”。

## RC-xu3-ch1-06: 禄随忌走，但单象自化忌例外
**use-when**: 处理禄忌组合、职业适性、事项因果
**source**: PDF pp18, 57, 65, 82 / printed pp14, 53, 61, 78
**provenance**: [许-高级班]

禄忌为因果组。一般情况下，忌代表结果或收束，禄代表原因、行业、缘起，因此“禄随忌走”：见忌要追禄，以说明因何而忌、靠何而得或失。

### 例外

单象自化忌时，许铨仁提示有时要倒过来作“忌随禄走”，因为自化本身还有时间与变化的结构，不能机械套一般飞宫。

**guardrail**: 财帛断得失常以忌为主，但“怎么得/怎么失”仍要追禄。职业适性也常由忌的宫位与禄的星曜、宫职共同决定。

## RC-xu3-ch1-07: 富贵贫贱格局先看禄权科位置与忌的内外
**use-when**: 判断先天格、后天局、六亲格
**source**: PDF pp37-39 / printed pp33-35
**provenance**: [许-高级班]

论格与局要分先后天：

| 类型 | 取法 |
|---|---|
| 先天格局 | 看生年四化落宫 |
| 后天格局 | 看命宫干四化落宫 |
| 六亲格局 | 以该六亲宫立太极，看其宫干四化落宫 |

### 富贵贫贱框架

- 禄、权、科多落六阳宫（命、夫、财、迁、官、福）主贵。
- 禄、权、科多落六阴宫（兄、子、疾、奴、田、父）主富。
- 忌在六内（命、财、疾、官、田、福）不冲，较可成格。
- 忌在六外（兄、夫、子、迁、奴、父）易破格。

**guardrail**: 后天局是努力所得，有局限；先天贵、后天亦贵才是大贵。六亲立太极时借原盘十二宫解，不是重新排一张盘。

## RC-xu3-ch1-08: 化权的核心是掌控、开创、外伤与升迁
**use-when**: 解释权象或官禄化权
**source**: PDF pp40-50 / printed pp36-46
**provenance**: [许-高级班]

化权主权柄、威严、占有、掌控、领导、性刚、才华专技、开创斗志。人际宫见权，多见争执摩擦；身体/人位见权，也有外来碰撞伤害的意味。

### 应用重点

- 断升迁、掌权、事业开创，重点看权。
- 官禄化权到命：靠自己开创事业或提升身份。
- 官禄化权到迁移：在外开创、外地发展事业。
- 官禄化权到财帛：事业扩大是为了赚钱，且越扩大越赚钱。
- 官禄化权到子女：可合伙做事业，但经营权多在合伙人。

**guardrail**: 只要是人位，权常归该宫位的人所有；不要自动把权都归命主。

## RC-xu3-ch1-09: 化科兼具功名、贵人、解厄、缘续
**use-when**: 判断功名考运、贵人、因果桃花
**source**: PDF pp51-55 / printed pp47-51
**provenance**: [许-高级班]

化科主科名、功名、声誉、聪明才艺、计划性、贵人、解厄、风度、和睦。许铨仁把四化缘分过程区分为：禄为缘起，科为缘续，权为缘变，忌为缘灭。

### 功名条件

- 断考运首重化科，其次化禄。
- 断升迁首重化权。
- 生科落命、父母、官禄，同时来因宫为父母或官禄时，可作功名、公职、服务体系之象。

**guardrail**: 禄的解厄偏健康寿元，科在人事物上都较能作贵人和解厄。功名不能只看单颗化科，还要看落宫与来因。

## RC-xu3-ch1-10: 化忌七义与六内六外得失
**use-when**: 解释忌象的具体含义
**source**: PDF pp57-59 / printed pp53-55
**provenance**: [许-高级班]

化忌可分为七类含义：

| 忌义 | 操作含义 |
|---|---|
| 收束 | 人位自化忌可为收敛，疾厄可见瘤，命宫可见痣 |
| 管束 | 人位化忌到人位，常为管束对方 |
| 收藏 | 财帛忌入六内主得，入六外主失 |
| 亏欠 | 生年忌坐六亲宫，有亏欠感 |
| 不顺 | 事与愿违、坎坷、纠缠 |
| 缘薄 | 人位忌冲人位，聚少离多或价值观差距 |
| 凶兆 | 严重时可到疾病、灾厄、死亡 |

### 财帛用法

财帛忌入六内主得；命、田、疾为实得或多，财、官、福为周转中。怎么得财看禄，时间看权。

**guardrail**: 忌入不是一律凶。财帛、田宅、疾厄等物事宫必须先判六内六外和得失，再论吉凶质量。

## RC-xu3-ch1-11: 自化代表变化与时间，是断应期的核心
**use-when**: 需要判断何时发生
**source**: PDF pp59-60 / printed pp55-56
**provenance**: [许-高级班]

自化表变化，含无中生有、有中化无，也含旧的结束和新的开始。自化的重点在“动”和时间；现象与吉凶要落到大运、流年、流月、流日、流时的宫位。

### 空间与时间

- 在天成象：空间有象，先示吉凶。
- 在地成形：时间走到，象数应验。
- 四正偏空间结构；三合偏时间触发。

**guardrail**: 没有自化，命运会变成机械轨迹。断命精细处往往卡在自化与生年四化如何平衡。

## RC-xu3-ch1-12: 无生年而有自化，先论格，再论运
**use-when**: 某宫无生年四化但有自化
**source**: PDF pp60-66 / printed pp56-62
**provenance**: [许-高级班]

无生年四化而有自化时，先论该宫位人的格局，再论运。六亲宫无生年而有自化，常作该六亲“自力更生格”。

### 解法

1. 先本宫论：以大运宫职为重心，本命宫职作形容词。
2. 再对宫论：对宫则以本命宫职为重心，大运宫职作形容词。
3. 合本宫与对宫，才是完整解义。

### 两人共一象

若大运人位与本命人位共用一个自化象，可出现“两人共一象”：大运宫职的人为主，本命宫职的人依附或附和。

**guardrail**: 自化的重心在较小的时间单位；大运是十年重心，本命宫职多作修饰，不要反客为主。

## RC-xu3-ch1-13: 自化应期取自化宫与同类生年四化宫
**use-when**: 判断结婚、创业、财运等应在何年何月
**source**: PDF pp63-65 / printed pp59-61
**provenance**: [许-高级班]

自化应期有两类感应点：

1. 自化宫成为流命或相关流宫时。
2. 同类生年四化宫成为流命或相关流宫时。

若两者交会，气最强，常可定最终应期。许铨仁用“最大公倍数、最小公倍数”的比喻说明：多个可能年份中，取气象重叠最强者。

**guardrail**: 自化权要找自化权宫和生年权宫；自化忌要找自化忌宫和生年忌宫。流年多用于应期，不负责重新决定有无。

## RC-xu3-ch1-14: 自化交易按禄忌因果、权科互因成组
**use-when**: 自化后还要看哪一个四化作平衡
**source**: PDF pp65-70, 81-83 / printed pp61-66, 77-79
**provenance**: [许-高级班]

自化不是孤立读法。禄忌为因果，权科为互因，因此：

- 看化权时同时参看化科。
- 看化忌时同时参看化禄。
- 双象必须有忌，常见为禄忌、权忌、科忌。
- 若命盘先天为某组，如权忌组，后天自化忌就找权，自化权就找忌来平衡先天组。

**guardrail**: 不需要把四个飞化全取。要按先天组取舍，找能与先天定数组成同组平衡的象。

## RC-xu3-ch1-15: 大运是断命断运的关键枢纽
**use-when**: 区分本命、大运、流年的职责
**source**: PDF pp68-72 / printed pp64-68
**provenance**: [许-高级班]

大运决定十年空间象，是断命断运的枢纽。由“大”往上归本命宫职，决定空间象的吉凶、得失；流年主要负责应在哪一年发生，流月再下推。

### 两种批流年

| 方法 | 取法 |
|---|---|
| 大运十二宫飞归本命 | 先定十年吉凶得失，再看流年感应 |
| 以流命为太岁 | 流命宫干飞化，先归大运宫职，再归本命宫职 |

**guardrail**: 流年不宜拿来重新论有无与大吉凶；它更多是空间象已经成立后的时间触发层。

## RC-xu3-ch1-16: 万象归一元分“入”与“出”
**use-when**: 生年四化与同类自化互动
**source**: PDF pp78-80, 90 / printed pp74-76, 86
**provenance**: [许-高级班]

当命宫、大运命宫或流年命宫走到某个生年四化时，其余十一宫的同类自化都归入该宫，称为“入”。例如大命逢生年忌，大财自化忌，可解为忌入而主得财，仍须按忌的属性说明财型。

若反过来大命逢自化忌，则生年忌所在宫位被带动而不稳定，可称为“出”，常表现为该生年宫职的震荡或损失。

**guardrail**: 入多作有、成局、主得；出多作不稳定、带出、震荡。不能把同类自化一律视为凶。

---


## RC-xu3-ch1-17: 六体六用决定象是实有还是相对有
**use-when**: 判断某象是否一定存在
**source**: PDF pp80-81 / printed pp76-77
**provenance**: [许-高级班]

生年四化是独立体、实有，可再分六体与六用：

| 类别 | 宫位 |
|---|---|
| 六体宫 | 命、夫妻、疾厄、田宅、福德、父母 |
| 六用宫 | 兄弟、子女、财帛、迁移、官禄、交友 |

来因宫或大命为六体宫时，其四化所落宫位肯定存有象。来因宫或大命为六用宫时，其象为相对存有，要结合用处、对待与事项解释。

**guardrail**: 夫妻宫为体，见生年禄可先说婚姻象实有；子女、财帛、官禄等用宫，要回到“为谁所用、用于何事”再断。

## RC-xu3-ch1-18: 四凤三旗两仪标以发射宫为标地，按距离定象变
**use-when**: 自化禄再飞忌，或自化忌再飞禄时
**source**: PDF pp83-87 / printed pp79-83
**provenance**: [许-高级班]

以自化所在宫为标地宫，也就是发射宫。自化禄再看飞忌到哪里；自化忌则看飞禄到哪里。按相对距离解释：

|---|---|
| 邻宫 | 两仪，相欠债 |
| 第三宫 | 三旗，有对待，必有吉凶 |
| 第四宫 | 四凤，主变、形变 |
| 三合宫 | 入宫显吉凶，另一三合宫指示时间 |
| 第六宫 | 主破耗，因奴仆为绝情宫 |
| 对宫 | 主化，形变加质变 |

**guardrail**: 四凤三旗两仪标主要处理无生年有自化的自转关系；还要另看它与生年四化的公转关系。

## RC-xu3-ch1-19: 自转公转：自化先自我互动，再与生年四化互动
**use-when**: 自化与生年四化同时存在
**source**: PDF pp87-90 / printed pp83-86
**provenance**: [许-高级班]

自化禄再飞忌、自化忌再飞禄，是宫位自身气与气的互动，可比作自转；自化再与生年四化互动，是后天变数与先天定数互动，可比作公转。

### 应用

- 先标出生年四化，因为它们是先天定数。
- 判断命盘是单象盘还是双象盘。
- 双象盘至少要有一个忌；同宫、对宫、邻宫、三合都可能构成双象。
- 后天飞化或自化要找能平衡先天组的象。

**guardrail**: 自转关系不等于最终吉凶；最终仍要回到生年四化这四个“定数钉子”。

## RC-xu3-ch1-20: 洛书宫职重叠是宫职累进循环法则
**use-when**: 用宫职重叠判断桃花、健康、寿元、运势
**source**: PDF pp129-131 / printed pp125-127
**provenance**: [许-高级班]

洛书应用的核心是宫职重叠。大运宫位自动移动，飞宫也制造宫职重叠；象成立后，重叠宫职决定事项类别。

| 洛书关系 | 重点事项 |
|---|---|
| 戴九履一 / 命财官一体 | 命、财、官合参，官禄看运势荣枯、求学事业 |
| 二四为肩 / 兄子一体 | 桃花、学生缘、异性缘 |
| 左三右七 | 驿马、外勤、迁动 |
| 六八为足 / 疾奴重叠 | 身体健康 |
| 奴疾田重叠 | 健康兼寿元，凶象成立时可论生死风险 |

**guardrail**: 寿元最怕奴、疾、田重叠再遇凶象。只有象学凶象成立且宫职重叠到位时，才可进入寿元判断。

## RC-xu3-ch1-21: 宫星象气数都讲往来
**use-when**: 本宫与对宫互相解释，或宫职互借
**source**: PDF pp131-132 / printed pp127-128
**provenance**: [许-高级班]

气象宫星一律言往来。本宫与对宫不是孤立两点，而是同一轴线的互相说明。

### 宫职往来示例

- 命宫与迁移宫都是我，迁移是在外的我。
- 兄弟与奴仆可互言，奴仆是广义兄弟。
- 夫妻与官禄互通，夫妻宫也可作官禄，官禄也可作夫妻。
- 财帛与福德互通：财帛是今生因缘果报，福德是三世因果；福德亦可作交际应酬财。
- 父母与疾厄互通：父母可论头面、遗传，疾厄亦可论遗传。

**guardrail**: 往来不是随意替换宫位，而是在本对轴线或既定宫职关系中取互相注脚。

## RC-xu3-ch1-22: 阴阳事项有不同表达伦理
**use-when**: 面对寿元、婚姻、桃花、福德等敏感事项
**source**: PDF pp44-46 / printed pp40-42
**provenance**: [许-高级班]

许铨仁把事项分阴阳：阳为生不带来、死不带走，如财官，可较直断；阴为生会带来、死会带走，如婚姻、桃花、寿元、福德，可讲细，但不宜明断到刺激或误导当事人的程度。

### 咨询原则

- 敏感事项要先确认来问者与当事人的关系、动机和事实背景。
- 命理是相对论，不是绝对论；基准点不同，同一气数对应的现实量级不同。
- 问得越清楚，断语越能落地；答非所问时，只能保持在较泛的层级。

**guardrail**: 寿元、重病、婚姻生离死别等事项，不应脱离来问动机与伦理直接明断。
